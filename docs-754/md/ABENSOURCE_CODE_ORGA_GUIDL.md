Source Code Organization

ABAP source code is saved in the ABAP Repository in the [standard database](ABENSTANDARD_DB_GLOSRY.md "Glossary Entry") of an AS ABAP, and not in source code files like in most other programming environments. Source codes can be split into individual units by using include programs. There are also [macros](ABENMACROS_GUIDL.md "Guideline") that are handled for callable modularization units in the context of these guidelines.

-   [Source code modularization](ABENSOURCE_CODE_MODULAR_GUIDL.md "Guideline")

-   [Multiple use of include programs](ABENMULTIPLE_USE_INCLUDE_GUIDL.md "Guideline")
