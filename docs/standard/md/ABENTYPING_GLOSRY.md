[Generic](ABENGENERIC_TYPING_GLOSRY.html) or [complete](ABENCOMPLETE_TYPING_GLOSRY.html) definition of a [data type](ABENDATA_TYPE_GLOSRY.html) for [formal parameters](ABENFORMAL_PARAMETER_GLOSRY.html) or [field symbols](ABENFIELD_SYMBOL_GLOSRY.html). Specifies in which operand positions a formal parameter or field symbol can be used and which type properties are used. When parameters are passed, it is checked whether the data type of the assigned [actual parameter](ABENACTUAL_PARAMETER_GLOSRY.html), or memory area is [compatible](ABENCOMPATIBLE_GLOSRY.html) with the typing.
