Programs that are executed in [dialog](ABENDIALOG_PROCESSING_GLOSRY.html) and from SAP GUI can use the methods of the class `CL_GUI_FRONTEND_SERVICES` to access files or to execute operating system commands on the current [presentation server](ABENPRESENTATION_SERVER_GLOSRY.html). Furthermore, there is a set of statements for [OLE automation](ABENOLE_AUTOMATION_GLOSRY.html).

The function module `GUI_IS_AVAILABLE` can be used in advance to determine whether a SAP GUI is available.

-   [Files on the Presentation Server](ABENFRONTEND_FILES.html)
-   [Operating System Statements of the Presentation Server](ABENABAP_SYSTEM_COMMANDS_PRES.html)
-   [OLE Automation](ABENOLE2.html)
