`... option="...,..."`

The attribute `option` can be used to override the associated mapping rule, which depends on the data type, from and to [asXML](ABENABAP_XSLT_ASXML.html) for certain data types. The possible options can be specified as the content of the attribute, in a list separated by commas. Any options that are mutually exclusive produce a syntax check warning.

-   [Mapping Rules for Elementary Types](ABENST_OPTION_FORMAT.html)
-   [Mapping Rules for XML Fragments](ABENST_OPTION_XSDANY.html)
