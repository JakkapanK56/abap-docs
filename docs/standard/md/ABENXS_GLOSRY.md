Short for SAP HANA extended application services. XS is a framework for developing and managing native applications in [SAP HANA](ABENSAP_HANA_GLOSRY.html). The current version, [XSA](ABENXSA_GLOSRY.html), replaces the previous version, [XSC](ABENXSC_GLOSRY.html). XS objects can be accessed in ABAP.
