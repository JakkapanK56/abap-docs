[Catchable exception](ABENCATCHABLE_EXCEPTION_GLOSRY.html), which is represented by an [exception object](ABENEXCEPTION_OBJECT_GLOSRY.html) of an [exception class](ABENEXCEPTION_CLASS_GLOSRY.html). Class-based exceptions replace the obsolete [catchable runtime errors](ABENCATCHABLE_RUNTIME_ERROR_GLOSRY.html) and non-class-based exceptions. A class-based exception can either terminate the current context or be a [resumable exception](ABENRESUMABLE_EXCEPTION_GLOSRY.html).
