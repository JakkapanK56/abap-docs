[1\. Loading Multiple Persistent Objects](#ABAP_MODIFICATION_1@4@)

[2\. Query Service](#ABAP_MODIFICATION_2@4@)

From ABAP release 7.0, the interface `IF_OS_CA_PERSISTENCY` of the [persistence service](ABENABAP_OBJECT_SERVICES_PERSIST.html) contains the methods `GET_PERSISTENT_BY_OID_TAB`, `GET_PERSISTENT_BY_KEY_TAB`, and `GET_PERSISTENT_BY_QUERY`. This makes it possible to retrieve more than one persistent object at once from the database and to create the appropriate instances in the ABAP program.

From ABAP release 7.0, the [object services](ABENOBJECT_SERVICES_GLOSRY.html) offer a [query service](ABENABAP_OBJECT_SERVICES_QUERY.html) to find and load [persistent objects](ABENPERSISTENT_OBJECT_GLOSRY.html) from the database.
