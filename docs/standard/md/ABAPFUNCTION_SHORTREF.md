`FUNCTION func.    ...  ENDFUNCTION.`

Implements a function module `func`. A function module is defined in [Function Builder](ABENFUNCTION_BUILDER_GLOSRY.html) and implemented between `FUNCTION` and `ENDFUNCTION`.
