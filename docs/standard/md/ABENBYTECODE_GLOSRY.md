Result of the generation of an ABAP program using the [ABAP Compiler](ABENABAP_COMPILER_GLOSRY.html). The statements in the byte code are linked with C functions. When the [ABAP program](ABENABAP_PROGRAM_GLOSRY.html) is executed, the byte code is loaded into the [PXA](ABENPXA_GLOSRY.html) as program [Load](ABENLOAD_GLOSRY.html) and is interpreted by the [ABAP runtime framework](ABENABAP_RUNTIME_FRMWK_GLOSRY.html) (virtual machine), whereby the associated C functions are called.
