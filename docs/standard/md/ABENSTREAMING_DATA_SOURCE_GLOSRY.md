Repository from which a reader [data stream](ABENDATA_STREAM_GLOSRY.html) takes its data in [streaming](ABENSTREAMING_GLOSRY.html). A streaming data source can be an external resource such as a [LOB](ABENLOB_GLOSRY.html) column of a [DDIC database table](ABENDDIC_DB_TABLE_GLOSRY.html) or a [file](ABENFILE_GLOSRY.html), or an internal resource such as a [string](ABENSTRING_GLOSRY.html) or an [internal table](ABENINTERNAL_TABLE_GLOSRY.html). [Streaming](ABENSTREAMING_GLOSRY.html) is currently implemented for [DDIC database tables](ABENDDIC_DB_TABLE_GLOSRY.html), [strings](ABENSTRING_GLOSRY.html), and [internal tables](ABENINTERNAL_TABLE_GLOSRY.html).
