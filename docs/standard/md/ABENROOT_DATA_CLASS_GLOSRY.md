A global [shared memory-enabled class](ABENSHM_CLASS_GLOSRY.html) that must be assigned to an [area](ABENAREA_GLOSRY.html) when it is defined. A non-empty [area instance version](ABENAREA_INSTANCE_VERSION_GLOSRY.html) contains at least one instance of the area root class as the [root object](ABENROOT_OBJECT_GLOSRY.html).
