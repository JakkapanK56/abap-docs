Method used to navigate to the associated tool of a [repository object](ABENREPOSITORY_OBJECT_GLOSRY.html) within the [ABAP Workbench](ABENABAP_WORKBENCH_GLOSRY.html). To do this, the [cursor](ABENCURSOR_GLOSRY.html) must be positioned on the object name and the [object](ABENOBJECT_GLOSRY.html) must be selected.
