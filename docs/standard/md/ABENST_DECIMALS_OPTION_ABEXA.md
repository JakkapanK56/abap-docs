This example demonstrates the representation of numeric ABAP types and the time stamp type `utclong` with specifications for decimal places.

This example passes various types of numeric ABAP data including an UTC time stamp to the transformation `DEMO_ST_DECIMALS_OPTION` and displays the result for XML and [JSON](ABENABAP_JSON.html). The transformation applies the option [`decimals`](ABENST_OPTION_FORMAT.html) to the ABAP data:

The transformation creates [JSON-XML](ABENJSON_XML_GLOSRY.html) to represent both XML and JSON. The UTC time stamp in the packed number `datetimel` is typed with the [special domain](ABENABAP_XSLT_ASXML_SCHEMA.html)\\ `XSDDATETIME_LONG_Z`. This time stamp is still handled as such after the handling by `decimals`. For the number `timestmpl` of type `p`, the corresponding effect is obtained by using the additional time stamp formats specified with `format`. Time stamps in the time stamp fields of the type `utclong` are handled as such by default.
