Expression-oriented implementation of a Jawbreaker game.

This example uses the implementation of a Jawbreaker game to demonstrate the expression-oriented ABAP programming as introduced in ABAP release 7.40. As a comparison, the program `DEMO_JAWBREAKER_HTML_700` shows a functionally identical implementation without expressions, which is restricted to mainly using language elements of ABAP release 7.00.

To play the game, use the left mouse button to remove related blocks with the same color.
