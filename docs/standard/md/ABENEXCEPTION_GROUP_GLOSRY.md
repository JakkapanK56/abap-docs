Grouping of multiple [catchable runtime errors](ABENCATCHABLE_RUNTIME_ERROR_GLOSRY.html) in an organizational unit. The catchable runtime errors of an exception group, together with the obsolete statement [`CATCH - ENDCATCH`](ABAPCATCH_SYS.html), can be handled by specifying the group name.
