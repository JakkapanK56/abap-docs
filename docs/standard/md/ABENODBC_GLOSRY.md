Short for Open Database Connectivity. A Microsoft application programming interface for accessing databases on networks based on a common language. Communication between the [SAP HANA database](ABENHANA_DATABASE_GLOSRY.html) and external systems via a [SAP HANA remote source](ABENSAP_HANA_REMOTE_SOURCE_GLOSRY.html) is based on the ODBC protocol. The [SQL services](ABENABAP_SQL_SERVICES_GLOSRY.html), which provide an access mechanism to [AS ABAP](ABENAS_ABAP_GLOSRY.html) for external consumers, is also based on ODBC.
