`WSP` for short. A TCP based protocol for the bidirectional communication between a client and a server. Using [ABAP Push Channels](ABENABAP_PUSH_CHANNELS_GLOSRY.html), an [AS ABAP](ABENAS_ABAP_GLOSRY.html) can be used as a server and as a client for the WebSocket protocol.
