[Reference](ABAPEDITOR-CALL_FOR_REPORT.html)

`EDITOR-CALL FOR \{\ \{REPORT prog [DISPLAY-MODE]\}\                  |\ \{itab [TITLE title]\                          [\{DISPLAY-MODE\}|\{BACKUP INTO jtab\}]\}\ \}.`

**Obsolete:** Calls ABAP Editor for a [program](ABAPEDITOR-CALL_FOR_REPORT.html)\\ `prog` or a text editor for text in an [internal table](ABAPEDITOR-CALL_FOR_ITAB.html)\\ `itab`.

-   [`DISPLAY-MODE`](ABAPEDITOR-CALL_FOR_REPORT.html)\\
    Opens the editor in display mode
-   [`TITLE title`](ABAPEDITOR-CALL_FOR_ITAB.html)\\
    Displays the title `title` in the title bar of the text editor.
-   [`BACKUP INTO jtab`](ABAPEDITOR-CALL_FOR_ITAB.html)\\
    Assigns the content of `itab` to the table `jtab` before opening the editor.
