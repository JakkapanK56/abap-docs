[File](ABENFILE_GLOSRY.html) on a [host computer](ABENHOST_COMPUTER_GLOSRY.html) of an [AS instance](ABENAS_INSTANCE_GLOSRY.html) that was opened using the addition [`LEGACY`](ABAPOPEN_DATASET_MODE.html) of the statement [`OPEN DATASET`](ABAPOPEN_DATASET_MODE.html). In a legacy file, the [byte order](ABENBYTE_ORDER_GLOSRY.html) and the [code page](ABENCODEPAGE_GLOSRY.html) for handling the content of the file can be specified.
