Special [function module](ABENFUNCTION_MODULE_GLOSRY.html) for setting and removing [SAP locks](ABENSAP_LOCK_GLOSRY.html). When creating a [lock object](ABENLOCK_OBJECT_GLOSRY.html), a function module with the prefix `ENQUEUE_` is created to set the lock, and a function module with the prefix `DEQUEUE_` is created to remove the lock.
