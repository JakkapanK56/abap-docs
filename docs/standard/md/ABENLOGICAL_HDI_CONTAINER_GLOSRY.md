Part of the definition of an [ABAP-managed HDI container (AMHC)](ABENAMHC_GLOSRY.html) to which a physical [HDI container](ABENHDI_CONTAINER_GLOSRY.html) in the [SAP HANA database](ABENHANA_DATABASE_GLOSRY.html) is mapped. A logical HDI container can be used as a [logical schema](ABENLOGICAL_SCHEMA_GLOSRY.html) in an [AMDP macro](ABENAMDP_MACRO_GLOSRY.html).
