The default value of profile parameter `rdisp/max_alt_modes`, that determines the possible number of [ABAP sessions](ABENABAP_SESSION_GLOSRY.html) per [user session](ABENUSER_SESSION_GLOSRY.html), was enhanced from 6 to 16 and is now the same as the maximum number of ABAP sessions per user session.
