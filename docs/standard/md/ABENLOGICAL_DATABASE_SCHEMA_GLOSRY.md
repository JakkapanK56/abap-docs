A [logical database schema](ABENLOGICAL_DB_SCHEMA_GLOSRY.html) in which the assignment of a physical [ABAP database schema](ABENABAP_DB_SCHEMA_GLOSRY.html) of the [SAP HANA database](ABENHANA_DATABASE_GLOSRY.html) takes place in a mapping table. A logical local database schema is represented by a transportable [repository object](ABENREPOSITORY_OBJECT_GLOSRY.html) that defines its properties.
