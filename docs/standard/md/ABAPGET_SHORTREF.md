[Reference](ABAPGET-.html)

`GET node [LATE]\ [FIELDS f1 f2 ...].`

**Obsolete:** Introduces an event block whose event is raised by a [logical database](ABENLOGICAL_DATA_BASE_GLOSRY.html) when data records of the node `node` are read.

-   `LATE`\\
    Event if the logical database has read all the data records of a node `node`.
-   `FIELDS f1 f2 ...`\\
    Specifies which fields are read from the logical database.
