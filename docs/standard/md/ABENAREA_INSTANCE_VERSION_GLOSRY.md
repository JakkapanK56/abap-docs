Variant of an [area](ABENAREA_GLOSRY.html) in the [shared objects memory](ABENSHARED_OBJECTS_MEMORY_GLOSRY.html). In an area instance version, [shared objects](ABENSHARED_OBJECTS_GLOSRY.html) are stored in the form of [anonymous data objects](ABENANONYMOUS_DATA_OBJECT_GLOSRY.html) or instances of [shared memory-enabled classes](ABENSHM_CLASS_GLOSRY.html), where for every area instance version, there is at least one instance of the [root data class](ABENROOT_DATA_CLASS_GLOSRY.html) of the [area](ABENAREA_GLOSRY.html). The creation of and access to area instance versions takes place exclusively using [area handles](ABENAREA_HANDLE_GLOSRY.html). Each area instance version has a name. Area instance versions with the same name form an [area instance](ABENAREA_INSTANCE_GLOSRY.html).
