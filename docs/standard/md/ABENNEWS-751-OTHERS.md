[1\. Full Text Search in the ABAP Keyword Documentation for SAP GUI](#ABAP_MODIFICATION_1@4@)

[2\. Static ABAP with restricted object use](#ABAP_MODIFICATION_2@4@)

Double quotation marks (*"*) can now be used in the input field of the documentation display in the SAP GUI version of the ABAP keyword documentation to force a full text search for a search term. Up until now, the full text search was only available by choosing radio buttons in the dialog box of the documentation.

In ABAP release 7.51, a new [ABAP version](ABENABAP_VERSION_GLOSRY.html)\\ [Static ABAP with restricted object use](ABENABAP_STATIC_RESTR_GLOSRY.html) was introduced. The internal version ID in column `UCCHECK` of system table `TRDIR` is *3*.
