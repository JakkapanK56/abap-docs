Macros enable [source code modularization](ABENSOURCE_CODE_MODULARIZAT_GLOSRY.html) within an ABAP program. They have the following properties:

[Only use macros in exceptional cases.](ABENMACROS_GUIDL.html)

-   Defined between the statements [`DEFINE`](ABAPDEFINE.html) and [`END-OF-DEFINITION`](ABAPEND-OF-DEFINITION.html) or in database table `TRMAC`.
-   They are [included](ABENMACRO_INCLUDE.html) by specifying their name.
