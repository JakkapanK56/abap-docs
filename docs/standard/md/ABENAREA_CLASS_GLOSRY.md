[Final](ABENFINAL_GLOSRY.html) global [class](ABENCLASS_GLOSRY.html) whose instances are used as [area handles](ABENAREA_HANDLE_GLOSRY.html) for [shared objects](ABENSHARED_OBJECTS_GLOSRY.html). The [attributes](ABENATTRIBUTE_GLOSRY.html) of the instances of an area class represent the properties of an [area](ABENAREA_GLOSRY.html). The [methods](ABENMETHOD_GLOSRY.html) of an area class are used to access the area. All area classes are [subclasses](ABENSUBCLASS_GLOSRY.html) of [`CL_ABAP_MEMORY_AREA`](ABENSHM_CL_ABAP_MEMORY_AREA.html). When creating areas, area classes of the same name are generated as subclasses of [`CL_SHM_AREA`](ABENSHM_CL_SHM_AREA.html). A special predefined area class is [`CL_IMODE_AREA`](ABENSHM_CL_IMODE_AREA.html), whose instance represents the area handle for the [internal session](ABENINTERNAL_SESSION_GLOSRY.html) of a program.
