Group of multiple [AS instances](ABENAS_INSTANCE_GLOSRY.html) of an [AS ABAP](ABENAS_ABAP_GLOSRY.html) that are provided to process asynchronous [remote function calls](ABENREMOTE_FUNCTION_CALL_GLOSRY.html) (RFC) concurrently. RFC server groups are created using [transaction code](ABENTRANSACTION_GLOSRY.html)\\ `RZ12`.
