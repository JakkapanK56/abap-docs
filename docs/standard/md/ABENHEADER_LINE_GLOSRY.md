Obsolete [work area](ABENWORK_AREA_GLOSRY.html) of an [internal table](ABENINTERNAL_TABLE_GLOSRY.html) whose type is the [line type](ABENROW_TYPE_GLOSRY.html) and has the same name as the internal table. When an internal table with a header line is used in an [operand position](ABENOPERAND_POSITION_GLOSRY.html), the header line is usually addressed. To force access to the [table body](ABENTABLE_BODY_GLOSRY.html), square brackets `[]` can be specified after the table name.
