BFS for short. [Repository object](ABENREPOSITORY_OBJECT_GLOSRY.html) that groups multiple [business functions](ABENBUSINESS_FUNCTION_GLOSRY.html) to form a unit that covers the requirements of an industry solution within the scope of [Switch Framework](ABENSWITCH_FRAMEWORK_GLOSRY.html). For each [AS ABAP](ABENAS_ABAP_GLOSRY.html) only one business function set can be activated. Within a business function set, it is ensured that all business functions are compatible with one another. Business function sets are created in [transaction](ABENTRANSACTION_GLOSRY.html)\\ `SFW3` and switched in transaction `SFW5`.
