``... [`element1`](ABENCDS_ELEMENT_APV.html), [`element2`](ABENCDS_ELEMENT_APV.html), ...``

Defines the [elements](ABENCDS_ELEMENT_APV.html) of a [CDS analytical projection view](ABENCDS_ANALYTICAL_PV_GLOSRY.html). The elements are specified in a comma-separated list in curly brackets \\{ \\}.

The names of the elements of an analytical projection view must be unique. These are either

-   the names of the elements taken from the data sources, or
-   alternative element names specified using `AS`.
