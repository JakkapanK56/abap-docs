[1\. Bulk Access in ADBC](#ABAP_MODIFICATION_1@4@)

[2\. Recommended Use](#ABAP_MODIFICATION_2@4@)

The new method `SET_PARAM_TABLE` of the class `CL_SQL_STATEMENT` enables bulk access in [ADBC](ABENADBC.html) (see the [executable example](ABENADBC_DML_DDL_BULK_ABEXA.html)).

New developments in [Native SQL](ABENNATIVE_SQL_GLOSRY.html) are now only possible in [ADBC](ABENADBC_GLOSRY.html), which means that ADBC is now recommended in new programs instead of the [static embedding](ABENNATIVESQL.html) of Native SQL.
