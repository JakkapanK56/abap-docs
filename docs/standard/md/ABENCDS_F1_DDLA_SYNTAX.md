Alongside the [DDL statements for data definitions](ABENCDS_F1_DDL_SYNTAX.html), the [CDS DDL](ABENCDS_DDL_GLOSRY.html) in ABAP CDS makes it possible to define [CDS annotations](ABENCDS_ANNOTATION_GLOSRY.html) as [CDS objects](ABENCDS_OBJECT_GLOSRY.html) in [`DDLA` source code](ABENDDLA_SOURCE_CODE_GLOSRY.html). The following statement is used here:

-   [`DEFINE ANNOTATION`](ABENCDS_F1_DEFINE_ANNOTATION.html)

-   The [`DDLA` source code](ABENDDL_SOURCE_CODE_GLOSRY.html) of annotation definitions has its own editor in [ADT](ABENADT_GLOSRY.html), which is documented there. `DDLA` source code can also be displayed in [Repository Browser](ABENREPOSITORY_BROWSER_GLOSRY.html) in [ABAP Workbench](ABENABAP_WORKBENCH_GLOSRY.html).
-   Two slashes (`//`) can be used as [comment characters](ABENCDS_GENERAL_SYNTAX_RULES.html) in [`DDLA` source code](ABENDDLA_SOURCE_CODE_GLOSRY.html) for annotation definitions, but not two dashes (`--`).
