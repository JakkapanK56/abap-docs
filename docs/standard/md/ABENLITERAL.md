In addition to the [named data objects](ABENNAMED_DATA_OBJECT_GLOSRY.html) that can be addressed by their names in ABAP programs (this includes the text symbols) and the [anonymous data objects](ABENANONYMOUS_DATA_OBJECT_GLOSRY.html) that are created by the statement [`CREATE DATA`](ABAPCREATE_DATA.html), there are also [literals](ABENABAP_LITERAL_GLOSRY.html) that are defined in the source code of a program by specifying a string that represents a value. Possible literals are:

The [literal operator](ABENLITERAL_OPERATOR.html)\\ `&` can be used to link multiple literals of the same type to a literal.

In general ABAP, only [untyped literals](ABENUNTYPED_LITERAL_GLOSRY.html) are available. In [ABAP SQL](ABENABAP_SQL_GLOSRY.html)\\ [typed literals](ABENTYPED_LITERAL_GLOSRY.html) can and should be used as well.
