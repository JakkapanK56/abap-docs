The CDS language elements described in this subnode are obsolete and are only available for reasons of compatibility with older releases. These statements may still be encountered in older repository objects but should no longer be used in new repository objects.

There are replacement constructs for all obsolete language elements, which improve on their predecessor version.

-   [Obsolete CDS Entities](ABENCDS_ENTITIES_OBSOLETE.html)
-   [Obsolete Access to CDS Entities](ABENCDS_ACCESS_OBSOLETE.html)
-   [Obsolete CDS Annotations](ABENCDS_ANNOTATION_OBSOLETE.html)
-   [Obsolete Client Handling in CDS Table Functions](ABENCDS_FUNC_CLIENT_HANDLING_OBS.html)
