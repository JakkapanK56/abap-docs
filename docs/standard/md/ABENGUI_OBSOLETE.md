Various UI technologies can be used in the ABAP context:

If possible, [SAPUI5](ABENSAPUI5_GLOSRY.html) or [Web Dynpro](ABENWEB_DYNPRO_GLOSRY.html) should be used. If classic technologies are still used for some reason, the following constructs must be avoided:

-   Classic SAP GUI technologies

-   [Classic dynpros](ABENABAP_DYNPROS.html)
-   [Selection screens](ABENSELECTION_SCREEN.html)
-   [Classic lists](ABENABAP_DYNPRO_LIST.html)

-   Web-based technologies

-   [Business Server Pages](ABENBUSINESS_SERVER_PAGES_GLOSRY.html)
-   [Web Dynpro ABAP](ABENWEB_DYNPRO_GLOSRY.html)
-   [SAPUI5](ABENSAPUI5_GLOSRY.html)

-   [Obsolete statements in dynpro flow logic](ABENDYNPRO_OBSOLET.html)
-   [Obsolete ABAP statements for dynpros](ABENABAP_DYNPRO_OBSOLET.html)
-   [Obsolete statements in list processing](ABENLISTS_OBSOLETE.html)
-   [Obsolete editor calls](ABENTEXTEDITOR.html)
