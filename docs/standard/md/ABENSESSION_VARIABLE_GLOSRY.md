A global variable of a [database system](ABENDATABASE_SYSTEM_GLOSRY.html) that contains information about the current context. The [SAP HANA database](ABENHANA_DATABASE_GLOSRY.html) has special [ABAP-specific session variables](ABENHANA_SESSION_VARIABLES.html) whose content matches the value of certain [ABAP system fields](ABENSYSTEM_FIELD_GLOSRY.html) when accessed from ABAP. In [CDS views](ABENCDS_VIEW_GLOSRY.html), these ABAP-specific session variables can be accessed on all supported platforms.
