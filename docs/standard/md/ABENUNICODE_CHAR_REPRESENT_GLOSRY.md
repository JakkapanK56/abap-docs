[Character representation](ABENCHAR_REPRESENTATION_GLOSRY.html) defined for [Unicode](ABENUNICODE_GLOSRY.html). SAP supports the representations [UTF-8](ABENUTF8_GLOSRY.html), [UTF-16](ABENUTF16_GLOSRY.html), and [UTF-32](ABENUTF32_GLOSRY.html), and the [byte order](ABENBYTE_ORDER_GLOSRY.html) can be platform-dependent. UTF-16 is the [system code page](ABENSYSTEM_CODEPAGE_GLOSRY.html) of a [Unicode system](ABENUNICODE_SYSTEM_GLOSRY.html). The programming language ABAP supports the subset [UCS-2](ABENUCS2_GLOSRY.html) without [surrogates](ABENSURROGATES_GLOSRY.html).
