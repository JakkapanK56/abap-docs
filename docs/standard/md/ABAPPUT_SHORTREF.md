[Reference](ABAPPUT.html)

`PUT \{ node | <node>\ \}.`

**Obsolete:** Raises the event `GET node` in a [logical database](ABENLOGICAL_DATA_BASE_GLOSRY.html), if data is available in the [table work area](ABENTABLE_WORK_AREA_GLOSRY.html) of the node `node`. For node type A, a field symbol `<node>` is specified which is assigned a data object of the data type that is required in the statement `NODES` in the linked [executable program](ABENEXECUTABLE_PROGRAM_GLOSRY.html).
