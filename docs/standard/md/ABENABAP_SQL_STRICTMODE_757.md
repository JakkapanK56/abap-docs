The strict mode of the syntax check in ABAP release 7.57 applies to all ABAP SQL statements that use one of the following features introduced in [ABAP release 7.57](ABENNEWS-757-ABAP_SQL.html):

The strict mode in ABAP release 7.57 covers all rules of the [strict mode in ABAP release 7.56](ABENABAP_SQL_STRICTMODE_756.html).

-   Usage of [`ORDER BY n`](ABAPORDERBY_CLAUSE.html).
