[Reference](ABAPTYPES_OCCURS.html)

`TYPES dtype \{\ \{TYPE [REF TO] type\}\              |\ \{LIKE [REF TO] dobj\}\ \} OCCURS n.`

**Obsolete:** Declares a [standard table](ABENSTANDARD_TABLE_GLOSRY.html) type with a [standard key](ABENSTANDARD_KEY_GLOSRY.html) and the [initial memory requirement](ABENINITIAL_MEM_REQ_GLOSRY.html)\\ `n`.

-   `TYPE`\\
    Defines the line type with a reference to a data type.
-   `LIKE`\\
    Defines the line type with a reference to a data object.
-   `REF TO`\\
    Creates the line type as a reference type.
