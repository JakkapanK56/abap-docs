[Object reference variable](ABENOBJECT_REFER_VARIABLE_GLOSRY.html) of the [static type](ABENSTATIC_TYPE_GLOSRY.html) of an [interface](ABENOO_INTF_GLOSRY.html). An interface reference variable can be used to access the visible [interface components](ABENINTERFACE_COMPONENT_GLOSRY.html) of an object that exists in the corresponding interface. This applies to both dynamic and static access.
