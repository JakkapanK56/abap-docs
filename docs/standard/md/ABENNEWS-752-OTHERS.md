[2\. Standard ABAP with restricted object use](#ABAP_MODIFICATION_2@4@)

[3\. Surrogate Areas in iXML Library](#ABAP_MODIFICATION_3@4@)

In the documentation display of the SAP GUI version of the ABAP keyword documentation, the search for a search term can be restricted to the current subtree. This option can be selected from the context menu of a node or in the dialog window of the documentation.

In ABAP release 7.52, a new [ABAP version](ABENABAP_VERSION_GLOSRY.html)\\ [Standard ABAP with Restricted Object Use](ABENSTANDARD_ABAP_RESTR_OBJ_GLOSRY.html) was introduced. The internal version ID in column `UCCHECK` of system table `TRDIR` is *4*.

The iXML Library now supports [surrogate areas](ABENABAP_IXML_LIB.html) with the same problems as in [ABAP](ABENCHARACTER_SETS.html).
