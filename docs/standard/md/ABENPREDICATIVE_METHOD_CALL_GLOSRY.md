Call of a [functional method](ABENFUNCTIONAL_METHOD_GLOSRY.html) as the only [ABAP operand](ABENABAP_OPERAND_GLOSRY.html) of a [relational expression](ABENRELATIONAL_EXPRESSION_GLOSRY.html). The result of the relational expression is true if the result of the [functional method call](ABENFUNCTIONAL_METHOD_CALL_GLOSRY.html) is not initial. See also [predicate method](ABENPREDICATE_METHOD_GLOSRY.html).
