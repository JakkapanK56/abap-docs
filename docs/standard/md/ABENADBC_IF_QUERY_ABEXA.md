This example demonstrates the execution of a [query](ABENADBC_IF_QUERY.html) with ADBC.

Using methods of the statement builder, three columns of the database table `SFLIGHT` are read for the [current client](ABENCURRENT_CLIENT_GLOSRY.html). An appropriate internal table is bound to the result set using the method `SET_OUTPUT_PARAMS_BY_ITAB` of the interface `IF_SQL_RESULT_SET`. Using the method `NEXT_PACKAGE_GET_ROWS_PRCSSD`, all the rows of the result set are transported to the internal table.

Using the `LITERAL` method for user input, quotation marks are added automatically to prevent [SQL injections](ABENSQL_INJECTION_GLOSRY.html).
