[Chaining](ABENCHAINING_GLOSRY.html) in which multiple [functional methods](ABENFUNCTIONAL_METHOD_GLOSRY.html) or [constructor expressions](ABENCONSTRUCTOR_EXPRESSION_GLOSRY.html) are linked via [component selectors](ABENCOMPONENT_SELECTOR_GLOSRY.html). The [return values](ABENRETURN_VALUE_GLOSRY.html) or results of the individual methods or expressions are each references to the next methods, respectively, and the return value of the last method references an [attribute](ABENATTRIBUTE_GLOSRY.html) that is used as an [ABAP operand](ABENABAP_OPERAND_GLOSRY.html). See also [chained identifiers](ABENCHAINED_NAME_GLOSRY.html).
