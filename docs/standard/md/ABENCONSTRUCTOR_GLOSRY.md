A special [method](ABENMETHOD_GLOSRY.html) in a [class](ABENCLASS_GLOSRY.html) that sets a defined initial value for the status ([attributes](ABENATTRIBUTE_GLOSRY.html)) of the class or its [objects](ABENOBJECT_GLOSRY.html). There is an [instance constructor](ABENINSTANCE_CONSTRUCTOR_GLOSRY.html) for [instance attributes](ABENINSTANCE_ATTRIBUTE_GLOSRY.html) and a [static constructor](ABENSTATIC_CONSTRUCTOR_GLOSRY.html) for [static attributes](ABENSTATIC_ATTRIBUTE_GLOSRY.html). The constructors always exist implicitly but nevertheless must be declared explicitly for separate implementations. Special [constructor expressions](ABENCONSTRUCTOR_EXPRESSION_GLOSRY.html) also exist.
