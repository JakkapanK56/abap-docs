This example loads an Excel file from the presentation server and converts it into an internal table.

After a file is loaded with the method `GUI_UPLOAD` of the class `CL_GUI_FRONTEND_SERVICES`, an attempt is made to handle it as an Excel file and to convert it into an internal table.
