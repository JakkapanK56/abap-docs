This example demonstrates an automatically started mini [ABAP daemon](ABENABAP_DAEMON_GLOSRY.html).

The class `CL_DEMO_ABAP_MINI_DAEMON_AUTO` used in the above example is the same as `CL_DEMO_ABAP_MINI_DAEMON` from the example [Simple ABAP Daemon](ABENABAP_MINI_DAEMON_ABEXA.html). In contrast to that class, the repository object `CL_DEMO_ABAP_MINI_DAEMON_AUTO` has the property *Automatically Started Daemon*. The daemon can be administered to start automatically at given point of times, for example each time an application server is instantiated. In the calling class above, such an automatic start is simulated in the method `SETUP` by creating the necessary administration data and using them to start the daemon. A method `TEARDOWN` deletes the administration data again.
