This example demonstrates complex case distinctions in [SQL expressions](ABAPSQL_EXPR.html).

In a list of columns specified after `SELECT`, [`CASE`](ABENSQL_SIMPLE_CASE.html) is used to make a complex case distinction for the content of two columns. The result is a string taken from a [host variable](ABENABAP_SQL_HOST_VARIABLES.html). Using the alias name defined after `AS`, the result is assigned to the identically named column in an internal table declared inline, `results`. [`GROUP BY`](ABAPLOOP_AT_ITAB_GROUP_BY.html) is used to group this table by the results column.
