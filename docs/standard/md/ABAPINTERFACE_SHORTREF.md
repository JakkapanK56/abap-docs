`INTERFACE intf.    [components]\  ENDINTERFACE.`

Declares an interface `intf`. In the declaration, the `components` of the interface are declared with`ALIASES`, `[CLASS-]DATA`, `[CLASS-]METHODS` and `[CLASS-]EVENTS`.
