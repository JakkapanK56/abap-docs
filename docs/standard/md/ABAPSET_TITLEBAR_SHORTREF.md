[Reference](ABAPSET_TITLEBAR_DYNPRO.html)

 `SET TITLEBAR title [OF PROGRAM prog]\                     [WITH text1 ... text9].`

Sets the [GUI title](ABENGUI_TITLE_GLOSRY.html).

-   [`OF PROGRAM prog`](ABAPSET_TITLEBAR_DYNPRO.html)\\
    Specifies the program `prog` in which the GUI title is defined.
-   [`WITH text1 ... text9`](ABAPSET_TITLEBAR_DYNPRO.html)\\
    Replaces the placeholders for the GUI title with the content of the data objects `text1`, ..., `text9`.
