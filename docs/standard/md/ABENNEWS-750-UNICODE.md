From ABAP release 7.50, AS ABAP can only run as a [Unicode system](ABENUNICODE_SYSTEM_GLOSRY.html). Systems with non-Unicode code pages are no longer supported. Therefore, all runnable ABAP programs have the [ABAP language version](ABENABAP_VERSION_GLOSRY.html)\\ [Standard ABAP (Unicode)](ABENSTANDARD_ABAP_GLOSRY.html) or higher. Programs with the language version [Obsolete ABAP (Non-Unicode)](ABENNON_UC_PROG_GLOSRY.html) can no longer be executed. The syntax check can now only run using the rules from the [Unicode check](ABENUNICODE_CHECK_GLOSRY.html). The [ABAP language version](ABENABAP_VERSIONS.html)\\ [Obsolete ABAP (Non-Unicode)](ABENNON_UC_PROG_GLOSRY.html) can no longer be used.
