Checks whether the current [ABAP user](ABENABAP_USER_GLOSRY.html) has a specific [authorization](ABENAUTHORIZATION_GLOSRY.html). For each [authorization field](ABENAUTHORIZATION_FIELD_GLOSRY.html) of an [authorization object](ABENAUTHORIZATION_OBJECT_GLOSRY.html), the system checks a value against the corresponding entries in the [user master record](ABENUSER_MASTER_RECORD_GLOSRY.html). [Check indicators](ABENCHECK_INDICATOR_GLOSRY.html) control whether an authorization check takes place. Authorization checks are either implicit when programs are called, or explicit when the [ABAP statement](ABENABAP_STATEMENT_GLOSRY.html)\\ [`AUTHORITY-CHECK`](ABAPAUTHORITY-CHECK.html) is used.
