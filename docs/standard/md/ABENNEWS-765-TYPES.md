The statement [`TYPES BEGIN OF ENUM`](ABAPTYPES_ENUM.html) can be used to define [enumerated types](ABENENUMERATED_TYPE_GLOSRY.html) for [enumerated variables](ABENENUMERATED_VARIABLE_GLOSRY.html), which can have only [enumerated values](ABENENUMERATED_VALUE_GLOSRY.html) defined for this type.

The document [Enumerated Objects](ABENENUMERATED_TYPES_USAGE.html) summarizes the properties of enumerated types and enumerated objects.
