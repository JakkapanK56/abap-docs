Object of the [class](ABENCLASS_GLOSRY.html)\\ `CL_ABAP_TIMER_MANAGER` that is accessed using the [interface](ABENOO_INTF_GLOSRY.html)\\ `IF_ABAP_TIMER_MANAGER`. A timer manager allows an [ABAP Timer](ABENABAP_TIMER_GLOSRY.html) to be started and stopped in [non-blocking mode](ABENNON_BLOCKING_MODEL_GLOSRY.html). When the timer is started, a runtime and an [ABAP Timer handler](ABENABAP_TIMER_HANDLER_GLOSRY.html) are defined.
