[Data object](ABENDATA_OBJECT_GLOSRY.html) of the [date type](ABENDATE_TYPE_GLOSRY.html)\\ [`d`](ABENBUILTIN_TYPES_DATE_TIME.html) for storing a character-like [calendar date](ABENCALENDAR_DATE_GLOSRY.html). Valid values have the format *yyyymmdd*. The internal representation takes place in the [character representation](ABENCHAR_REPRESENTATION_GLOSRY.html) of the current [code page](ABENCODEPAGE_GLOSRY.html). When assignments are made to data objects of [numeric data types](ABENNUMERIC_DATA_TYPE_GLOSRY.html) and in calculations, valid values are converted to the number of days since 01.01.0001.
