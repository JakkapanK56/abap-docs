[Repository object](ABENREPOSITORY_OBJECT_GLOSRY.html) that can be used to group and activate [activatable checkpoints](ABENACTIVATABLE_CHECKPOINT_GLOSRY.html). To create checkpoint groups and maintain the activation settings, use [transaction](ABENTRANSACTION_GLOSRY.html)\\ `SAAB`. The activation settings of a checkpoint group are valid for all checkpoints assigned to the group. Settings for a set of checkpoint groups can be created using [activation variants](ABENACTIVATION_VARIANT_GLOSRY.html).
