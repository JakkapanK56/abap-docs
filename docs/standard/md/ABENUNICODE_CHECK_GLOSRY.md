[Syntax check](ABENSYNTAX_CHECK_GLOSRY.html) for [ABAP programs](ABENABAP_PROGRAM_GLOSRY.html) that are to be executed in a [Unicode system](ABENUNICODE_SYSTEM_GLOSRY.html). The Unicode check is carried out for [ABAP language versions](ABENABAP_VERSION_GLOSRY.html)\\ [Standard ABAP](ABENSTANDARD_ABAP_GLOSRY.html) or higher. Only Unicode systems are supported in the current [AS ABAP](ABENAS_ABAP_GLOSRY.html) release and therefore no programs of the obsolete ABAP language version [Obsolete ABAP (Non-Unicode)](ABENNON_UNICODE_ABAP_GLOSRY.html) are supported.
