The statement [`SET MARGIN`](ABAPSET_MARGIN.html) can be used to define the upper and left margins of a spool list.

When spooling is active, the statement [`PRINT-CONTROL`](ABAPPRINT-CONTROL.html) can be used to write formatting information for optical archiving to the spool system.

When spooling is active, an [obsolete variant](ABAPPRINT-CONTROL_OBSOLETE.html) of the statement `PRINT-CONTROL` can be used to write control statements for output devices (printers) to the spool system.
