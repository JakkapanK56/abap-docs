ADF for short. A framework based on [classes](ABENCLASS_GLOSRY.html) and [interfaces](ABENOO_INTF_GLOSRY.html) for using [ABAP daemons](ABENABAP_DAEMON_GLOSRY.html). ADF includes APIs both for communication with ABAP daemons in [ABAP programs](ABENABAP_PROGRAM_GLOSRY.html) and for handling events using ABAP daemons in [ABAP daemon sessions](ABENABAP_DAEMON_SESSION_GLOSRY.html).
