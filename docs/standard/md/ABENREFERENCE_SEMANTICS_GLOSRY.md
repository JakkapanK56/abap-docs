Reference semantics means that the [reference](ABENREFERENCE_GLOSRY.html) itself is used when accessing a [deep](ABENDEEP_GLOSRY.html)\\ [data object](ABENDATA_OBJECT_GLOSRY.html). In ABAP, reference semantics apply to [assignments](ABENASSIGNMENT_GLOSRY.html) and [comparisons](ABENCOMPARISON_GLOSRY.html) of [reference variables](ABENREFERENCE_VARIABLE_GLOSRY.html) but not when accessing [dynamic data objects](ABENDYNAMIC_DATA_OBJECT_GLOSRY.html) or [field symbols](ABENFIELD_SYMBOL_GLOSRY.html). See also [value semantics](ABENVALUE_SEMANTICS_GLOSRY.html).
