Must be specified when the statement [`MESSAGE`](ABAPMESSAGE.html) is used to send a [message](ABENMESSAGE_GLOSRY.html). It determines the representation of the message in the [SAP GUI](ABENSAP_GUI_GLOSRY.html) and the subsequent program flow. The possible message types are [information message](ABENINFORMATION_MESSAGE_GLOSRY.html), [status message](ABENSTATUS_MESSAGE_GLOSRY.html), [error message](ABENERROR_MESSAGE_GLOSRY.html), [warning](ABENWARNING_GLOSRY.html), [termination message](ABENTERMINATION_MESSAGE_GLOSRY.html), and [exit message](ABENEXIT_MESSAGE_GLOSRY.html).
