Pseudo comments are [program directives](ABENPROGRAM_DIRECTIVE_GLOSRY.html) for managing checks and test runs. Pseudo comments have mostly become [obsolete](ABENPSEUDO_COMMENTS_OBSOLETE.html) and have been replaced by [pragmas](ABENPRAGMA.html) or real additions.

Only one pseudo comment can be specified per program line. To specify multiple pseudo comments for a statement, they must be split into multiple lines.

-   [Pseudo Comments for Code Inspector](ABENPSEUDO_COMMENT_CI.html)

-   [Pseudo Comments for the Extended Program Check](ABENPSEUDO_COMMENT_SLIN.html)
-   [Pseudo Comments for Test Classes](ABENPSEUDO_COMMENT_TEST_CLASS.html)
