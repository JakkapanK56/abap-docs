[Assignment](ABENASSIGNMENT_GLOSRY.html) between two non-[compatible](ABENCOMPATIBLE_GLOSRY.html)\\ [data objects](ABENDATA_OBJECT_GLOSRY.html), where the source field contains a valid value, the [conversion](ABENCONVERSION_GLOSRY.html) is lossless, and the target field contains a valid value after the assignment. The [lossless operator](ABENLOSSLESS_OPERATOR_GLOSRY.html)\\ [`EXACT`](ABENCONSTRUCTOR_EXPRESSION_EXACT.html) and the addition `EXACT` of the obsolete statement [`MOVE`](ABAPMOVE_OBS.html)\\ [](ABENASSIGNMENT_GLOSRY.html)can be used to check whether an assignment is lossless.
