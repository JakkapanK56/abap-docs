Indivisible and non-nestable modularization unit of an [ABAP program](ABENABAP_PROGRAM_GLOSRY.html). Possible processing blocks are [procedures](ABENPROCEDURE_GLOSRY.html), [dialog modules](ABENDIALOG_MODULE_GLOSRY.html), and [event blocks](ABENEVENT_BLOCK_GLOSRY.html). They contain statements that are structured into [statement blocks](ABENSTATEMENT_BLOCK_GLOSRY.html) by [control structures](ABENCONTROL_STRUCTURE_GLOSRY.html). Each non-declarative statement in an ABAP program is part of a processing block.
