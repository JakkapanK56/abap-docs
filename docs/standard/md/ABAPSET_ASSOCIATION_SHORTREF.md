`SET ASSOCIATION mesh_path \{ = wa \}\                          |\ \{ LIKE wa    \}\                          |\ \{ INITIAL \}.`

Sets associations for a [mesh path](ABENMESH_PATH_GLOSRY.html).

-   `= wa`\\
    Sets key values for follow-on nodes.
-   `LIKE wa`\\
    Sets key values from initial nodes.
-   `INITIAL`\\
    Sets initial values.
