Property of an [arithmetic expression](ABENARITHMETIC_EXPRESSION_GLOSRY.html) depending on the [numeric data types](ABENNUMERIC_DATA_TYPE_GLOSRY.html) involved. The calculation type determines the calculation rule used to evaluate an arithmetic expression. There are different rules for the possible calculation types `i`, `int8`, `p`, `decfloat34`, and `f`.
