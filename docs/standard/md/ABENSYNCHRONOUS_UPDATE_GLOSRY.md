[Update](ABENUPDATE_GLOSRY.html) in which [update function modules](ABENUPDATE_FUNCTION_MODULE_GLOSRY.html) of high priority are executed in the update work process whereby the calling program waits for completion. Synchronous updates are started via [`COMMIT WORK`](ABAPCOMMIT.html) with the addition `AND WAIT` and are executed in a separate [update session](ABENUPDATE_SESSION_GLOSRY.html).
