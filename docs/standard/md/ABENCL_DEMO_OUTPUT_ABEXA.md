The class shows the output produced by [`CL_DEMO_OUTPUT`](ABENCL_DEMO_OUTPUT.html) for some ABAP data types. The output mode can be entered. More usage of `CL_DEMO_OUTPUT` can be found in many examples of the ABAP Keyword Documentation.
