This example demonstrates the execution of a [query](ABENADBC_QUERY.html) with ADBC.

The method `EXECUTE_QUERY` from the class `CL_SQL_STATEMENT` is used to read three columns of the database table `SFLIGHT` for the [current client](ABENCURRENT_CLIENT_GLOSRY.html). An appropriate internal table is bound to the result set using the method `SET_PARAM_TABLE` of the class `CL_SQL_RESULT_SET`. Using the method `NEXT_PACKAGE`, all the rows of the result set are transported to the internal table.

Quotation marks are escaped in the user input to prevent [SQL injections](ABENSQL_INJECTION_GLOSRY.html).
