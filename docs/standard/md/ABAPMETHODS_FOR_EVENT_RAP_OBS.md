`METHODS ... FOR EVENT ... FOR ...`\\ 

Obsolete declaration of [ABP handler methods](ABENABP_HANDLER_METHOD_GLOSRY.html) in the [handler class](ABENABP_HANDLER_CLASS_GLOSRY.html) of an [ABAP behavior pool](ABENBEHAVIOR_POOL_GLOSRY.html).

The declaration [`METHODS ... FOR ENTITY EVENT ... FOR ...`](ABENMETHOD_FOR_ENTITY_EVENT.html) should be used.
