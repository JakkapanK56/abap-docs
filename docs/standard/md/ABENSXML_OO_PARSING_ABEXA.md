Simple [object-oriented](ABENABAP_SXML_LIB_PARSE_OO.html) reading of XML data.

The example has the same basic functions as the [executable example](ABENSXML_PARSING_ABEXA.html) for token-based parsing. Instead of the method `NEXT_NODE`, however, the method `READ_NEXT_NODE` is used here, and its return value is analyzed. Depending on the current node type, a downcast is performed to a suitable interface and attributes and methods are used to read the attributes of the current node.
