An optional input parameter with the predefined name `REFVAL` can be created for the function modules for [conversion exits](ABENCONVERSION_EXITS.html). When a dynpro field of the type [`CURR`](ABENDDIC_BUILTIN_TYPES.html) or [`QUAN`](ABENDDIC_BUILTIN_TYPES.html) is converted, this parameter is given the value of a an associated reference field of the type [`CUKY`](ABENDDIC_BUILTIN_TYPES.html) or [`UNIT`](ABENDDIC_BUILTIN_TYPES.html) automatically If [`WRITE USING EDIT MASK`](ABAPWRITE_TO_OPTIONS.html) is used, the input parameter `REFVAL` is not filled.
