This example demonstrates the various date formats for embedded expressions.

The current date is displayed in all possible [country-specific date formats](ABENCOUNTRY_FORMATS.html) using the parameter [`COUNTRY`](ABAPCOMPUTE_STRING_FORMAT_OPTIONS.html). For this purpose, the utility methods `setup` and `teardown` insert temporary lines with the possible values of the column `DATFM` into the database table `T005X` and delete them again once they have been used.
