An ABAP Dictionary [search help](ABENSEARCH_HELP_GLOSRY.html) can be bound directly to a dynpro field. Here, the first parameter of the search help is assigned to the dynpro field. Only values from the proposal list of the screen can be transported.

Obsolete input helps on dynpros are possible using the additions [`VALUES`](FIELD_VALUE_SELECT.html) and [`SELECT`](FIELD_VALUE_SELECT.html) of the dynpro logic statement [`FIELD`](FIELD.html).
