A [repository object](ABENREPOSITORY_OBJECT_GLOSRY.html) that is the basis of [authorizations](ABENAUTHORIZATION_GLOSRY.html). An authorization object consists of up to ten [authorization fields](ABENAUTHORIZATION_FIELD_GLOSRY.html). The combination of authorization fields that represent data and activities is used for [authorization assignments](ABENAUTHORIZATION_ASSIGN_GLOSRY.html) and [authorization checks](ABENAUTHORIZATION_CHECK_GLOSRY.html). Authorization objects are grouped in authorization object classes. They are maintained using [transaction](ABENTRANSACTION_CODE_GLOSRY.html)\\ `SU21`.
