[Class](ABENCLASS_GLOSRY.html) of an [ABAP program](ABENABAP_PROGRAM_GLOSRY.html) that can be accessed statically from the ABAP program only. Local classes can be created in all [program types](ABENPROGRAM_TYPE_GLOSRY.html), with the exception of [interface pools](ABENINTERFACE_POOL_GLOSRY.html) and [type pools](ABENTYPE_POOL_GLOSRY.html). In particular, [class pools](ABENCLASS_POOL_GLOSRY.html) can contain local classes for use in the [global class](ABENGLOBAL_CLASS_GLOSRY.html) or by frameworks. All local classes of an ABAP program share the same namespace together with the [local interfaces](ABENLOCAL_INTERFACE_GLOSRY.html) and the program-global [data types](ABENDATA_TYPE_GLOSRY.html) declared with [`TYPES`](ABAPTYPES.html).
