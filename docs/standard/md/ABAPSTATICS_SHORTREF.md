`STATICS dobj options.`

Declares a static local data object `dobj` in a [subroutine](ABENSUBROUTINE_GLOSRY.html), a [function module](ABENFUNCTION_MODULE_GLOSRY.html), or a [static method](ABENSTATIC_METHOD_GLOSRY.html).

-   `options`\\
    As in [`DATA`](ABAPDATA_SHORTREF.html), with the exception that the addition `READ-ONLY` cannot be specified.
