Eclipse-based tool for database objects of the [SAP HANA database](ABENHANA_DATABASE_GLOSRY.html), especially for [XSC](ABENXSC_GLOSRY.html) objects stored in the [SAP HANA Repository](ABENSAP_HANA_REPOSITORY_GLOSRY.html). In [XSA](ABENXSA_GLOSRY.html), SAP HANA Studio is replaced by the [SAP Web IDE for SAP HANA](ABENSAP_WEB_IDE_FOR_SAP_HAN_GLOSRY.html) and the [SAP HANA Cockpit](ABENSAP_HANA_COCKPIT_GLOSRY.html).
