The strict mode of the syntax check in ABAP release 7.80 applies to all ABAP SQL statements that use one of the following features introduced in [ABAP release 7.80](ABENNEWS-780-ABAP_SQL.html):

The strict mode in ABAP release 7.80 covers all rules of the [strict mode in ABAP release 7.79](ABENABAP_SQL_STRICTMODE_779.html).

-   Use of the SQL function [`TO_BLOB`](ABENSQL_TYPE_CONV_FUNC.html)
-   Use of [typed literals](ABENABAP_SQL_TYPED_LITERALS.html)
