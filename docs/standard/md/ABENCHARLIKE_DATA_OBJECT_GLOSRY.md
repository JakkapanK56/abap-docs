[Data object](ABENDATA_OBJECT_GLOSRY.html) that can be interpreted as character-like if its content is appropriate. Besides data objects with [character-like data types](ABENCHARLIKE_DATA_TYPE_GLOSRY.html) or [date types](ABENDATE_TYPE_GLOSRY.html) and [time types](ABENDATE_TYPE_GLOSRY.html), [structures](ABENSTRUCTURE_GLOSRY.html) with purely [flat](ABENFLAT_GLOSRY.html) character-like [structure components](ABENSTRUCTURE_COMPONENT_GLOSRY.html) can also be character-like in many [operand positions](ABENOPERAND_POSITION_GLOSRY.html).
