[Interface](ABENOO_INTF_GLOSRY.html) of an [ABAP program](ABENABAP_PROGRAM_GLOSRY.html) that can be accessed statically in this program only. Local interfaces can be created in all [program types](ABENPROGRAM_TYPE_GLOSRY.html) except for [interface pools](ABENINTERFACE_POOL_GLOSRY.html) and [type pools](ABENTYPE_POOL_GLOSRY.html). All local interfaces of an ABAP program share the same namespace together with the [local classes](ABENLOCAL_CLASS_GLOSRY.html) and the program-global [data types](ABENDATA_TYPE_GLOSRY.html) declared using [`TYPES`](ABAPTYPES.html).
