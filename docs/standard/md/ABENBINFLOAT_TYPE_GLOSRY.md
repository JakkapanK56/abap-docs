[Numeric data type](ABENNUMERIC_DATA_TYPE_GLOSRY.html) used for the binary representation of numbers with mantissa and exponents. The associated [built-in ABAP type](ABENBUILTIN_ABAP_TYPE_GLOSRY.html) is [`f`](ABENBUILTIN_TYPES_NUMERIC.html). The [built-in DDIC type](ABENBUILTIN_DDIC_TYPE_GLOSRY.html) is [`FLTP`](ABENDDIC_BUILTIN_TYPES.html#@@ITOC@@ABENDDIC_BUILTIN_TYPES_2). The internal representation is in accordance with the norm IEEE-754 (double precision) by one bit for the plus/minus sign, 11 bits for the binary exponent, and 52 bits for the mantissa, whereby [decimal numbers](ABENDECIMAL_NUMBER_GLOSRY.html) can be represented by 17 [places](ABENPLACE_GLOSRY.html). A [data object](ABENDATA_OBJECT_GLOSRY.html) of a binary floating point number type is a [binary floating point number](ABENBINFLOAT_GLOSRY.html).
