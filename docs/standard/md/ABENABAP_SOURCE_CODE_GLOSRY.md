[Source code](ABENSOURCE_CODE_GLOSRY.html) for [ABAP programs](ABENABAP_PROGRAM_GLOSRY.html) consisting of [ABAP statements](ABENABAP_STATEMENT_GLOSRY.html) and [comments](ABENCOMMENT_GLOSRY.html). Created and maintained in the [ABAP Editor](ABENABAP_EDITOR_GLOSRY.html) or with the [ABAP development tools for Eclipse (ADT)](ABENADT_GLOSRY.html). ABAP source code is compiled by the [ABAP Compiler](ABENABAP_COMPILER_GLOSRY.html) when the program is activated. It can be broken down into individual [repository objects](ABENREPOSITORY_OBJECT_GLOSRY.html) by using [include programs](ABENINCLUDE_PROGRAM_GLOSRY.html).
