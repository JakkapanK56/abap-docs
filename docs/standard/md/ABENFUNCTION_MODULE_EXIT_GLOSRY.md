[Customer exit](ABENCUSTOMER_EXIT_GLOSRY.html) for [ABAP programs](ABENABAP_PROGRAM_GLOSRY.html) in the form of empty or partially implemented [function modules](ABENFUNCTION_MODULE_GLOSRY.html) delivered with the standard SAP system. Integrated into ABAP programs using the statement [`CALL CUSTOMER-FUNCTION`](ABAPCALL_CUSTOMER-FUNCTION.html). The function module [interfaces](ABENINTF_GEN_GLOSRY.html) are predefined by SAP. The implementation can be empty or partially predefined. The customer must fully implement and activate them using [transaction](ABENTRANSACTION_GLOSRY.html)\\ `CMOD`. Function module exits should be replaced by [BAdIs](ABENBADI_GLOSRY.html).
