[AS ABAP](ABENAS_ABAP_GLOSRY.html) represents the application layer of an [ABAP system](ABENABAP_SYSTEM_GLOSRY.html) in which [ABAP programs](ABENABAP_PROGRAM_GLOSRY.html) are executed.
