[Include program](ABENINCLUDE_PROGRAM_GLOSRY.html) used for the [global declaration part](ABENGLOBAL_DECLARATION_SECT_GLOSRY.html) of an [ABAP program](ABENABAP_PROGRAM_GLOSRY.html). No [processing blocks](ABENPROCESSING_BLOCK_GLOSRY.html) should be implemented in top includes. This is checked by the [extended program check](ABENEXTENDED_PROGRAM_CHECK_GLOSRY.html). The corresponding top include is included whenever an individual include program of an ABAP program is compiled.
