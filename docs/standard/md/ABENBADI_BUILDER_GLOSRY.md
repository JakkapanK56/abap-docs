[ABAP Workbench](ABENABAP_WORKBENCH_GLOSRY.html) tool that can be used to create and maintain [BAdIs](ABENBADI_GLOSRY.html). The BAdI Builder is the BAdI-specific variant of access to the [Enhancement Builder](ABENENHANCEMENT_BUILDER_GLOSRY.html). Previously, the BAdI Builder was used exclusively for the maintenance of classic BAdIs. Called using [transactions](ABENTRANSACTION_CODE_GLOSRY.html)\\ `SE18` (definition) and `SE19` (implementation).
