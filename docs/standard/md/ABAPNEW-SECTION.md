[Short Reference](ABAPNEW-SECTION_SHORTREF.html)

`NEW-SECTION.`

This statement, which is forbidden in classes, is used to create a new spool request, has the same function as the following statement, and is replaced by this statement:

`[NEW-PAGE](ABAPNEW-PAGE.html)\ [PRINT ON NEW-SECTION](ABAPNEW-PAGE_PRINT.html).`

The obsolete statement does not allow the mandatory [spool parameters](ABENPRINT_PARAMETERS.html) to be specified.
