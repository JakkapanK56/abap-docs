Combines two character-like [operands](ABENABAP_OPERAND_GLOSRY.html) in a [string expression](ABENSTRING_EXPRESSION_GLOSRY.html). The only string operator currently possible is the [concatenation operator](ABENCONCATENATION_OPERATOR_GLOSRY.html)\\ [`&&`](ABENSTRING_OPERATORS.html), which concatenates the two operands into one string.
