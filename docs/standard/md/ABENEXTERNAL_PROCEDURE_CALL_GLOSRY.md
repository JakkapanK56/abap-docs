[Procedure](ABENPROCEDURE_GLOSRY.html) call from a different ABAP program of the same [AS ABAP](ABENAS_ABAP_GLOSRY.html). The first time one of its procedures is called, the required program is loaded into the [internal session](ABENINTERNAL_SESSION_GLOSRY.html) of the calling program, if not already there, and, apart from [class pools](ABENCLASS_POOL_GLOSRY.html), the [event](ABENEVENT_GLOSRY.html)\\ [`LOAD-OF-PROGRAM`](ABAPLOAD-OF-PROGRAM.html) is raised. See also [program group](ABENPROGRAM_GROUP_GLOSRY.html).
