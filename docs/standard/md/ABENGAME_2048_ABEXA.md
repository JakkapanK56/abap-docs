Expression-oriented implementation of a 2048 game.

This example uses the implementation of a 2048 game to demonstrate the expression-oriented ABAP programming (as introduced in ABAP release 7.40. As a comparison, the program `DEMO_GAME_2048_700` shows a functionally identical implementation without expressions, which is restricted to mainly using language elements of ABAP release 7.00.

To play the game, use the arrow keys on the keyboard to move all tiles in one of the four directions.
