Also referred to as [output stream](ABENOUTPUT_STREAM_GLOSRY.html). Either a [data stream](ABENDATA_STREAM_GLOSRY.html) that is connected to a [streaming data sink](ABENSTREAMING_DATA_SINK_GLOSRY.html) or a [filter stream](ABENFILTER_STREAM_GLOSRY.html) that writes data to another stream. The direction of a writer stream always goes toward the data sink. In a writer stream data can only be written.
