The strict mode of the syntax check in ABAP release 7.66 applies to all ABAP SQL statements that use one of the following features introduced in [ABAP release 7.66](ABENNEWS-766-ABAP_SQL.html):

The strict mode in ABAP release 7.66 covers all rules of the [strict mode in ABAP release 7.65](ABENABAP_SQL_STRICTMODE_765.html).

-   Use of `ONE TO MANY` in [`LEFT OUTER JOIN`](ABAPSELECT_JOIN.html)
