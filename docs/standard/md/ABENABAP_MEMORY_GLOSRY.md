Memory area within each [ABAP session](ABENABAP_SESSION_GLOSRY.html) that can be accessed jointly by the programs of a [call sequence](ABENCALL_SEQUENCE_GLOSRY.html) using the statements [`EXPORT`](ABAPEXPORT_DATA_CLUSTER.html) and [`IMPORT`](ABAPIMPORT_DATA_CLUSTER.html), and whose data is persisted for the duration of the [top-level transaction](ABENTOP_LEVEL_TRANSACTION_GLOSRY.html) of a call sequence.
