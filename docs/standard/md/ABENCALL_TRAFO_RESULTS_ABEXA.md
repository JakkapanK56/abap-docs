This example demonstrates the possible XML targets of the statement [`CALL TRANSFORMATION`](ABAPCALL_TRANSFORMATION.html).

The [identity transformation](ABENID_TRAFO_GLOSRY.html)\\ `ID` is called for all possible XML targets and the result is displayed. The XML source is always the same byte string, which contains the XML data in JSON-XML format.
