The strict mode of the syntax check in ABAP release 9.13 applies to all ABAP SQL statements that use one of the following features introduced in [ABAP release 9.13](ABENNEWS-913-ABAP_SQL.html):

The strict mode in ABAP release 9.13 covers all rules of the [strict mode in ABAP release 9.12](ABENABAP_SQL_STRICTMODE_912.html).

-   Operands of type `FLTP` or `f` are used in a [decimal floating point expression](ABENSQL_ARITH.html) or integer types are used in a [binary floating expression](ABENSQL_ARITH.html).
