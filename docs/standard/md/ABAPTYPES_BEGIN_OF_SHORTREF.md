``TYPES BEGIN OF struc_type.    ...    TYPES   ... [BOXED].    [`INCLUDE ...`](ABAPINCLUDE_TYPE_SHORTREF.html)\    ...  DATA END OF struc_type.``

Defines a structured data type `struc_type`, whose components can be defined by any `TYPES` statements or copied by `INCLUDE` from other structures.

-   [`BOXED`](ABAPTYPES_BOXED.html)\\
    Declares a substructure as a static [boxed component](ABENSTATIC_BOX_GLOSRY.html).
