The following sections list the classes and interfaces of iXML Library according to subareas.

Not all object types in iXML Library include the interface `IF_IXML_UNKNOWN`. Reference variables of this type can point to all objects in iXML Library. The interface `IF_IXML_UNKNOWN` contains the method `QUERY_INTERFACE`, which can be used for special [downcasts](ABENIXML_DOWN_CASTS.html) within iXML Library.
