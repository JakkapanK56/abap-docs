[Method](ABENMETHOD_GLOSRY.html) of a [class](ABENCLASS_GLOSRY.html) declared using [`CLASS-METHODS`](ABAPCLASS-METHODS.html) that can be used independently of a class [instance](ABENINSTANCE_GLOSRY.html). Static methods can only access [static attributes](ABENSTATIC_ATTRIBUTE_GLOSRY.html) and [static events](ABENSTATIC_EVENT_GLOSRY.html) of their own class.
