This example demonstrates how the asXML format is specified for embedded expressions.

This class takes data objects of different types and creates the asXML formats shown under [Mapping of Elementary ABAP Types](ABENABAP_XSLT_ASXML_ELEMENTARY.html). To keep the code short, the string template is placed exceptionally in a [macro](ABENMACRO_GLOSRY.html).
