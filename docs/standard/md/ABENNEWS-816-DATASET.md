In the statement [`OPEN DATASET`](ABAPOPEN_DATASET.html), the addition [`MESSAGE`](ABAPOPEN_DATASET_ERROR_HANDLING.html) must be used, otherwise a syntax warning occurs. Behind `MESSAGE`, an [inline declaration](ABENINLINE_DECLARATION_GLOSRY.html) is possible now.
