[Processing block](ABENPROCESSING_BLOCK_GLOSRY.html) without local [data area](ABENDATA_AREA_GLOSRY.html), which can be defined in every [ABAP program](ABENABAP_PROGRAM_GLOSRY.html) except for [type pools](ABENTYPE_POOL_GLOSRY.html), [class pools](ABENCLASS_POOL_GLOSRY.html), and [interface pools](ABENINTERFACE_POOL_GLOSRY.html). An event block is processed when the corresponding [event](ABENEVENT_GLOSRY.html) of the [ABAP runtime framework](ABENABAP_RUNTIME_FRMWK_GLOSRY.html) is raised. It starts with an [event keyword](ABENEVENT_KEYWORD_GLOSRY.html) and ends at the start of the next processing block.
