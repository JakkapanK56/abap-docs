`DEFINE macro.    ... &1 ... &9 ...  END-OF-DEFINITION.`

Defines a [macro](ABENMACRO_GLOSRY.html)\\ `macro` constructed from ABAP statements between `DEFINE` and `END-OF-DEFINITION` and that can contain placeholders `&1`, ..., `&9`. The macro can be included after its definition by specifying its name in the source code.
