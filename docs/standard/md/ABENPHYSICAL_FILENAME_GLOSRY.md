Platform-dependent name of a [file](ABENFILE_GLOSRY.html) on the [AS instance](ABENAS_INSTANCE_GLOSRY.html)\\ [host computer](ABENHOST_COMPUTER_GLOSRY.html). The statements of the [ABAP file interface](ABENFILE_INTERFACE_GLOSRY.html) work directly with the physical file names. Physical file names can be linked to [logical file names](ABENLOGICAL_FILENAME_GLOSRY.html) to enable platform-independent and secure programming.
