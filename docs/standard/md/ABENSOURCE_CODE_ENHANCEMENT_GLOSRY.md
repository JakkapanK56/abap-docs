A modification-free [enhancement](ABENENHANCEMENT_GLOSRY.html) to [ABAP source code](ABENABAP_SOURCE_CODE_GLOSRY.html), where [source code plug-ins](ABENSOURCE_CODE_PLUGIN_GLOSRY.html) are inserted into the code at [enhancement options](ABENENHANCEMENT_POINT_GLOSRY.html) that are defined implicitly or explicitly using the statement [`ENHANCEMENT-POINT`](ABAPENHANCEMENT-POINT.html), or that replace an enhancement option that was explicitly defined using [`ENHANCEMENT-SECTION`](ABAPENHANCEMENT-SECTION.html) and [`END-ENHANCEMENT-SECTION`](ABAPEND-ENHANCEMENT-SECTION.html).
