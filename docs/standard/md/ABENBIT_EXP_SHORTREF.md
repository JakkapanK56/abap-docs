[Reference](ABAPCOMPUTE_BIT.html)

`... \{\ [BIT-NOT] operand1        [\{BIT-AND|BIT-OR|BIT-XOR\}\ [BIT-NOT] operand2        [\{BIT-AND|BIT-OR|BIT-XOR\}\ [BIT-NOT] operand3        ...  ]]\ \} ...`

Bit expression that calculates a binary result from operand `operand`. Bit expressions can be parenthesized explicitly using [`( )`](ABENBIT_BRACKETS.html) and can occur in the [read positions of certain statements](ABENOPERANDS_EXPRESSIONS.html).

-   [`BIT-NOT`](ABAPCOMPUTE_BIT.html)\\
    Negates an operand `operand`.
-   [`BIT-AND|BIT-OR|BIT-XOR`](ABENBIT_OPERATORS.html)\\
    Bit operators.
