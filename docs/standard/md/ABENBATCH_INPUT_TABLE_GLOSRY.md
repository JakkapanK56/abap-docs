[Internal table](ABENINTERNAL_TABLE_GLOSRY.html) of [line type](ABENROW_TYPE_GLOSRY.html)\\ `BDCDATA`, which contains the data for a [dialog transaction](ABENDIALOG_TRANSACTION_GLOSRY.html) in the same format as in a [batch input session](ABENBATCH_INPUT_SESSION_GLOSRY.html). Batch input tables can be specified after the addition `USING` of the statement [`CALL TRANSACTION`](ABAPCALL_TRANSACTION.html) when calling [transactions](ABENTRANSACTION_GLOSRY.html).
