This section describes the declaration of object types, data types, data objects and field symbols.

-   [Classes and Interfaces](ABENCLASSES_AND_INTERFACES.html)
-   [Data Types](ABENTYPES_STATEMENTS.html)
-   [Data Objects](ABENOBJECTS_STATEMENTS.html)
-   [Field Symbols](ABENABAP_FIELD_SYMBOLS.html)
-   [Special Declarations](ABENDECLARATIONS_SPCL.html)

-   The statements [`PARAMETERS`](ABAPPARAMETERS.html) and [`SELECT OPTIONS`](ABAPSELECT-OPTIONS.html) for the declaration of input fields on [selection screens](ABENSELECTION_SCREEN_GLOSRY.html) are also declarative statements but are listed separately in the corresponding sections.
-   In addition to the explicit options for declaring data objects listed here, there are also statements and additions that implicitly declare data objects, such as [`SELECTION-SCREEN COMMENT`](ABAPSELECTION-SCREEN_COMMENT.html). However, most of these statements are obsolete, for example [`SUMMING`](ABAPSUMMING.html).
