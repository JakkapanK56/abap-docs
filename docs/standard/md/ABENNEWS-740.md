[ABAP release](ABENABAP_RELEASE_GLOSRY.html) 7.40 is based on [7.0, EhP3 or 7.3, EhP1](ABENNEWS-703.html). Therefore, apart from a few enhancements, this release is based almost in full on 7.0, [EhP2](ABENNEWS-71.html). The changes in ABAP release 7.40 are delivered in support packages that are coupled with kernel releases.
