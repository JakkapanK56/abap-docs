From ABAP release 7.0, [ABAP Workbench](ABENABAP_WORKBENCH_GLOSRY.html) offers a new [ABAP Enhancement Framework](ABENENHANCEMENT_FRAMEWORK_GLOSRY.html) which enables [enhancements](ABENENHANCEMENT_GLOSRY.html) to be controlled by [switches](ABENSWITCH_GLOSRY.html). Enhancements to [ABAP source code](ABENABAP_SOURCE_CODE_GLOSRY.html) and using BAdIs are supported by the ABAP runtime framework.

[Implicit enhancement points](ABENIMPLICIT_ENH_POINTS.html) are also available in ABAP source code or parameter interfaces of procedures.

For enhancements using [BAdIs](ABENBADI_GLOSRY.html), the following new statements have been introduced:
