Data transfer technique that allows data sets to be transferred automatically to the [dynpros](ABENDYNPRO_GLOSRY.html) of [transactions](ABENTRANSACTION_GLOSRY.html), and thus to [AS ABAP](ABENAS_ABAP_GLOSRY.html). Batch input is controlled by [batch input sessions](ABENBATCH_INPUT_SESSION_GLOSRY.html). The processing of batch input sessions in batch input is partly implemented directly in the [ABAP kernel](ABENKERNEL_GLOSRY.html). See also [batch input tables](ABENBATCH_INPUT_TABLE_GLOSRY.html).
