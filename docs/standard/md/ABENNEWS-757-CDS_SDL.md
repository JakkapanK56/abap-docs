[1\. CDS Service Definition Extension](#ABAP_MODIFICATION_1@4@)

[2\. CDS Service Definition, Provider Contract](#ABAP_MODIFICATION_2@4@)

It is now possible to define [service definition extensions](ABENCDS_SERVICE_DEF_EXT_GLOSRY.html) in [CDS SDL](ABENCDS_SDL_GLOSRY.html) using the statement [`EXTEND SERVICE`](ABENSRVD_EXTEND_SERVICE.html).

The new statement [`PROVIDER CONTRACTS`](ABENSRVD_PROVIDER_CONTRACT.html) is now available for [CDS service definitions](ABENCDS_SERVICE_DEFINITION_GLOSRY.html). A provider contract defines the type of [service binding](ABENSERVICE_BINDING_GLOSRY.html) that is to be used for a service definition. The effect is that stricter syntax checks are applied.
