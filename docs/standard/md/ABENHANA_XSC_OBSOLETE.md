[SAP HANA XSA](ABENXSA_GLOSRY.html) is the successor of [SAP HANA XSC](ABENXSC_GLOSRY.html). The access methods designed specifically for XSC objects are not suitable for XSA objects.

The following access methods for XSC objects are obsolete:

This is also described in SAP Notes [3116165](https://launchpad.support.sap.com/#/notes/3116165) and [2465027](https://launchpad.support.sap.com/#/notes/2465027).

-   [DDIC external views](ABENEXTERNAL_VIEW_GLOSRY.html)
-   [Database procedure proxies](ABENDATABASE_PROC_PROXY_GLOSRY.html)
