AMHC for short. Special [HDI container](ABENHDI_CONTAINER_GLOSRY.html) managed by [AS ABAP](ABENAS_ABAP_GLOSRY.html). An AMHC is created and changed using transaction `SCTS_AMHC`. An AMHC and the contained [HDI objects](ABENHDI_OBJECT_GLOSRY.html) are connected to the [Change and Transport System](ABENCTS_GLOSRY.html) using [SAP HANA Transport for ABAP (HTA)](ABENHTA_GLOSRY.html). When an AMHC is created, a [logical HDI container](ABENLOGICAL_HDI_CONTAINER_GLOSRY.html) is defined and during its activation a physical HDI container is created in the [SAP HANA database](ABENHANA_DATABASE_GLOSRY.html). The logical HDI container can be a [logical schema](ABENLOGICAL_SCHEMA_GLOSRY.html) in an [AMDP macro](ABENAMDP_MACRO_GLOSRY.html) where it represents the [database schema](ABENDATABASE_SCHEMA_GLOSRY.html) of the physical HDI container.
