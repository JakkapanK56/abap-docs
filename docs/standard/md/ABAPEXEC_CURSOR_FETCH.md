`EXEC SQL.`\\ 
  `FETCH NEXT dbcur INTO ...`\\ 
`ENDEXEC.`

Uses an open database cursor `dbcur` to read data to the [host variables](ABAPEXEC_HOST.html) specified after [`INTO`](ABAPEXEC_INTO.html).
