[Method](ABENMETHOD_GLOSRY.html) that can handle an [event](ABENEVENT_GLOSRY.html) that was raised by [`RAISE EVENT`](ABAPRAISE_EVENT.html) in another method or in the same method. To ensure that an event handler handles a raised event, it must be registered with the statement [`SET HANDLER`](ABAPSET_HANDLER.html).
