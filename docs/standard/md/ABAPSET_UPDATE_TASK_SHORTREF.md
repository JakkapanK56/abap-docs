[Reference](ABAPSET_UPDATE_TASK_LOCAL.html)

`SET UPDATE TASK LOCAL.`

Determines that [update function modules](ABENUPDATE_FUNCTION_MODULE_GLOSRY.html) registered with `CALL FUNCTION ... IN UPDATE TASK` that have high priority are executed immediately in the current [work process](ABENWORK_PROCESS_GLOSRY.html) and within the current [database LUW](ABENDATABASE_LUW_GLOSRY.html) when the `COMMIT WORK` statement is executed.
