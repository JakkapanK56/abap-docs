The strict mode of the syntax check in ABAP release 7.71 applies to all ABAP SQL statements that use one of the following features introduced in [ABAP release 7.71](ABENNEWS-771-ABAP_SQL.html):

The strict mode in ABAP release 7.71 covers all rules of the [strict mode in ABAP release 7.70](ABENABAP_SQL_STRICTMODE_770.html).

-   Use of one of the new [date/time functions](ABENABAP_SQL_DATE_TIME_FUNCTIONS.html)\\ `TIMS_IS_VALID`, `TSTMP_IS_VALID`, `TSTMP_CURRENT_UTCTIMESTAMP`, `TSTMP_SECONDS_BETWEEN`, `TSTMP_ADD_SECONDS`, `TSTMP_TO_DATS`, `TSTMP_TO_TIMS`, `TSTMP_TO_DST`, or `DATS_TIMS_TO_TSTMP`.
