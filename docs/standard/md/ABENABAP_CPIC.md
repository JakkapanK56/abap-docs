This section describes cross-system communications between two ABAP programs which is implemented using the [`COMMUNICATION`](ABAPCOMMUNICATION.html) statement.

SDK for CPI-C is no longer supported. The CPI-C libraries and the documentation for programming are no longer delivered. For new programs, SDK and the libraries for [RFC](ABENRFC_GLOSRY.html) must be used. The API for CPI-C will remain for the time being to support existing programs and for internal technical purposes.
