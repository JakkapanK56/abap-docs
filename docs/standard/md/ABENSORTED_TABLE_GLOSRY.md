[Table category](ABENTABLE_CATEGORY_GLOSRY.html) of an [internal table](ABENINTERNAL_TABLE_GLOSRY.html) that is managed using a [primary table index](ABENPRIMARY_TABLE_INDEX_GLOSRY.html) and is always sorted according to its [primary table key](ABENPRIMARY_TABLE_KEY_GLOSRY.html). Sorting occurs according to the [comparison rules](ABENLOGEXP_RULES.html) for the data types of the key fields. The associated [generic data type](ABENGENERIC_DATA_TYPE_GLOSRY.html) is [`sorted table`](ABENBUILT_IN_TYPES_GENERIC.html).
