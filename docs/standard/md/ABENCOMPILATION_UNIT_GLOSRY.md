ABAP program that can be compiled as a standalone program. [Executable programs](ABENEXECUTABLE_PROGRAM_GLOSRY.html), [module pools](ABENMODUL_POOL_GLOSRY.html), [function pools](ABENFUNCTION_POOL_GLOSRY.html), [class pools](ABENCLASS_POOL_GLOSRY.html), [interface pools](ABENINTERFACE_POOL_GLOSRY.html), [subroutine pools](ABENSUBROUTINE_POOL_GLOSRY.html), and [type pools](ABENTYPE_POOL_GLOSRY.html) are compilation units. In contrast, [include programs](ABENINCLUDE_PROGRAM_GLOSRY.html) are not compilation units but must be included in a compilation unit in order to be compiled. Compilation units that are implicitly modularized in multiple include programs by the development tools are called [master programs](ABENMASTER_PROGRAM_GLOSRY.html).
