The programming language ABAP and the ABAP runtime framework are identical in ABAP release 7.0, EhP3 and 7.3, EhP1. ABAP release 7.3, EhP1 is a result of a renaming of ABAP release 7.0, EhP3. In both cases, the name of the AS ABAP (`sy-saprl`) is AS ABAP 7.31. With the exception of the following extensions, this corresponds to ABAP release 7.0, EhP2.

With respect to the ABAP and ABAP runtime framework, ABAP release 7.31 is not the successor to ABAP release 7.3 from the track 7.1, 7.2, 7.3. The following ABAP release 7.3 features are not included:

-   [ABAP Objects in ABAP Release 7.0, EhP3 and 7.3, EhP1](ABENNEWS-703-ABAP-OBJECTS.html)
-   [Selection Screens and Lists in ABAP Release 7.0, EhP3 and 7.3, EhP1](ABENNEWS-703-SCREENS.html)
-   [Further Changes in ABAP Release 7.0, EhP3 and 7.3, EhP1](ABENNEWS-703-OTHERS.html)

-   The operational package concept in which package errors are detected by the syntax check and produced syntax errors
-   Support for class-based exceptions in RFC.
