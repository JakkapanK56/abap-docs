[Reference](ABAPBREAK-POINT.html)

`BREAK-POINT [\ \{ID group\}\ |\ \{log_text\}\ ].`

Defines a [breakpoint](ABENBREAKPOINT_GLOSRY.html) for branching to ABAP Debugger.

-   `ID group`\\
    Assigns the breakpoint to a [checkpoint group](ABENCHECKPOINT_GROUP_GLOSRY.html)\\ `group` that controls its activation. Without this specification, a breakpoint is always active.
-   `log_text`\\
    Specifies a supplementary text for the system log.
