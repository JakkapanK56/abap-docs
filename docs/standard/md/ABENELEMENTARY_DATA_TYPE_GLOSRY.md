A scalar [data type](ABENDATA_TYPE_GLOSRY.html) of fixed or variable length that is neither [structured](ABENSTRUCTURED_TYPE_GLOSRY.html), nor a [table type](ABENTABLE_TYPE_GLOSRY.html) or a [reference type](ABENREFERENCE_TYPE_GLOSRY.html).

An [enumerated type](ABENENUM_TYPE_GLOSRY.html) is a special kind of an elementary data type that can be defined by users in ABAP and in ABAP CDS.

-   In ABAP, user-defined elementary data types are based on the [built-in ABAP types](ABENBUILTIN_ABAP_TYPE_GLOSRY.html).
-   In [ABAP Dictionary](ABENABAP_DICTIONARY_GLOSRY.html), user-defined elementary data types are represented as [DDIC data elements](ABENDATA_ELEMENT_GLOSRY.html) that are based on elementary built-in dictionary types.
-   In [ABAP CDS](ABENABAP_CDS_GLOSRY.html), user-defined elementary data types are represented as [CDS simple types](ABENCDS_SIMPLE_TYPE_GLOSRY.html) that are based on elementary built-in dictionary types.
