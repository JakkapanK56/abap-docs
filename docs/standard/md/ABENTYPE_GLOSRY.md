Either a [data type](ABENDATA_TYPE_GLOSRY.html) for a [data object](ABENDATA_OBJECT_GLOSRY.html) or an [object type](ABENOBJECT_TYPE_GLOSRY.html) for an object in [ABAP Objects](ABENABAP_OBJECTS_GLOSRY.html).
