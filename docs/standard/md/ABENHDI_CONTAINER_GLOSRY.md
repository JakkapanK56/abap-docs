An HDI container is a [database schema](ABENDATABASE_SCHEMA_GLOSRY.html) of the [SAP HANA database](ABENHANA_DATABASE_GLOSRY.html) for [HDI objects](ABENHDI_OBJECT_GLOSRY.html). Special [ABAP Managed HDI Containers (AMHC)](ABENAMHC_GLOSRY.html) can be created for HDI objects accessed in ABAP.
