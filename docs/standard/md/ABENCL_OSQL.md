The following system interfaces and classes are available for extending the scope of the ABAP SQL language:

-   [`IF_ABAP_SQL_CONNECTION`](ABENIF_ABAP_SQL_CONNECTION.html) for ABAP SQL connection objects
-   [`CL_OSQL_EXTENDED_RESULT`](ABENCL_OSQL_EXTENDED_RESULT.html) handles extended results
-   [`CL_OSQL_REPLACE`](ABENCL_OSQL_REPLACE.html) provides a replacement service for executing [unit tests](ABENUNIT_TEST_GLOSRY.html)
-   [`CL_ABAP_DBFEATURES`](ABENCL_ABAP_DBFEATURES.html) returns features of the current database
