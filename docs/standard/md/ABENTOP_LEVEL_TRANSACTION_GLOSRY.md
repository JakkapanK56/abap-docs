The first program of a [call sequence](ABENCALL_SEQUENCE_GLOSRY.html). The [ABAP memory](ABENABAP_MEMORY_GLOSRY.html) assigned to a call sequence is retained for as long as the [internal session](ABENINTERNAL_SESSION_GLOSRY.html) of the top level transaction remains loaded. The initial [SAP Easy Access](ABENSAP_EASY_ACCESS_GLOSRY.html) program does not usually work as a top level transaction, except in [area instance versions](ABENAREA_INSTANCE_VERSION_GLOSRY.html).
