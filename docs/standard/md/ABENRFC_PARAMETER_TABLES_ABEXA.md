This example demonstrates parameter passing with parameter and exception tables for sRFC.

The function module `DEMO_RFM_PARAMETERS` is called using [sRFC](ABENSRFC_GLOSRY.html). Parameters are passed and exceptions are handled dynamically using internal tables `ptab` and `etab`.
