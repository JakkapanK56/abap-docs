`SELECTION-SCREEN BEGIN OF LINE.  ...  SELECTION-SCREEN END OF LINE.`

Defines a line on the current [selection screen](ABENSELECTION_SCREEN_GLOSRY.html) in which screen elements can be positioned using [`SELECTION-SCREEN POSITION`](ABAPSELECTION-SCREEN_SHORTREF.html).
