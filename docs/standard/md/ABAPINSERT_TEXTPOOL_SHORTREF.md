[Reference](ABAPINSERT_TEXTPOOL.html)

`INSERT TEXTPOOL prog FROM itab LANGUAGE lang.`

Inserts the text elements in the internal table `itab` of the type `TEXTPOOL` into the [repository](ABENREPOSITORY_GLOSRY.html). They are inserted as a text pool of the language specified in `lang` for the program specified in `prog`.
