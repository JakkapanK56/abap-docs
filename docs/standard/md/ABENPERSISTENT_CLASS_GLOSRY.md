Special [class](ABENCLASS_GLOSRY.html) whose [attributes](ABENATTRIBUTE_GLOSRY.html) are linked to [DDIC database tables](ABENDDIC_DB_TABLE_GLOSRY.html) via object-relational mapping. Can be created using the Mapping Assistant in [Class Builder](ABENCLASS_BUILDER_GLOSRY.html). The [objects](ABENOBJECT_GLOSRY.html) of persistent classes are managed by [object services](ABENOBJECT_SERVICES_GLOSRY.html).
