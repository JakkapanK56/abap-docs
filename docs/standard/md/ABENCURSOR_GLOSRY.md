A cursor determines a position. First, the term *cursor* means the position of a pointer in a display window. Second, a [list cursor](ABENLIST_CURSOR_GLOSRY.html) determines the output position in a [list](ABENCLASSIC_LIST_GLOSRY.html), and there is also a [database cursor](ABENDATABASE_CURSOR_GLOSRY.html) to read data from [database tables](ABENDATABASE_TABLE_GLOSRY.html). In this sense, the [file pointer](ABENFILE_POINTER_GLOSRY.html) can be seen as a cursor as well.
