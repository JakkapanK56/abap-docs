A static cache improves performance when a view is repeatedly accessed on an SAP HANA database. Called [*Result Cache*](https://help.sap.com/docs/SAP_HANA_PLATFORM/bed8c14f9f024763b0777aa72b5436f6/8ea43b3ff2e541e398e27d7e5caefab7) on SAP HANA. The cache is kept stable for a predefined retention time, and invalidated when this time is over.

In contrast to [dynamic caches](ABENDYNAMIC_CACHE_GLOSRY.html), static caches are not yet supported by the [ABAP technology](ABENABAP_TECHNO_GLOSRY.html).
