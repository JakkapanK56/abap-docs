A table that stores a set of different [edges](ABENEDGE_GLOSRY.html). Among them, there are at least two additional columns that reference the key column of the [vertex table](ABENVERTEX_TABLE_GLOSRY.html). One of the columns identifies the source [vertex](ABENVERTEX_GLOSRY.html), and the other identifies the target vertex of an [edge](ABENEDGE_GLOSRY.html).
