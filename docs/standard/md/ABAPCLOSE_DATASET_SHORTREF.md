`CLOSE DATASET dset.`

Closes a file on the current [AS instance](ABENAS_INSTANCE_GLOSRY.html)\\ [host computer](ABENHOST_COMPUTER_GLOSRY.html) that was specified in `dset` and opened previously using [`OPEN DATASET`](ABAPOPEN_DATASET.html).
