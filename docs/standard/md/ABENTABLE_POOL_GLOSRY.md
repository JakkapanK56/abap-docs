Before [ABAP release](ABENABAP_RELEASE_GLOSRY.html) 7.73, [DDIC database table](ABENDDIC_DB_TABLE_GLOSRY.html) in the [database](ABENDATABASE_GLOSRY.html) that contains the data of multiple [pooled tables](ABENPOOLED_TABLE_GLOSRY.html). Pooled tables are no longer supported and all table pools have been removed.
