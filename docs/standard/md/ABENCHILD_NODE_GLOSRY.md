A [hierarchy node](ABENHIERARCHY_NODE_GLOSRY.html) that inherits a [parent-child relationship](ABENPCR_GLOSRY.html) from a [parent node](ABENPARENT_NODE_GLOSRY.html). A child node can have multiple parent nodes. A child node is a [descendant node](ABENDESCENDANT_NODE_GLOSRY.html) of its parent nodes, their parent nodes, and so on. A hierarchy in which each child node has only one parent node is a [tree hierarchy](ABENTREE_HIERARCHY_GLOSRY.html), where a [root node](ABENROOT_NODE_GLOSRY.html) is the origin of a tree.
