The strict mode of the syntax check in ABAP release 7.72 applies to all ABAP SQL statements that use one of the following features introduced in [ABAP release 7.72](ABENNEWS-772-ABAP_SQL.html):

The strict mode in ABAP release 7.72 covers all rules of the [strict mode in ABAP release 7.71](ABENABAP_SQL_STRICTMODE_771.html).

-   Use of one of the new [time zone functions](ABENSQL_TIMEZONE_FUNC.html)\\ `ABAP_SYSTEM_TIMEZONE` or `ABAP_USER_TIMEZONE`.
-   Use of the relational expression [`IS [NOT] INITIAL`](ABENWHERE_LOGEXP_INITIAL.html).
