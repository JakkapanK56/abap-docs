`CONTROLS contrl TYPE \{ TABLEVIEW USING SCREEN dynnr \}\                     |\ \{ TABSTRIP \}.`

Declares the use of a [control](ABENCONTROL_GLOSRY.html)\\ `contrl` on a [dynpro](ABENDYNPRO_GLOSRY.html) of the program. An identically named structure is created for controlling the control.

-   [`TABLEVIEW USING SCREEN dynnr`](ABAPCONTROLS_TABLEVIEW.html)\\
    Declares the use of a [table control](ABENTABLE_CONTROL_GLOSRY.html) on the dynpro specified in `dynnr`.
-   [`TABSTRIP`](ABAPCONTROLS_TABSTRIP.html)\\
    Declares the use of a [tabstrip control](ABENTABSTRIP_CONTROL_GLOSRY.html).
