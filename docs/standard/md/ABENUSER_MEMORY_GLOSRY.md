Memory area of the current [AS instance](ABENAS_INSTANCE_GLOSRY.html) to which all [ABAP sessions](ABENABAP_SESSION_GLOSRY.html) of a [user session](ABENUSER_SESSION_GLOSRY.html) have access. [ABAP programs](ABENABAP_PROGRAM_GLOSRY.html) have access to [SPA/GPA parameters](ABENSPA_GPA_PARAMETER_GLOSRY.html) stored in the user memory. [ABAP daemons](ABENABAP_DAEMON_GLOSRY.html) access the subarea [ABAP daemon memory](ABENABAP_DAEMON_MEMORY_GLOSRY.html) of the user memory
