A new interface has been developed for [tRFC](ABENTRFC_2_GLOSRY.html) and [qRFC](ABENQRFC_GLOSRY.html), which will exist in parallel with the old interface from ABAP release 7.0. The statement for calling a transactional RFC from the new interface is [`CALL FUNCTION IN BACKGROUND UNIT`](ABAPCALL_FUNCTION_BACKGROUND_UNIT.html).

Complete shipment from [ABAP release 7.0, EhP2](ABENNEWS-71-RFC.html) only.
