In ABAP release 7.40 and higher, special access types for entities in the [SAP HANA database](ABENHANA_DATABASE_GLOSRY.html) are possible (if this database is used as the [standard database](ABENSTANDARD_DB_GLOSRY.html) for an AS ABAP).

[1\. Access to SAP HANA Views](#ABAP_MODIFICATION_1@4@)

[2. **SQLScript** Call](#ABAP_MODIFICATION_2@4@)

[DDIC external views](ABENEXTERNAL_VIEWS.html) in ABAP Dictionary make [SAP HANA views](ABENHANA_VIEW_GLOSRY.html) known in ABAP programs. They also allow type references and access using ABAP SQL.

The new statement [`CALL DATABASE PROCEDURE`](ABAPCALL_DATABASE_PROCEDURE.html) enables [SQLScript](ABENSQL_SCRIPT_GLOSRY.html) to be called from ABAP programs.
