Repository of objects in the [SAP HANA database](ABENHANA_DATABASE_GLOSRY.html). More specifically, [XSC](ABENXSC_GLOSRY.html) objects are stored in the SAP HANA Repository. [XSA](ABENXSA_GLOSRY.html) objects, on the other hand, are not stored in the SAP HANA Repository but are usually deployed using [HDI](ABENHDI_GLOSRY.html).
