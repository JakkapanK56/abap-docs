[Interface](ABENCLASS_GLOSRY.html) of the [class library](ABENCLASS_LIBRARY_GLOSRY.html) that is included in [system classes](ABENSYSTEM_CLASS_GLOSRY.html) and is used to implement system functions in separate classes or is required as a [tag interface](ABENTAG_INTERFACE_GLOSRY.html). The name usually begins with `IF_ABAP_` or `IF_SYSTEM_`. Using the [input help](ABENINPUT_HELP_GLOSRY.html) for`IF_ABAP_*` or `IF_SYSTEM_*` in the [Class Builder](ABENCLASS_BUILDER_GLOSRY.html) can give an overview of all system interfaces.
