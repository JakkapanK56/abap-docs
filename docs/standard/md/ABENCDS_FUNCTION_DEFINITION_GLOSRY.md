A [CDS object](ABENCDS_OBJECT_GLOSRY.html) that defines a [CDS function](ABENCDS_FUNCTION_GLOSRY.html) using language elements of the [CDS FDL](ABENCDS_FDL_GLOSRY.html) in [ABAP CDS](ABENABAP_CDS_GLOSRY.html) in [FDL source code](ABENFDL_SOURCE_CODE_GLOSRY.html). Currently, [CDS scalar function definitions](ABENCDS_DSFD_GLOSRY.html) are the only type of CDS function definition available.
