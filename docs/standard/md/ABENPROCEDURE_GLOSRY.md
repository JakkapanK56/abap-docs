[Processing block](ABENPROCESSING_BLOCK_GLOSRY.html) that can be called using an [ABAP statement](ABENABAP_STATEMENT_GLOSRY.html). After ending a procedure, processing continues behind the call position in the ABAP program. A procedure has a [parameter interface](ABENPARAMETER_INTERFACE_GLOSRY.html) and a local [data area](ABENDATA_AREA_GLOSRY.html). Possible procedures are [methods](ABENMETHOD_GLOSRY.html), [function modules](ABENFUNCTION_MODULE_GLOSRY.html), and [subroutines](ABENSUBROUTINE_GLOSRY.html).
