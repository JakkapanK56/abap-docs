[Table index](ABENTABLE_INDEX_GLOSRY.html) for the administration of [sorted](ABENSORTED_KEY_GLOSRY.html)\\ [secondary table keys](ABENSECONDARY_TABLE_KEY_GLOSRY.html). A secondary table index of a non-unique key is not updated immediately for each line insertion or deletion in an [internal table](ABENINTERNAL_TABLE_GLOSRY.html); it is only updated when the internal table is accessed via the associated secondary key.
