The following statements process lists that are already saved as [screen lists](ABENSCREEN_LIST_GLOSRY.html) in the list buffers:

[Spool lists](ABENSPOOL_LIST_GLOSRY.html) that are sent page by page to the SAP spool system cannot be processed in this manner.

-   [`READ LINE`](ABAPREAD_LINE.html),
-   [`MODIFY LINE`](ABAPMODIFY_LINE.html),
-   [`SCROLL LIST`](ABAPSCROLL.html), and
-   [`DESCRIBE LIST`](ABAPDESCRIBE_LIST.html)
