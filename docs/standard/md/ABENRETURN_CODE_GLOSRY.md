Numeric value that is placed in the [system field](ABENSYSTEM_FIELD_GLOSRY.html)\\ [`sy-subrc`](ABENSYSTEM_FIELDS.html) for many [ABAP statements](ABENABAP_STATEMENT_GLOSRY.html) when the statement is executed. The value 0 usually indicates a successful completion.
