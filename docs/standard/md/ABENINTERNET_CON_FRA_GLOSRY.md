ICF for short. A software layer in [AS ABAP](ABENAS_ABAP_GLOSRY.html) that provides an ABAP interface for HTTP, HTTPS, and SMTP requests. Using the ICF, [ABAP programs](ABENABAP_PROGRAM_GLOSRY.html) can communicate with the internet in both the sever and the client role. The ICF receives and sends web-based ABAP calls using the [Internet Communication Manager](ABENINTERNET_CON_MAN_GLOSRY.html).
