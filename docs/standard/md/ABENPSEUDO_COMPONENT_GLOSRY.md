Expression `table_line` that can be specified in accesses to [internal tables](ABENINTERNAL_TABLE_GLOSRY.html) instead of a [structure component](ABENSTRUCTURE_COMPONENT_GLOSRY.html). The entire line of the internal table is then interpreted as the only component. The name `table_line` is a reserved name. In ABAP and in [ABAP Dictionary](ABENABAP_DICTIONARY.html), no structure components called `table_line` can be defined or created.
