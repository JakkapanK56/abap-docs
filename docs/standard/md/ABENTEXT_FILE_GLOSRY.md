[File](ABENFILE_GLOSRY.html) on the [host computer](ABENHOST_COMPUTER_GLOSRY.html) of an [AS instance](ABENAS_INSTANCE_GLOSRY.html) that has been opened using the addition [`TEXT`](ABAPOPEN_DATASET_MODE.html) of the statement [`OPEN DATASET`](ABAPOPEN_DATASET_MODE.html). In a text file, the content is handled as text in the [character representation](ABENCHAR_REPRESENTATION_GLOSRY.html) that was chosen when the file was opened.
