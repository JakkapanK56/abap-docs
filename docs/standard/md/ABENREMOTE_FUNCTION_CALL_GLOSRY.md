RFC for short. Call of a [function module](ABENFUNCTION_MODULE_GLOSRY.html) using the [RFC interface](ABENRFC_INTERFACE_GLOSRY.html) Calls be made in the same or in different systems. The target system is determined using an [RFC destination](ABENRFC_DEST_GLOSRY.html). Connections are possible between different [AS ABAP](ABENAS_ABAP_GLOSRY.html) or between an AS ABAP and an external system. In non-SAP systems, specially programmed functions whose [interface](ABENINTF_GEN_GLOSRY.html) simulates a function module are called in external systems instead of function modules. Function calls can be [synchronous](ABENSYNCHRONOUS_RFC_GLOSRY.html), [asynchronous](ABENASYNCHRONOUS_RFC_GLOSRY.html), or [transactional](ABENTRFC_1_GLOSRY.html).
