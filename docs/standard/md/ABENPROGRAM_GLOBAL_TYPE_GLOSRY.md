[Data type](ABENDATA_TYPE_GLOSRY.html) or [object type](ABENOBJECT_TYPE_GLOSRY.html) that is defined in an [ABAP program](ABENABAP_PROGRAM_GLOSRY.html) and is known in the entire program behind its declaration. Global types are declared in the [global declaration part](ABENGLOBAL_DECLARATION_SECT_GLOSRY.html) of a program using the statements [`TYPES`](ABAPTYPES.html), [`CLASS`](ABAPCLASS.html), and [`INTERFACE`](ABAPINTERFACE.html).
