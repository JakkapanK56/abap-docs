This example demonstrates the possible XML sources of the statement [`CALL TRANSFORMATION`](ABAPCALL_TRANSFORMATION.html).

The [identity transformation](ABENID_TRAFO_GLOSRY.html)\\ `ID` is called in the order of the possible formats with all XML sources that are possible for a format. The XML target is always a byte string, which is output after each successful transformation.
