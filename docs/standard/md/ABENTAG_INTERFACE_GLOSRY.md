Special predefined global interface. By implementing or including a tag interface, [classes](ABENCLASS_GLOSRY.html) or other [interfaces](ABENOO_INTF_GLOSRY.html) are particularly tagged for the [ABAP runtime framework](ABENABAP_RUNTIME_FRMWK_GLOSRY.html). A tag interface generally does not contain its own [interface components](ABENINTERFACE_COMPONENT_GLOSRY.html) but instead assigns a particular task to the integrating classes or interfaces and changes the way they are handled by the [ABAP Compiler](ABENABAP_COMPILER_GLOSRY.html).
