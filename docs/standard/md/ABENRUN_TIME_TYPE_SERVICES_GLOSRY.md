RTTS for short. Combination of [RTTC](ABENRUN_TIME_TYPE_CREATION_GLOSRY.html) and [RTTI](ABENRUN_TIME_TYPE_IDENTIFIC_GLOSRY.html) in a class hierarchy. The RTTS methods provide information on [data types](ABENDATA_TYPE_GLOSRY.html) or [classes](ABENCLASS_GLOSRY.html) and [interfaces](ABENOO_INTF_GLOSRY.html) of [objects](ABENOBJECT_GLOSRY.html), or they create new data types as [type description objects](ABENTYPE_OBJECT_GLOSRY.html). RTTS is implemented as a hierarchy of [type description classes](ABENTYPE_CLASS_GLOSRY.html) from which type description objects can be created.
