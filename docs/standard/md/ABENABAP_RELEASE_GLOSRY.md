Release of [ABAP](ABENABAP_GLOSRY.html) ([ABAP technology](ABENABAP_TECHNO_GLOSRY.html) including the [ABAP language](ABENABALA_GLOSRY.html) and its other programming languages). The counting of the releases is done by release numbers such as 7.57, 7.58, 8.16, ... These release numbers are based on the technical release names of the associated software component `SAP_BASIS` as it can be found in system field `sy-saprl`. ABAP is not released standalone but is shipped together with SAP products such as [SAP BTP ABAP environment](ABENSAP_BTP_ABAP_ENV_GLOSRY.html), [SAP S/4HANA Cloud Public Edition](ABENSAP_S4HANA_CPUB_GLOSRY.html) and [SAP S/4HANA](ABENSAP_S4HANA_GLOSRY.html), [SAP S/4HANA Cloud Private Edition](ABENSAP_S4HANA_CPRIV_GLOSRY.html). These products have other release numbers that can be mapped to the ABAP releases. The releases of the underlying [ABAP kernel](ABENKERNEL_GLOSRY.html) are also counted differently and can also be mapped to the ABAP releases.
