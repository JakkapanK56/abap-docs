The strict mode of the syntax check in ABAP release 7.91 applies to all ABAP SQL statements that use one of the following features introduced in [ABAP release 7.91](ABENNEWS-791-ABAP_SQL.html):

The strict mode in ABAP release 7.91 covers all rules of the [strict mode in ABAP release 7.90](ABENABAP_SQL_STRICTMODE_790.html).

-   [Path expressions](ABENABAP_SQL_PATH.html) that specify a [cardinality](ABENCARDINALITY_GLOSRY.html) using the syntax written in words.
-   Cardinality specification in a [join expression](ABAPSELECT_JOIN.html).
-   Use of addition [`PRIVILEGED ACCESS`](ABAPSELECT_PRIVILEGED_ACCESS.html)
