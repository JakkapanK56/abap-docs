Reference to an [object](ABENOBJECT_GLOSRY.html) of a [class](ABENCLASS_GLOSRY.html) that is ignored when the [Garbage Collector](ABENGARBAGE_COLLECTOR_GLOSRY.html) is executed, unlike [object references](ABENOBJECT_REFERENCE_GLOSRY.html) in [reference variables](ABENREFERENCE_VARIABLE_GLOSRY.html). An object that is only pointed to by weak references is deleted by the Garbage Collector. Weak references are represented by objects of the class [`CL_ABAP_WEAK_REFERENCE`](ABENCL_ABAP_WEAK_REFERENCE.html). See also [soft reference](ABENSOFT_REFERENCE_GLOSRY.html).
