Hierarchy nodes are the objects of a [hierarchy](ABENHIERARCHY_GLOSRY.html) linked using [hierarchy edges](ABENHIERARCHY_EDGE_GLOSRY.html). Hierarchy nodes can be [root nodes](ABENROOT_NODE_GLOSRY.html), [parent nodes](ABENPARENT_NODE_GLOSRY.html), [child nodes](ABENPARENT_NODE_GLOSRY.html), [leaf nodes](ABENLEAF_NODE_GLOSRY.html), [orphan nodes](ABENORPHAN_NODE_GLOSRY.html), [sibling nodes](ABENSIBLING_NODE_GLOSRY.html), [descendant nodes](ABENDESCENDANT_NODE_GLOSRY.html), and [ancestor nodes](ABENANCESTOR_NODE_GLOSRY.html).
