Text that can be accessed using the statement [`MESSAGE`](ABAPMESSAGE.html). Messages are always displayed in the [logon language](ABENLOGON_LANGUAGE_GLOSRY.html), which cannot be changed during a [user session](ABENUSER_SESSION_GLOSRY.html). Message texts are organized into [message classes](ABENMESSAGE_CLASS_GLOSRY.html) and [message numbers](ABENMESSAGE_NUMBER_GLOSRY.html) and are maintained in the [transaction](ABENTRANSACTION_GLOSRY.html)\\ [message maintenance](ABENMESSAGE_MAINTENANCE_GLOSRY.html). When a message is accessed using the statement [`MESSAGE`](ABAPMESSAGE.html), it is classified with a [message type](ABENMESSAGE_TYPE_GLOSRY.html) that defines the [behavior](ABENABAP_MESSAGES_TYPES.html) of the message. Messages are also used as [exception texts](ABENEXCEPTION_TEXT_GLOSRY.html) of [exception classes](ABENEXCEPTION_CLASS_GLOSRY.html).
