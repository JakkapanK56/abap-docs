In ABAP release 7.67, a new [ABAP version](ABENABAP_VERSION_GLOSRY.html)\\ [Standard ABAP with restricted object use](ABENSTANDARD_ABAP_RESTR_OBJ_GLOSRY.html) was introduced. The internal version ID in the column `UCCHECK` of the system table `TRDIR` is *4*.
