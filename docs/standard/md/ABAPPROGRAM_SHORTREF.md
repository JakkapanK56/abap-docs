[Reference](ABAPPROGRAM.html)

`PROGRAM prog [NO STANDARD PAGE HEADING]\               [LINE-SIZE width]\               [LINE-COUNT page_lines[(footer_lines)]]\               [MESSAGE-ID mid]\               [REDUCED FUNCTIONALITY].`

Initiates a [module pool](ABENMODUL_POOL_GLOSRY.html) or a [subroutine pool](ABENSUBROUTINE_POOL_GLOSRY.html)\\ `prog`.

-   `NO STANDARD PAGE HEADING`, `LINE-SIZE`, `LINE-COUNT`, `MESSAGE-ID`\\
    See [`REPORT`](ABAPREPORT_SHORTREF.html).
-   [`REDUCED FUNCTIONALITY`](ABAPPROGRAM.html)\\
    Minimizes the components loaded when calling a subroutine pool.
