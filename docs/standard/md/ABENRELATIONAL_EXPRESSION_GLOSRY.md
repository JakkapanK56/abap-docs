[Expression](ABENEXPRESSION_GLOSRY.html) whose result is a [truth value](ABENTRUTH_VALUE_GLOSRY.html). Relational expressions include [comparison expressions](ABENCOMPARISON_EXPRESSION_GLOSRY.html) and [predicates](ABENPREDICATE_GLOSRY.html). A single relational expression or multiple relational expressions combined using [Boolean operators](ABENBOOLEAN_OPERATOR_GLOSRY.html) form a [logical expression](ABENLOGICAL_EXPRESSION_GLOSRY.html) in ABAP.
