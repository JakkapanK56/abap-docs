Instance of an [area class](ABENAREA_CLASS_GLOSRY.html). An area handle allows an ABAP program access to an [area instance version](ABENAREA_INSTANCE_VERSION_GLOSRY.html) in the [shared memory](ABENSHARED_MEMORY_GLOSRY.html) and to the [shared objects](ABENSHARED_OBJECTS_GLOSRY.html) stored there. An area handle is created by the static methods of its area class and is bound to an area instance version. Each bound area handle sets an [area lock](ABENAREA_LOCK_GLOSRY.html) on the affected area instance version.
