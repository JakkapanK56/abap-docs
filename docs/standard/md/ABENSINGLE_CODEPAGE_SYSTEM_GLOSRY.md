System with only one [system code page](ABENSYSTEM_CODEPAGE_GLOSRY.html). Either a [Unicode system](ABENUNICODE_SYSTEM_GLOSRY.html) with [UTF 16](ABENUTF16_GLOSRY.html) or a non-Unicode system with a non-Unicode code page. From [AS ABAP](ABENAS_ABAP_GLOSRY.html)\\ [ABAP release](ABENABAP_RELEASE_GLOSRY.html) 7.0, only single code page systems are supported and from ABAP release 7.62 only Unicode systems. The [text environment](ABENTEXT_ENVIRONMENT_GLOSRY.html) for an [internal session](ABENINTERNAL_SESSION_GLOSRY.html) can only be set using this code page in a single code page system. See also [MDMP system](ABENMDMP-SYSTEM_GLOSRY.html).
