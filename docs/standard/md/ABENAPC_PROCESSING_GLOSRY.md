Processing of a WebSocket or TCP request using the [APC](ABENAPC_GLOSRY.html) framework in an [APC handler class](ABENAPC_HANDLER_CLASS_GLOSRY.html). APC processing takes place in a dedicated [APC session](ABENAPC_SESSION_GLOSRY.html). APC processing can be stateless or stateful. Stateful APC processing must take place in [non-blocking mode](ABENNON_BLOCKING_MODEL_GLOSRY.html), where blocking statements are forbidden.
