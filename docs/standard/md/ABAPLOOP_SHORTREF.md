`LOOP.    ...  ENDLOOP.`

**Obsolete:** Reads the [extract data set](ABENEXTRACT_DATASET_GLOSRY.html) of the program in a loop and assigns (for each loop pass) the content of the read line to the data objects that make up the [field group](ABENFIELD_GROUP_GLOSRY.html) defined using `INSERT INTO field_group`.
