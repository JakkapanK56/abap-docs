`LEAVE LIST-PROCESSING.`

Exits list processing and continues processing with the [PBO](ABENPBO_GLOSRY.html) event of the [dynpro](ABENDYNPRO_GLOSRY.html) from which the list processor was called using `LEAVE TO LIST-PROCESSING` or the dynpro that was specified using `AND RETURN TO SCREEN`.
