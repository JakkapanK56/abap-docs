The new addition [`INDICATORS`](ABAPTYPES_INDICATORS.html) of the statement [`TYPES`](ABAPTYPES.html) defines an [indicator structure](ABENINDICATOR_STRUCTURE_GLOSRY.html) as a [substructure](ABENSUBSTRUCTURE_GLOSRY.html) of a given [structured type](ABENSTRUCTURED_TYPE_GLOSRY.html). An indicator structure can be used as a [ABAP SQL indicator](ABENABAP_SQL_INDICATOR_GLOSRY.html) in ABAP SQL read and write statements.
