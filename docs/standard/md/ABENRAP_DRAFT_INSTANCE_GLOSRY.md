A [RAP BO entity instance](ABENRAP_BO_ENTITY_INST_GLOSRY.html) in a [draft-enabled RAP business object](ABENDRAFT_RAP_BO_GLOSRY.html) for which a commit triggers the saving to the [RAP draft table](ABENDRAFT_TABLE_GLOSRY.html). The [BDEF derived type](ABENRAP_DERIVED_TYPE_GLOSRY.html) of such an instance has the [draft indicator](ABENRAP_DRAFT_INDICATOR_GLOSRY.html) set to *true*. A draft instance can be produced by a [create operation](ABENRAP_CREATE_OPERATION_GLOSRY.html) or by the [draft action](ABENRAP_BO_DRAFT_ACTION_GLOSRY.html)\\ `Edit` in draft scenarios.

There are [edit-draft instances](ABENRAP_EDIT_DRAFT_INST_GLOSRY.html) and [new-draft instances](ABENRAP_NEW_DRAFT_INST_GLOSRY.html).
