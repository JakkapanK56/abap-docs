`REFRESH CONTROL contrl FROM SCREEN dynnr.`

Assigns the values predefined in the [dynpro](ABENDYNPRO_GLOSRY.html)\\ `dynnr` to components of the structure `contrl` defined for a [table control](ABENTABLE_CONTROL_GLOSRY.html) using `CONTROLS`.
