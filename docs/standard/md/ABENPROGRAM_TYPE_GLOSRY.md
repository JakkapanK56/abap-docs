[Program property](ABENPROGRAM_PROPERTY_GLOSRY.html) of an [ABAP program](ABENABAP_PROGRAM_GLOSRY.html). Defines which [processing blocks](ABENPROCESSING_BLOCK_GLOSRY.html) can be contained in a program and how the [ABAP runtime framework](ABENABAP_RUNTIME_FRMWK_GLOSRY.html) executes the program. Possible program types include [executable program](ABENEXECUTABLE_PROGRAM_GLOSRY.html), [class pool](ABENCLASS_POOL_GLOSRY.html), [function pool](ABENFUNCTION_POOL_GLOSRY.html), [interface pool](ABENINTERFACE_POOL_GLOSRY.html), [module pool](ABENMODUL_POOL_GLOSRY.html), [subroutine pool](ABENSUBROUTINE_POOL_GLOSRY.html), [type pool](ABENTYPE_POOL_GLOSRY.html), and [include program](ABENINCLUDE_PROGRAM_GLOSRY.html). The program type is maintained in the [ABAP Editor](ABENABAP_EDITOR_GLOSRY.html) in the program properties, or by the relevant tool.
