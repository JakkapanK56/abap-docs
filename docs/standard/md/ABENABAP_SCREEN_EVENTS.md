Selection screen events and list events are events of classic [dynpro](ABENDYNPRO_GLOSRY.html) processing that are transformed into ABAP events by the ABAP runtime framework and are handled directly in the ABAP program instead of in the [dynpro flow logic](ABENDYNPRO_FLOW_LOGIC_GLOSRY.html).

They are described in their respective environments.

-   [Selection screen events](ABENSELECTION_SCREEN_EVENTS.html) occur during [selection screen processing](ABENEXECUTABLE_PROGRAM_GLOSRY.html).
-   [List events](ABENABAP_LISTS_INTERACTIVE.html) occur during classic [list processing](ABENLIST_PROCESSING_GLOSRY.html).
