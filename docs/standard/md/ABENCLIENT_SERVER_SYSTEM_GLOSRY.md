Software system that consists of multiple software layers. These are usually the [presentation layer](ABENPRESENTATION_LAYER_GLOSRY.html), the [application layer](ABENAPPLICATION_LAYER_GLOSRY.html), and the [database layer](ABENDATABASE_LAYER_GLOSRY.html). [AS ABAP](ABENAS_ABAP_GLOSRY.html) represents the application layer of a client-server system ([ABAP system](ABENABAP_SYSTEM_GLOSRY.html)) in which [ABAP programs](ABENABAP_PROGRAM_GLOSRY.html) are executed.
