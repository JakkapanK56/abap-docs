[Event](ABENEVENT_GLOSRY.html) of the [ABAP runtime framework](ABENABAP_RUNTIME_FRMWK_GLOSRY.html) that occurs in [executable programs](ABENEXECUTABLE_PROGRAM_GLOSRY.html) when these are started via a [`SUBMIT`](ABAPSUBMIT.html) or via a [report transaction](ABENREPORT_TRANSACTION_GLOSRY.html). Reporting events map the [IPO](ABENEVA_GLOSRY.html) principle of [reporting](ABENREPORTING_GLOSRY.html) when used with a [logical database](ABENLOGICAL_DATA_BASE_GLOSRY.html). The [event key words](ABENEVENT_KEYWORD_GLOSRY.html) assigned to reporting events are [`INITIALIZATION`](ABAPINITIALIZATION.html), [`START-OF-SELECTION`](ABAPSTART-OF-SELECTION.html), [`GET`](ABAPGET-.html), and [`END-OF-SELECTION`](ABAPEND-OF-SELECTION.html).
