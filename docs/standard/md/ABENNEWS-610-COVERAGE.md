Coverage Analyzer is responsible for recording the number of calls for ABAP program units, by time period, user, or subject. The following table lists all the program units and their associated keywords.

A detailed description of Coverage Analyzer functions is available under the application help for the tool. Start Coverage Analyzer and choose *Help -> Application Help*.
