Position in a [repository object](ABENREPOSITORY_OBJECT_GLOSRY.html) that can be enhanced with the [ABAP Enhancement Framework](ABENENHANCEMENT_FRAMEWORK_GLOSRY.html). Enhancement options can be explicitly defined or available implicitly. Explicitly defined enhancement options are implemented using [enhancement spot element definitions](ABENENHANCEMENT_SPOT_DEF_GLOSRY.html) and [enhancement spot invocations](ABENENHANCEMENT_SPOT_INVOC_GLOSRY.html) and are administrated using [enhancement spots](ABENENHANCEMENT_SPOT_GLOSRY.html).
