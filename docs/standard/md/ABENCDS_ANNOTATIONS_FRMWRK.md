Framework-specific annotations are [SAP annotations](ABENSAP_ANNOTATION_GLOSRY.html) that define the technical and semantic properties of CDS objects that are required by specific software components, such as `ODATA`, `UI`, and Analytics. The following sections show the associated [annotation definitions](ABENCDS_ANNO_DEFINITION_GLOSRY.html) and the [annotation syntax](ABENCDS_ANNOTATION_SYNTAX_GLOSRY.html):

The documentation [CDS Annotations](https://help.sap.com/docs/ABAP_Cloud/f055b8bf582d4f34b91da667bc1fcce6/130e02a697e14bf8b05dd6672c56250b?version=sap_cross_product_abap) describes the meaning of framework-specific annotations, which SAP framework is evaluated by these annotations, and their allowed values.

When using the framework-specific annotations, note the corresponding established rules. Otherwise error messages can occur during framework-specific checks.

-   [Framework-specific annotation syntax](ABENCDS_ANNOTATIONS_FRMWRK_TABLES.html)
