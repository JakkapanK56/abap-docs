[Class](ABENCLASS_GLOSRY.html) that implements the [interface](ABENOO_INTF_GLOSRY.html)\\ `IF_ABAP_TIMER_HANDLER`. The interface method `ON_TIMEOUT` of the timer handler, which is specified with a [ABAP Timer Manager](ABENABAP_TIMER_MANAGER_GLOSRY.html) when an [ABAP Timer](ABENABAP_TIMER_GLOSRY.html) is started. It handles the timeout event of the timer.
