The program `DEMO_EXPRESSIONS` shows examples of the logical expressions in ABAP release 7.0, EhP2.

The new built-in [Boolean functions](ABENBOOLE_FUNCTIONS.html)\\ `boolc` and `boolx` accept logical expressions as arguments and return their truth values in the form of character strings or byte strings.

The new [predicate functions](ABENPREDICATE_FUNCTIONS.html) return truth values and can be used like relational expressions. Predicate functions for character-like arguments were introduced in ABAP release 7.0, EhP2.

The new Boolean operator [`EQUIV`](ABENLOGEXP_EQUIV.html) enables an equivalence relationship between two [logical expressions](ABENLOGICAL_EXPRESSION_GLOSRY.html).
