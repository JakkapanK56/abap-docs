All lines in an [internal table](ABENINTERNAL_TABLE_GLOSRY.html). Internal table and table body are identical for internal tables without a [header line](ABENHEADER_LINE_GLOSRY.html). For internal tables with a header line, the table body denotes the actual tabular data and can be addressed by attaching `[]` to the table name.
