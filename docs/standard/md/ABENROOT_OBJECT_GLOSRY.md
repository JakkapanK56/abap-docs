Special [shared object](ABENSHARED_OBJECT_GLOSRY.html). [Instance](ABENINSTANCE_GLOSRY.html) of the [area root class](ABENROOT_DATA_CLASS_GLOSRY.html). A non-empty [area instance version](ABENAREA_INSTANCE_VERSION_GLOSRY.html) must contain a root object. The [attribute](ABENATTRIBUTE_GLOSRY.html)\\ `ROOT` of an associated [area handle](ABENAREA_HANDLE_GLOSRY.html) points to the root object, which can contain references to the other shared objects of the area instance version.
