This example demonstrates group level processing for an unsorted internal table.

Group level processing using an unsorted internal table with the group key `col1`. Group breaks result from each change in the content of this column. A grouping using [`GROUP BY`](ABAPLOOP_AT_ITAB_GROUP_BY.html) for comparison produces a different result, since it is independent of the sorting of the internal table. Group level processing for the table sorted by `col1` has the same result as the grouping.
