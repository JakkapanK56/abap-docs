`SET LEFT SCROLL-BOUNDARY [COLUMN col].`

Defines the left boundary of the horizontal scroll range of a list page.

-   `COLUMN col`\\
    Excludes all places left of `col` from scrolling. If this is not specified, the current cursor position is used.
