[Reference](ABAPSET_LOCALE.html)

`SET LOCALE LANGUAGE lang [COUNTRY cntry]\ [MODIFIER mod].`

Sets the [text environment](ABENTEXT_ENVIRONMENT_GLOSRY.html).

-   `LANGUAGE lang`\\
    Specifies the language in `lang`.
-   [`COUNTRY cntry`](ABAPSET_LOCALE_OBSOLETE.html)\\
    **Obsolete:** Specifies the country key in `cntry`.
-   [`MODIFIER mod`](ABAPSET_LOCALE_OBSOLETE.html)\\
    **Obsolete:** Specifies the ID for a specific locale in `mod`.
