This example demonstrates table comprehensions and table reductions for mesh paths.

This example uses the same mesh paths as the executable example for [`LOOP AT`](ABENMESH_LOOPS_ABEXA.html) and thus produces the same result sets. Unlike [`LOOP AT`](ABENMESH_LOOP.html), the lines in the result set are produced in a results table using [table comprehensions](ABENTABLE_COMPREHENSION_GLOSRY.html) and not as individual lines.

For the first mesh path, a [table reduction](ABENTABLE_REDUCTION_GLOSRY.html) is also performed using the reduction operator [`REDUCE`](ABENCONSTRUCTOR_EXPRESSION_REDUCE.html). This totals the distances, with the conversion from kilometers to miles using a [`SWITCH`](ABENCONDITIONAL_EXPRESSION_SWITCH.html) expression.
