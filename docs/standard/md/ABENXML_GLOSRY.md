Short for Extensible Markup Language, see [www.w3.org/XML/](http://www.w3.org/XML/). XML is a generic model for the structuring of data. XML has a generic syntax (markup) used to represent elements of the model in a text format. The representation in text format enables the representation of structured data without using the program that created this data. ABAP data can be transformed to XML and back using [XSL transformations](ABENXSL_TRANSFORMATION_GLOSRY.html) and [Simple Transformations](ABENSIMPLE_TRANSFORMATION_GLOSRY.html). The methods in the [iXML Library](ABENABAP_IXML_LIB_CLOUD.html) and the [sXML Library](ABENSXML_LIBRARY_GLOSRY.html) make it possible to process XML data.
