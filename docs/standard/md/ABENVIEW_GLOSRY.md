Grouping of columns from one or more [database tables](ABENDATABASE_TABLE_GLOSRY.html) or other [database objects](ABENDB_OBJECT_GLOSRY.html). In an [AS ABAP](ABENAS_ABAP_GLOSRY.html), views are defined in the [ABAP Dictionary](ABENABAP_DICTIONARY_GLOSRY.html) as [DDIC views](ABENDDIC_VIEW_GLOSRY.html) or in [ABAP CDS](ABENABAP_CDS_GLOSRY.html) as [CDS views](ABENCDS_VIEW_GLOSRY.html). Both can be referenced as [data types](ABENDATA_TYPE_GLOSRY.html) in ABAP programs and represent [flat structures](ABENFLAT_STRUCTURE_GLOSRY.html) there. Views can be used like database tables in reading [ABAP SQL](ABENABAP_SQL_GLOSRY.html) statements.
