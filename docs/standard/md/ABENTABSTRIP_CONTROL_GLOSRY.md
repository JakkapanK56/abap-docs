Tabstrip controls allow tab pages to be displayed on dynpros. A tabstrip control is a [screen element](ABENSCREEN_ELEMENT_GLOSRY.html) built from several [tab pages](ABENTAB_PAGE_GLOSRY.html) or tabs.
