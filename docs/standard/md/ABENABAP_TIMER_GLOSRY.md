A timer for wait times in [non-blocking mode](ABENNON_BLOCKING_MODEL_GLOSRY.html) integrated into the [ABAP runtime framework](ABENABAP_TIMER_MANAGER_GLOSRY.html). An ABAP Timer is started using a [timer manager](ABENABAP_TIMER_MANAGER_GLOSRY.html) for a given runtime and at the end a timeout event is raised, which can be responded to with a [timer handler](ABENABAP_TIMER_HANDLER_GLOSRY.html).
