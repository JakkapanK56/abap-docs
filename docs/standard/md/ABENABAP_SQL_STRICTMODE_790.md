The strict mode of the syntax check in ABAP release 7.90 applies to all ABAP SQL statements that use one of the following features introduced in [ABAP release 7.90](ABENNEWS-790-ABAP_SQL.html):

The strict mode in ABAP release 7.90 covers all rules of the [strict mode in ABAP release 7.89](ABENABAP_SQL_STRICTMODE_789.html).

-   Use of [SQL expressions](ABENABAP_SQL_EXPR_LOGEXP.html) as operands on the right side of an SQL condition.
