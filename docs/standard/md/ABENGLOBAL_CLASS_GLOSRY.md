[Class](ABENCLASS_GLOSRY.html) that is defined as a [global type](ABENGLOBAL_TYPE_GLOSRY.html). Global classes are declared in [class pools](ABENCLASS_POOL_GLOSRY.html) and are marked as such there using the addition [`PUBLIC`](ABAPCLASS_OPTIONS.html) of the statement [`CLASS`](ABAPCLASS.html). A class pool can only contain one global class.
