[Reference](ABAPFORM_DEFINITION.html)

`FORM subr DEFINITION [ ... ].  \  FORM subr IMPLEMENTATION.    ...  ENDFORM.`

**Obsolete:** Defines a subroutine `subr`. The definition of the subroutine is shared between a declaration part and an implementation part.

-   [`DEFINITION`](ABAPFORM_DEFINITION.html)\\
    Declaration part of the subroutine. The declaration of the parameter interface is the same as in [`FORM`](ABAPFORM_SHORTREF.html).
-   [`IMPLEMENTATION`](ABAPFORM_DEFINITION.html)\\
    Implementation part of the subroutine. The implementation takes place between `FORM` and `ENDFORM`.
