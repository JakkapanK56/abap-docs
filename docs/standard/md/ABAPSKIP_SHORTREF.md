`SKIP \{\ [n]\       |\ \{TO LINE line\}\ \}.`

Positions the list cursor in a line of the current list.

-   `n`\\
    Positions the [list cursor](ABENLIST_CURSOR_GLOSRY.html)\\ `n` lines underneath the current line. If not specified, `n` is set implicitly to 1.
-   `TO LINE line`\\
    Positions the list cursor in the line `line`.
