The text environment is part of the [language settings](ABENLANGUAGE_SETTING_GLOSRY.html) of an [ABAP program](ABENABAP_PROGRAM_GLOSRY.html) and consists of a [text environment language](ABENTEXT_ENV_LANGU_GLOSRY.html), the [system code page](ABENSYSTEM_CODEPAGE_GLOSRY.html), and a [locale](ABENLOCALE_GLOSRY.html). All programs of an [internal session](ABENINTERNAL_SESSION_GLOSRY.html) have a common text environment.
