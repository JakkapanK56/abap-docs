ABAP Dictionary offers [customizing includes](ABENCUSTOMIZING_INCLUDE_GLOSRY.html), [append structures](ABENAPPEND_STRUCTURE_GLOSRY.html), and [DDIC append views](ABENDDIC_APPEND_VIEW_GLOSRY.html) to extend existing [DDIC structures](ABENDDIC_STRUCTURES.html), [DDIC database tables](ABENDDIC_DATABASE_TABLES.html), [DDIC database views](ABENDDIC_DATABASE_VIEWS.html), or [DDIC domains](ABENDDIC_DOMAINS.html) without modifying the original object. Customizing includes indicate points already marked by SAP for extensions, whereas append structures can be appended anywhere.

[Indexes](ABENINDEX_GLOSRY.html) of a [DDIC database table](ABENDDIC_DB_TABLE_GLOSRY.html) can be extended by [extension indexes](ABENEXTENSION_INDEX_GLOSRY.html).

[Search helps](ABENSEARCH_HELP_GLOSRY.html) can be extended using [append search helps](ABENAPPEND_SEARCH_HELP_GLOSRY.html).

[DDIC extensions](ABENDDIC_EXTENSION_GLOSRY.html) are connected to [Switch Framework](ABENSWITCH_FRAMEWORK_GLOSRY.html) whenever they are defined in a [package](ABENPACKAGE_GLOSRY.html) that is assigned a [switch](ABENSWITCH_GLOSRY.html).

For [CDS entities](ABENCDS_ENTITY_GLOSRY.html), [CDS entity extensions](ABENCDS_ENTITY_EXTEND_GLOSRY.html) are an option for extensions that is based on the [CDS DDL](ABENCDS_DDL_GLOSRY.html) of [ABAP CDS](ABENABAP_CDS_GLOSRY.html).
