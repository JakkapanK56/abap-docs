Special include program in which the [test classes](ABENTEST_CLASS_GLOSRY.html) of an [ABAP program](ABENABAP_PROGRAM_GLOSRY.html) are listed. Except for test includes for [class pools](ABENCLASS_POOL_GLOSRY.html) and [function pools](ABENFUNCTION_POOL_GLOSRY.html) no test includes can currently be created. The associated [ABAP Workbench](ABENABAP_WORKBENCH_GLOSRY.html) tools or [ABAP development tools for Eclipse (ADT)](ABENADT_GLOSRY.html) access the associated test include automatically. The name of the test include of a class pool ends with `CCAU` and the name of a function pool ends with `T99`. For other programs, the naming convention of the test include is the name of the program that is padded with `=` up to and including the 34th position and ending in `T`.
