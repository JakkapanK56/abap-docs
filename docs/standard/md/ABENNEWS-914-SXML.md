[Header options](ABENABAP_SXML_LIB_RENDER.html) for the XML declaration are now available in the sXML Library:

-   `CO_OPT_VAL_NO` (no XML header)
-   This is the default option.
-   `CO_OPT_VAL_WITHOUT_ENCODING` (version)
-   `CO_OPT_VAL_FULL` (version and encoding)
