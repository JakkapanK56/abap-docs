The example demonstrates a Simple Transformation used to create name attributes in [JSON data](ABENJSON_OVIEW.html).

This example works in a similar way to the [executable example](ABENABAP_ST_JSON_TABLE_ABEXA.html) for an internal table, but the values of the attributes `name` of the elements `object` in JSON-XML are also filled from the internal table in the Simple Transformation `DEMO_ST_JSON_TABLE_ATTRIBUTES`. The ST program is as follows:

<?sap.transform simple?>\\ \\n<tt:transform xmlns:tt="http://www.sap.com/transformation-templates">\\ \\n  <tt:root name="CARRIERS"/>\\ \\n  <tt:template>\\ \\n    <array>\\ \\n      <tt:loop ref=".CARRIERS">\\ \\n        <object>\\ \\n          <str>\\ \\n            <tt:attribute name="name">\\ \\n              <tt:value ref="$ref.carrname"/>\\ \\n            </tt:attribute>\\ \\n            <tt:value ref="$ref.url"/>\\ \\n          </str>\\ \\n        </object>\\ \\n      </tt:loop>\\ \\n    </array>\\ \\n  </tt:template>\\ \\n</tt:transform>
