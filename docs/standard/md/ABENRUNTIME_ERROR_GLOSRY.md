Caused by uncatchable exceptions when a program is executed, or it can be forced by the language elements [`RAISE SHORTDUMP`](ABAPRAISE_SHORTDUMP.html), [`THROW SHORTDUMP`](ABENCONDITIONAL_EXPRESSION_RESULT.html), or [`ASSERT`](ABAPASSERT.html) or by [exit messages](ABENEXIT_MESSAGE_GLOSRY.html). Every runtime error terminates the program, which in turn raises a [database rollback](ABENDATABASE_ROLLBACK_GLOSRY.html) and is documented by default in a [short dump](ABENSHORT_DUMP_GLOSRY.html).
