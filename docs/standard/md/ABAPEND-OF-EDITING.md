[Short Reference](ABAPEND-OF-EDITING_SHORTREF.html)

`END-OF-EDITING.`

The statement `END-OF-EDITING` defines an [event block](ABENEVENT_BLOCK_GLOSRY.html) in an [executable program](ABENEXECUTABLE_PROGRAM_GLOSRY.html). The event of the program is raised by the [ABAP runtime framework](ABENABAP_RUNTIME_FRMWK_GLOSRY.html) during the [executable program flow](ABENREPORTING_PROCESS.html) directly after [`END-OF-SELECTION`](ABAPEND-OF-SELECTION.html).

An implementation of the event block `END-OF-EDITING` is unnecessary.
