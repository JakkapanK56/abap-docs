An obsolete [formal parameter](ABENFORMAL_PARAMETER_GLOSRY.html) of [function modules](ABENFUNCTION_MODULE_GLOSRY.html) or [subroutines](ABENSUBROUTINE_GLOSRY.html) typed as a [standard table](ABENSTANDARD_TABLE_GLOSRY.html) with a [header line](ABENHEADER_LINE_GLOSRY.html). A table parameter is defined using the addition `TABLES` to the statements [`FUNCTION`](ABENFUNCTION.html) and [`FORM`](ABAPFORM.html).
