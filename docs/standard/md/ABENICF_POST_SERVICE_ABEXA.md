This example demonstrates calls of an HTTP service in [ICF](ABENICF_GLOSRY.html) and a Web browser. The input data is passed using the `POST` method.

This example has the same structure as the executable example [Calling an HTTP Service](ABENICF_SERVICE_ABEXA.html). In this case, the method `IF_HTTP_EXTENSION~HANDLE_REQUEST` of the class `CL_HTTP_EXT_SERVICE_POST_DEMO` is used to demonstrate how user input sent by a HTML page form generated by the handler class is sent using the `POST` method and read using the method `GET_CDATA`.

The HTTP service must be activated in transaction `SICF` before the example can work.
