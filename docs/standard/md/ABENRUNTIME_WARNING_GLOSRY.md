A runtime warning indicates critical situations that may cause a [runtime error](ABENRUNTIME_ERROR_GLOSRY.html) during program execution, such as high memory consumption. A runtime warning creates a [short dump](ABENSHORT_DUMP_GLOSRY.html), but does not terminate the current program.
