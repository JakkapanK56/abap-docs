Determines the order in which the binary representation of a number of the [data types](ABENDATA_TYPE_GLOSRY.html)\\ [`i`](ABENBUILTIN_TYPES_NUMERIC.html), [`int8`](ABENBUILTIN_TYPES_NUMERIC.html), [`decfloat16`](ABENBUILTIN_TYPES_NUMERIC.html), [`decfloat34`](ABENBUILTIN_TYPES_NUMERIC.html), [`f`](ABENBUILTIN_TYPES_NUMERIC.html), [`s`](ABENBUILTIN_TYPES_NUMERIC.html), or a character is stored in the memory. A distinction is made between [big](ABENBIG_ENDIAN_GLOSRY.html) and [little endian](ABENLITTLE_ENDIAN_GLOSRY.html). In the former case, the system writes the byte with the highest level to the first position in the memory and in the latter case it writes the byte with the lowest level.
