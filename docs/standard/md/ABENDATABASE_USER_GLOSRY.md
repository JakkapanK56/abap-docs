Person or system logged on to a [database system](ABENDATABASE_SYSTEM_GLOSRY.html) using a [database user name](ABENDATABASE_USER_NAME_GLOSRY.html). When the database is accessed using the [database connections](ABENDATABASE_CONNECTION_GLOSRY.html) of an [AS ABAP](ABENAS_ABAP_GLOSRY.html), the database user is a [work process](ABENWORK_PROCESS_GLOSRY.html) that implicitly accesses the [database schema](ABENDATABASE_SCHEMA_GLOSRY.html) assigned to the user name of the connection.
