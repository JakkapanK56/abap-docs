[Internal table](ABENINTERNAL_TABLE_GLOSRY.html) with the columns `SIGN`, `OPTION`, `LOW`, and `HIGH`. Declared with the addition [`TYPE RANGE OF`](ABAPTYPES_RANGES.html) of the statements [`DATA`](ABAPDATA.html) and [`TYPES`](ABAPTYPES.html), or the obsolete statement [`RANGES`](ABAPRANGES.html). A ranges table is used to store a [ranges condition](ABENRANGES_CONDITION_GLOSRY.html) internally. A ranges table can be evaluated in a [comparison expression](ABENRELATIONAL_EXPRESSION_GLOSRY.html) using the [comparison operator](ABENCOMP_OPERATOR_GLOSRY.html)\\ [`IN`](ABENLOGEXP_SELECT_OPTION.html), where each line of the table represents a separate comparison. A [selection table](ABENSELECTION_TABLE_GLOSRY.html) declared with [`SELECT-OPTIONS`](ABAPSELECT-OPTIONS.html) has the same structure as a ranges table.
