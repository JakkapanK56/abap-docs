Violation of the ABAP syntax rules. A program that has syntax errors cannot be executed. Syntax errors are returned after the [syntax check](ABENSYNTAX_CHECK_GLOSRY.html) that can be called in [ABAP Editor](ABENABAP_EDITOR_GLOSRY.html).
