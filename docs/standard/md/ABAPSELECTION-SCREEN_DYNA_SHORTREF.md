`SELECTION-SCREEN DYNAMIC SELECTIONS                   FOR \{NODE|TABLE\} node [ID id].`

**Obsolete:** Flags a node `node` of the structure of a [logical database](ABENLOGICAL_DATA_BASE_GLOSRY.html) for dynamic selections.

-   `ID id`\\
    Defines an ID `id` for the node, with a maximum of three characters.
