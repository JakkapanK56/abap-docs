[Type name](ABENTYPE_NAME_GLOSRY.html), constructed as a path in a [character string](ABENCHARACTER_STRING_GLOSRY.html), that uniquely specifies the [context](ABENOBJ_CONTEXT_GLOSRY.html) of a [data type](ABENDATA_TYPE_GLOSRY.html), a [class](ABENCLASS_GLOSRY.html), or an [interface](ABENOO_INTF_GLOSRY.html), and which can be used in dynamic type specifications. The [RTTS](ABENRUN_TIME_TYPE_SERVICES_GLOSRY.html) can determine the absolute type name of any [object](ABENOBJECT_GLOSRY.html). See also [relative type name](ABENRELATIVE_TYPE_NAME_GLOSRY.html).
