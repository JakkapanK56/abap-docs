Means of structuring a [processing block](ABENPROCESSING_BLOCK_GLOSRY.html) in [statement blocks](ABENSTATEMENT_BLOCK_GLOSRY.html). Possible control structures are [sequence](ABENSEQUENCE_GLOSRY.html), [branch](ABENBRANCH_GLOSRY.html), and [loop](ABENLOOP_GLOSRY.html) as well as special control structures for exception handling. Control structures can be nested, so that statement blocks can contain other control structures.
