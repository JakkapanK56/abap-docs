The example demonstrates the various time formats for embedded expressions.

An internal table `timetable` is filled with different times and these times are displayed in all possible [country-specific time formats](ABENCOUNTRY_FORMATS.html) using the parameter [`COUNTRY`](ABAPCOMPUTE_STRING_FORMAT_OPTIONS.html). For this purpose, the utility methods `setup` and `teardown` insert temporary lines with the possible values of the column `TIMEFM` into the database table `T005X` and delete them again once they have been used.
