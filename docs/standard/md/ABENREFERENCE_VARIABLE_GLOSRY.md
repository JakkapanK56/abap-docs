[Data object](ABENDATA_OBJECT_GLOSRY.html) that contains a [reference](ABENREFERENCE_GLOSRY.html). Reference variables are either [data reference variables](ABENDATA_REFERENCE_VARIABLE_GLOSRY.html) or [object reference variables](ABENOBJECT_REFER_VARIABLE_GLOSRY.html). Reference variables have a length of 8 bytes and are opaque, which means the reference cannot be accessed directly. They are typed with the addition [`REF TO`](ABAPTYPES_REFERENCES.html) that defines their [static type](ABENSTATIC_TYPE_GLOSRY.html). The static type is always more general than or the same as the [dynamic type](ABENDYNAMIC_TYPE_GLOSRY.html). For assignments between reference variables, [reference semantics](ABENREFERENCE_SEMANTICS_GLOSRY.html) apply.
