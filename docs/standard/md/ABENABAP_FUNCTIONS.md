Function modules are reusable cross-program [procedures](ABENPROCEDURE_GLOSRY.html) that are organized into [function pools](ABENFUNCTION_POOL_GLOSRY.html) and whose functionality is implemented between the following statements:

[FUNCTION](ABAPFUNCTION.html)\\ \\n  ... \\n[ENDFUNCTION](ABAPENDFUNCTION.html)

Function modules and their [interfaces](ABENFUNCTION.html) are created in the [Function Builder](ABENFUNCTION_BUILDER_GLOSRY.html) in the ABAP Workbench or in the [ABAP development tools for Eclipse (ADT)](ABENADT_GLOSRY.html).

Function modules in function pools are the predecessor technology of public [methods](ABENMETHOD_GLOSRY.html) in global classes.
