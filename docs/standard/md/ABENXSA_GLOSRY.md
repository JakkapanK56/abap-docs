Short for SAP HANA extended application services Advanced Model. Current version of [XS](ABENXS_GLOSRY.html) and successor to [XSC](ABENXSC_GLOSRY.html). [SAP Web IDE for SAP HANA](ABENSAP_WEB_IDE_FOR_SAP_HAN_GLOSRY.html) and [SAP HANA Cockpit](ABENSAP_HANA_COCKPIT_GLOSRY.html) can be used to develop and manage XSA objects. Objects are usually deployed using [HDI](ABENHDI_GLOSRY.html).
