`LEAVE \{ SCREEN |\ \{TO SCREEN dynnr\}\ \}.`

Exits the current [dynpro](ABENDYNPRO_GLOSRY.html) and continues processing using the [next dynpro](ABENNEXT_DYNPRO_GLOSRY.html).

-   `TO SCREEN dynnr`\\
    Specifies the next dynpro in `dynnr`.
