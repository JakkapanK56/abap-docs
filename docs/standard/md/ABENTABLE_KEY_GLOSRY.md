Columns of an [internal table](ABENINTERNAL_TABLE_GLOSRY.html) or [DDIC database table](ABENDDIC_DB_TABLE_GLOSRY.html) whose content identifies table lines. For internal tables, distinction is made between [primary table keys](ABENPRIMARY_TABLE_KEY_GLOSRY.html) and [secondary table keys](ABENSECONDARY_TABLE_KEY_GLOSRY.html), which are [technical type properties](ABENTECHNICAL_TYPE_PRPT_GLOSRY.html) of the [table type](ABENTABLE_TYPE_GLOSRY.html).
