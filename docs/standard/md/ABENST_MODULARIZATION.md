An ST program can contain any number of templates that are defined with `tt:template`. All templates but the [main template](ABENST_TT_TEMPLATE_MAIN.html) are subtemplates that can be called in the main template.

-   [`tt:template`, `tt:context`, Create Subtemplate](ABENST_TT_TEMPLATE_SUB.html)
-   [`tt:apply`, Call Subtemplate](ABENST_TT_APPLY.html)
