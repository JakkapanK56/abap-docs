Database-specific [SQL](ABENSQL_GLOSRY.html) statements that are handled by the [Native SQL interface](ABENNATIVE_SQL_INTERFACE_GLOSRY.html) of the [database interface](ABENDATABASE_INTERFACE_GLOSRY.html). Native SQL is either passed from [ABAP programs](ABENABAP_PROGRAM_GLOSRY.html) to the Native SQL interface using [objects](ABENOBJECT_GLOSRY.html) from the [ADBC](ABENADBC_GLOSRY.html)\\ [classes](ABENCLASS_GLOSRY.html) or specified statically between the statements [`EXEC SQL`](ABAPEXEC.html) and [`ENDEXEC`](ABAPENDEXEC.html). [ABAP Managed Database Procedures (AMDP)](ABENABAP_MANAGED_DB_PROC_GLOSRY.html) use Native SQL to call and manage [database procedures](ABENDATABASE_PROCEDURE_GLOSRY.html) and [database functions](ABENDATABASE_FUNCTION_GLOSRY.html) with platform-dependent implementations.
