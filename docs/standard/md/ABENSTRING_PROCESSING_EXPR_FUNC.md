Expressions and functions for character string processing can be listed at many [operand positions](ABENEXPRESSION_POSITIONS.html).

An advantage of the expressions and functions for string processing over the [statements for character and byte string processing](ABENSTRING_PROCESSING_STATEMENTS.html) is that expressions can be specified for all operands, which is only possible to a very limited extent in the statements.

-   [String Expressions (`string_exp`)](ABAPCOMPUTE_STRING.html)
-   [Character String Functions (`cstring_func`)](ABENSTRING_FUNCTIONS.html)
-   [Regular Expressions (regex)](ABENREGULAR_EXPRESSIONS.html)
