Separator between the integer part and fractional part of a [decimal number](ABENDECIMAL_NUMBER_GLOSRY.html). Within an [ABAP program](ABENABAP_PROGRAM_GLOSRY.html), the [decimal point](ABENDECIMAL_POINT_GLOSRY.html) is always used. The representation of the decimal separator in the output depends on according to the current [formatting setting](ABENCOUNTRY.html) of the [language environment](ABENLANGUAGE_ENVIRONMENT_GLOSRY.html) that can be set using [`SET COUNTRY`](ABAPSET_COUNTRY.html). See also [decimal comma](ABENDECIMAL_COMMA_GLOSRY.html) and [decimal point](ABENDECIMAL_POINT_GLOSRY.html).
