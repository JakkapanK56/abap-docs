Global subclass of `CL_ABAP_DAEMON_EXT_BASE`. The instances of an ABAP daemon class are [ABAP daemons](ABENABAP_DAEMON_GLOSRY.html) created by the [ABAP daemon framework](ABENABAP_DAEMON_FRAMEWORK_GLOSRY.html), each of which are executed in a separate [ABAP daemon session](ABENABAP_DAEMON_SESSION_GLOSRY.html). An ABAP daemon class implements interface methods inherited from the superclass to react to [ABAP daemon events](ABENABAP_DAEMON_EVENT_GLOSRY.html).
