CDS DDL expressions are expressions that can be specified in a CDS DDL operand position of a [CDS DDIC-based view (obsolete)](ABENCDS_V1_VIEW_GLOSRY.html).

The following CDS DDL expressions exist:

-   [Arithmetic expressions](ABENCDS_ARITHMETIC_EXPRESSION_V1.html)
-   [Aggregate expressions](ABENCDS_AGGREGATE_FUNCTIONS_V1.html)
-   [Case distinctions](ABENCDS_CASE_EXPRESSION_V1.html)
-   [Cast expressions](ABENCDS_CAST_EXPRESSION_V1.html)
-   [Calls of built-in CDS functions](ABENCDS_BUILTIN_FUNCTIONS_V1.html)
