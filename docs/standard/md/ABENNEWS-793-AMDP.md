The new addition [`CLIENT INDEPENDENT`](ABAPMETHODS_AMDP_OPTIONS_CLIENT.html) can be used after [`AMDP OPTIONS`](ABAPMETHODS_AMDP_OPTIONS.html) to declare an [AMDP method](ABENAMDP_METHOD_GLOSRY.html) as [client-independent](ABENCLIENT_INDEPENDENT_GLOSRY.html). When used, only client-independent database objects can be accessed and specified after the addition [`USING`](ABAPMETHOD_BY_DB_PROC.html).
