Any statements that block the program flow are not allowed in an [ABAP programs](ABENABAP_PROGRAM_GLOSRY.html) that is processed in non-blocking mode. There are values of the non-blocking mode for [APC](ABENAPC_NON_BLOCKING_MODEL.html) and [ADF](ABENADF_NON_BLOCKING_MODEL.html).
