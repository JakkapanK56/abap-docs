String or sequence of bytes. Content of any data object. However, direct access to the bytes of a byte chain is possible only in [byte-like data objects](ABENBYTE_LIKE_DATA_OBJECT_GLOSRY.html), or by using appropriately typed [field symbols](ABENFIELD_SYMBOL_GLOSRY.html).
