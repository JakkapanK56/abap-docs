Principle of software architecture in which a single software instance is run for different [tenants](ABENTENANT_GLOSRY.html) on a single server. In a multitenancy [AS ABAP](ABENAS_ABAP_GLOSRY.html), [application clients](ABENAPPLICATION_CLIENT_GLOSRY.html) assume the role of tenants. Multitenancy is not supported by the [ABAP runtime framework](ABENABAP_RUNTIME_FRMWK_GLOSRY.html) in the current release.
