Grouping of [data objects](ABENDATA_OBJECT_GLOSRY.html) for transient and persistent storage in a storage medium. A data cluster can be processed using the statements [`IMPORT`](ABAPIMPORT_DATA_CLUSTER.html), [`EXPORT`](ABAPEXPORT_DATA_CLUSTER.html), and [`DELETE FROM`](ABAPDELETE_CLUSTER.html).
