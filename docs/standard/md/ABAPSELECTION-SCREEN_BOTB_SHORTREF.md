[Reference](ABAPSELECTION-SCREEN_TABBED.html)

`SELECTION-SCREEN BEGIN OF TABBED BLOCK tblock FOR n LINES [NO INTERVALS].  ...  SELECTION-SCREEN END OF BLOCK tblock.`

Defines a [tabstrip](ABENTABSTRIP_CONTROL_GLOSRY.html) control or [subscreen](ABENSUBSCREEN_ARREA_GLOSRY.html) area named `tblock` on the current [selection screen](ABENSELECTION_SCREEN_GLOSRY.html).

-   `FOR n LINES`\\
    Defines the number of lines in the tabstrip control in `n`.
-   `NO INTERVALS`\\
    Creates a narrower area.
