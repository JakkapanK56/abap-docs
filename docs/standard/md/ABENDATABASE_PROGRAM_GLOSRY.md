[ABAP program](ABENABAP_PROGRAM_GLOSRY.html) of a [logical database](ABENLOGICAL_DATA_BASE_GLOSRY.html), which provides other programs with data in a [table work area](ABENTABLE_WORK_AREA_GLOSRY.html) and, in doing so, raises the [reporting event](ABENREPORTING_EVENT_GLOSRY.html)\\ [`GET`](ABAPGET-.html) with the statement [`PUT`](ABAPPUT.html). The functionality of a database program is implemented in [subroutines](ABENSUBROUTINE_GLOSRY.html), which are called by the [ABAP runtime framework](ABENABAP_RUNTIME_FRMWK_GLOSRY.html) during the execution of an [executable program](ABENEXECUTABLE_PROGRAM_GLOSRY.html) or the [function module](ABENFUNCTION_MODULE_GLOSRY.html)\\ `LDB_PROCESS`. The execution time of the individual subroutines is related to certain reporting and [selection screen events](ABENSELECTION_SCREEN_EVENT_GLOSRY.html).
