Special [ABAP program](ABENABAP_PROGRAM_GLOSRY.html) that provides other ABAP programs with data from the nodes of a hierarchical tree structure (obsolete). Is either linked to an [executable program](ABENEXECUTABLE_PROGRAM_GLOSRY.html) in its [program properties](ABENPROGRAM_PROPERTY_GLOSRY.html) or is called using [function module](ABENFUNCTION_MODULE_GLOSRY.html)\\ `LDB_PROCESS`. A logical database is made up of a hierarchical structure with nodes, an ABAP [database program](ABENDATABASE_PROGRAM_GLOSRY.html) and its own [standard selection screen](ABENSTANDARD_SELSCREEN_GLOSRY.html). Logical databases are maintained in the [Logical Database Builder](ABENLOGICAL_DATABASE_BUILD_GLOSRY.html).
