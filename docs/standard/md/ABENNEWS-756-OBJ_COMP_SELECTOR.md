The following syntax can be used for the [object component selector `->`](ABENOBJECT_COMPONENT_SELECTOR.html) to access components and attributes dynamically now:

`... \{ dref->(comp_name) \}\ |\ \{ cref->(attr_name) \}\ |\ \{ iref->(attr_name) \} ...`

Before, this was possible for [dynamic components](ABAPASSIGN_DYNAMIC_COMPONENTS.html) and [dynamic access](ABAPASSIGN_MEM_AREA_DYNAMIC_ACCESS.html) in the statement `ASSIGN` only.
