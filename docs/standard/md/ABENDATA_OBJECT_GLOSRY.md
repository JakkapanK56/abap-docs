[Instance](ABENINSTANCE_GLOSRY.html) of a [data type](ABENDATA_TYPE_GLOSRY.html). Exists in the [internal session](ABENINTERNAL_SESSION_GLOSRY.html) of an [ABAP program](ABENABAP_PROGRAM_GLOSRY.html) or as a [shared object](ABENSHARED_OBJECT_GLOSRY.html) in the [shared memory](ABENSHARED_MEMORY_GLOSRY.html). Is either created as a [named](ABENNAMED_DATA_OBJECT_GLOSRY.html), statically declared data object (statement [`DATA`](ABAPDATA.html) and similar) when a program or [procedure](ABENPROCEDURE_GLOSRY.html) is loaded, or dynamically as an [anonymous data object](ABENANONYMOUS_DATA_OBJECT_GLOSRY.html) at runtime ( statement [`CREATE DATA`](ABAPCREATE_DATA.html) or [instance operator](ABENINSTANCE_OPERATOR_GLOSRY.html)\\ [`NEW`](ABENCONSTRUCTOR_EXPRESSION_NEW.html)). In addition, the [literals](ABENABAP_LITERAL_GLOSRY.html) defined as part of the source code are also handled as data objects. The data type of a data object is always complete (not [generic](ABENGENERIC_DATA_TYPE_GLOSRY.html)) and can be [bound](ABENBOUND_DATA_TYPE_GLOSRY.html) or [standalone](ABENSTAND-ALONE_DATA_TYPE_GLOSRY.html). See also [static data object](ABENSTATIC_DATA_OBJECT_GLOSRY.html) and [dynamic data object](ABENDYNAMIC_DATA_OBJECT_GLOSRY.html).
