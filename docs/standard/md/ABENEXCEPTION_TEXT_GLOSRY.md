Text that describes an exception and is specified in an [exception class](ABENEXCEPTION_CLASS_GLOSRY.html). An exception text can be given parameters using the attributes of the exception class. Exception texts are usually defined with reference to [messages](ABENMESSAGE_GLOSRY.html) and in [system classes](ABENSYSTEM_CLASS_GLOSRY.html) with reference to the Online Text Repository (OTR).
