[Enumerated type](ABENENUM_TYPE_GLOSRY.html) in ABAP. An [enumerated variable](ABENENUMERATED_VARIABLE_GLOSRY.html) of an enumerated type can only accept the [enumerated values](ABENENUMERATED_VALUE_GLOSRY.html) defined there that exist as the content of [enumerated constants](ABENENUMERATED_CONSTANT_GLOSRY.html) or components of an [enumerated structure](ABENENUMERATED_STRUCTURE_GLOSRY.html). An enumerated type is defined with [`TYPES BEGIN OF ENUM`](ABAPTYPES_ENUM.html).
