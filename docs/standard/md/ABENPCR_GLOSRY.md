PCR for short. A parent-child relationship is a directed [hierarchy edge](ABENHIERARCHY_EDGE_GLOSRY.html) between two [hierarchy nodes](ABENHIERARCHY_NODE_GLOSRY.html) that defines the type of the nodes. A hierarchy node with outbound edges is a [parent node](ABENPARENT_NODE_GLOSRY.html). A hierarchy node with inbound edges is a [child node](ABENPARENT_NODE_GLOSRY.html). A hierarchy node with no inbound edges is a [root node](ABENROOT_NODE_GLOSRY.html) or an [orphan node](ABENORPHAN_NODE_GLOSRY.html). A hierarchy node with no outbound edges is a [leaf node](ABENLEAF_NODE_GLOSRY.html).
