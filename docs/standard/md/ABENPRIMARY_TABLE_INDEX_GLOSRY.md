[Table index](ABENTABLE_INDEX_GLOSRY.html) of [index tables](ABENINDEX_TABLE_GLOSRY.html) ([standard tables](ABENSTANDARD_TABLE_GLOSRY.html) and [sorted tables](ABENSORTED_TABLE_GLOSRY.html)). Every index table has a primary table index, administrated by the [ABAP runtime framework](ABENABAP_RUNTIME_FRMWK_GLOSRY.html), which contains a unique line number for each table line. The primary table index of an index table is updated each time a line is added or deleted from an internal table.
