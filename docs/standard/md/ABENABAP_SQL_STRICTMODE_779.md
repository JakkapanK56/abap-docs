The strict mode of the syntax check in ABAP release 7.79 applies to all ABAP SQL statements that use one of the following features introduced in [ABAP release 7.79](ABENNEWS-779-ABAP_SQL.html):

The strict mode in ABAP release 7.79 covers all rules of the [strict mode in ABAP release 7.78](ABENABAP_SQL_STRICTMODE_778.html).

-   Use of the window function [`NTILE`](ABENSQL_WIN_FUNC.html)
-   Use of the SQL function [`TO_CLOB`](ABENSQL_TYPE_CONV_FUNC.html)
