An event can trigger the processing of a [processing block](ABENPROCESSING_BLOCK_GLOSRY.html), without actually knowing the processing block.

Either a component of a [class](ABENCLASS_GLOSRY.html) or an [interface](ABENOO_INTF_GLOSRY.html) declared using [`[CLASS-]`](ABAPCLASS-EVENTS.html)[`EVENTS`](ABAPEVENTS.html), or an event of the [ABAP runtime framework](ABENABAP_RUNTIME_FRMWK_GLOSRY.html). Events in classes trigger [event handlers](ABENEVENT_HANDLER_GLOSRY.html).

Events of the [ABAP runtime framework](ABENABAP_RUNTIME_FRMWK_GLOSRY.html) trigger the processing of [event blocks](ABENEVENT_BLOCK_GLOSRY.html).
