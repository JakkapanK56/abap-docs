SAPUI5 is a tool set for developing user interfaces for Web applications in HTML5. SAPUI5 supports the Model View Controller (MVC) approach and is based on the user interface elements delivered in JavaScript and CSS libraries. The data connection to the application server is established using the data formats [XML](ABENXML_GLOSRY.html), [JSON](ABENJSON_GLOSRY.html) or OData. Interfaces developed with SAPUI5 can be connected to SAP and non-SAP platforms. In particular, SAPUI5 user interfaces can be developed for ABAP applications in [AS ABAP](ABENAS_ABAP_GLOSRY.html). The connection is then made using [ICF](ABENICF_GLOSRY.html) and [APC](ABENAPC_GLOSRY.html).
