A method of passing data from [actual parameters](ABENACTUAL_PARAMETER_GLOSRY.html) to [formal parameters](ABENFORMAL_PARAMETER_GLOSRY.html) when a [procedure](ABENPROCEDURE_GLOSRY.html) is called, which is defined in the [parameter interface](ABENPARAMETER_INTERFACE_GLOSRY.html) of the procedure. Pass by reference does not create a local [data object](ABENDATA_OBJECT_GLOSRY.html) for the actual parameter. Instead a reference to the actual parameter is passed to the procedure and it works with the actual parameter itself. [Input parameters](ABENINPUT_PARAMETER_GLOSRY.html) passed by reference must not be modified in the procedure. See also [pass by value](ABENPASS_BY_VALUE_GLOSRY.html).
