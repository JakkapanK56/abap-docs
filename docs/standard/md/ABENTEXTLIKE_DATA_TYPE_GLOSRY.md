[Data type](ABENDATA_TYPE_GLOSRY.html) whose [data objects](ABENDATA_OBJECT_GLOSRY.html) contain texts. The corresponding [built-in data types](ABENBUILTIN_DATA_TYPE_GLOSRY.html) are [`c`](ABENBUILTIN_TYPES_CHARACTER.html) and [`string`](ABENBUILTIN_TYPES_CHARACTER.html). The corresponding [generic data type](ABENGENERIC_DATA_TYPE_GLOSRY.html) is [`csequence`](ABENBUILT_IN_TYPES_GENERIC.html).
