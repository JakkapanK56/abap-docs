[ABAP keyword](ABENABAP_KEYWORD_GLOSRY.html) or combination of an ABAP keyword with one or more following [ABAP words](ABENABAP_WORD_GLOSRY.html) or [ABAP operands](ABENABAP_OPERAND_GLOSRY.html). An ABAP language element uniquely describes the semantics of an [ABAP statement](ABENABAP_STATEMENT_GLOSRY.html). Examples of different ABAP language elements are `GET BIT`, `GET TIME`, `DELETE itab`, or `DELETE dbtab`. ABAP language elements can be enhanced using [ABAP language element additions](ABENABAP_LANGUAGE_ELMNT_ADD_GLOSRY.html).
