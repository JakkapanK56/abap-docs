`DATA BEGIN OF COMMON PART [name].    ...    DATA ...    ...  DATA END OF COMMON PART [name].`

**Obsolete:** Defines a global [interface work area](ABENINTERFACE_WORK_AREA_GLOSRY.html) that can be shared by the programs in a [program group](ABENPROGRAM_GROUP_GLOSRY.html).

-   `name`\\
    Defines a name for the interface work area.
