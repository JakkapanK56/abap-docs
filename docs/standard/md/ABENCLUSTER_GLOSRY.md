Grouping of different data in shared storage. Currently only used as [data clusters](ABENDATA_CLUSTER_GLOSRY.html) in different storage media for [data objects](ABENDATA_OBJECT_GLOSRY.html). Before [ABAP release](ABENABAP_RELEASE_GLOSRY.html) 7.73, also used as a [table cluster](ABENTABLE_CLUSTER_GLOSRY.html) for special [DDIC database tables](ABENDDIC_DB_TABLE_GLOSRY.html) ([cluster tables](ABENCLUSTER_TABLE_GLOSRY.html)).
