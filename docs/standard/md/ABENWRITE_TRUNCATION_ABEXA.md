This example demonstrates the [truncation behavior](ABENWRITE_TRUNCATIONS.html) of the statement [`WRITE TO`](ABAPWRITE_TO.html) for all elementary data types.

The class `demo` contains an attribute for every [built-in ABAP type](ABENBUILT_IN_TYPES_COMPLETE.html) and for [time stamps in packed numbers](ABENTIME_STAMPS_PACKED.html). These attributes are filled with example values and assigned to a data object using [`WRITE TO`](ABAPWRITE_TO.html), whose length can be selected on the selection screen.

The [country-specific formats](ABENCOUNTRY_FORMATS.html) for numbers, dates, and times set by [`SET COUNTRY`](ABAPSET_COUNTRY.html) can also be specified on the selection screen. This example demonstrates these formats as well.

The output list shows the result of the formatting for each data type. If an exception is raised, the character *!* is output in a different color.
