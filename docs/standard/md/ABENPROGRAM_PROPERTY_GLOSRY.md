Property of an [ABAP program](ABENABAP_PROGRAM_GLOSRY.html). Program properties are used to administrate the program and affect how it is handled in [syntax checks](ABENSYNTAX_CHECK_GLOSRY.html) and in the [ABAP runtime framework](ABENABAP_RUNTIME_FRMWK_GLOSRY.html). Examples of important program properties are [program type](ABENPROGRAM_TYPE_GLOSRY.html) or the [ABAP language version](ABENABAP_VERSION_GLOSRY.html), which specifies whether or not a program has the ABAP language version [Standard ABAP](ABENSTANDARD_ABAP_GLOSRY.html).
