[Database connection](ABENDATABASE_CONNECTION_GLOSRY.html) between an [AS ABAP](ABENAS_ABAP_GLOSRY.html) and its [standard database](ABENSTANDARD_DB_GLOSRY.html). A standard connection is always opened for each [work process](ABENWORK_PROCESS_GLOSRY.html) of an AS ABAP. The [ABAP database schema](ABENDATABASE_USER_GLOSRY.html) is assigned to the [database user](ABENABAP_DB_SCHEMA_GLOSRY.html) of the standard connection. By default, reads in [ABAP SQL](ABENABAP_SQL_GLOSRY.html), [Native SQL](ABENNATIVE_SQL_GLOSRY.html), and [AMDP](ABENAMDP_GLOSRY.html) are performed on this [database schema](ABENDATABASE_SCHEMA_GLOSRY.html) using this standard connection.
