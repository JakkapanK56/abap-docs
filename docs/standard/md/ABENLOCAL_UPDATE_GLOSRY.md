[Update](ABENUPDATE_GLOSRY.html) in the current [work process](ABENWORK_PROCESS_GLOSRY.html). It is not executed in a separate [update session](ABENUPDATE_SESSION_GLOSRY.html) but in a separate [internal session](ABENINTERNAL_SESSION_GLOSRY.html) of the current [ABAP session](ABENABAP_SESSION_GLOSRY.html). The local update can be activated for an [SAP LUW](ABENSAP_LUW_GLOSRY.html) by using the statement [`SET UPDATE TASK LOCAL`](ABAPSET_UPDATE_TASK_LOCAL.html).
