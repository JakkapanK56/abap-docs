A structure can be displayed and modified using [dictionary DDL](ABENDICTIONARY_DDL_GLOSRY.html) in the [ABAP development tools for Eclipse (ADT)](ABENADT_GLOSRY.html). The source code of a structure is not stored and transported. Instead, the source code of an existing structure is generated from the metadata of the associated system tables. When saved, the metadata is derived directly from the source code and stored in the system tables.

All properties of a structure described in the preceding sections can be defined using `DEFINE STRUCTURE` or in transaction `SE11`.

-   [`DEFINE STRUCTURE`](ABENDDICDDL_DEFINE_STRUCTURE.html)
