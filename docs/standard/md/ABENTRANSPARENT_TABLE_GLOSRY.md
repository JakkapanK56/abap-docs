[Table category](ABENDDIC_DATABASE_TABLES_TAB_CAT.html) of a [DDIC database table](ABENDDIC_DB_TABLE_GLOSRY.html) in the [ABAP Dictionary](ABENABAP_DICTIONARY_GLOSRY.html) that has exactly one identically named instance with the same columns as the definition in the ABAP Dictionary. Besides using [ABAP SQL](ABENABAP_SQL_GLOSRY.html), the data of transparent tables can also be processed using [Native SQL](ABENNATIVE_SQL_GLOSRY.html) and [AMDP](ABENAMDP_GLOSRY.html) and outside [AS ABAP](ABENAS_ABAP_GLOSRY.html) by using the programming interface of the database. The latter is [not recommended](ABENDATABASE_ACCESS_RECOMM.html).
