Special [data objects](ABENDATA_OBJECT_GLOSRY.html) that can be used as a cross-program [interface](ABENINTF_GEN_GLOSRY.html) between programs and [dynpros](ABENDYNPRO_GLOSRY.html), and between programs and [logical databases](ABENLOGICAL_DATA_BASE_GLOSRY.html). Interface work areas are either [table work areas](ABENTABLE_WORK_AREA_GLOSRY.html) or [common areas](ABENCOMMON_AREA_GLOSRY.html) (obsolete). All programs in a [program group](ABENPROGRAM_GROUP_GLOSRY.html) access the same data of an interface work area.
