bgRFC for short. Asynchronous remote function call for which the called [RFMs](ABENRFM_GLOSRY.html) can be executed transactionally and in a predefined call sequence. bgRFC replaces [tRFC](ABENTRFC_2_GLOSRY.html) and [qRFC](ABENQRFC_GLOSRY.html) apart from the no-send scenario of qRFC. The latter is replaced by the [local data queue (LDQ)](ABENLOCAL_DATA_QUEUE_GLOSRY.html).
