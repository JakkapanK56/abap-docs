Execution of [update function modules](ABENUPDATE_FUNCTION_MODULE_GLOSRY.html) that are registered using [`CALL FUNCTION ... IN UPDATE TASK`](ABAPCALL_FUNCTION_UPDATE.html) and triggered by [`COMMIT WORK`](ABAPCOMMIT.html). Updates are used to bundle the changing database accesses of an [SAP LUW](ABENSAP_LUW_GLOSRY.html) in a [database LUW](ABENDATABASE_LUW_GLOSRY.html). See also [synchronous](ABENSYNCHRONOUS_UPDATE_GLOSRY.html), [asynchronous](ABENASYNCHRONOUS_UPDATE_GLOSRY.html), and [local updates](ABENLOCAL_UPDATE_GLOSRY.html).
