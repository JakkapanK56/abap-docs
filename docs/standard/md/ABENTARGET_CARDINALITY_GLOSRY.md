The [cardinality](ABENCARDINALITY_GLOSRY.html) of a [join](ABENJOIN_GLOSRY.html) or [association](ABENASSOCIATION_GLOSRY.html) can consist of a [source cardinality](ABENSOURCE_CARDINALITY_GLOSRY.html) and a target cardinality. The target cardinality describes the number of matching entries for the entries of the original data source. For example, a target cardinality of many means that each record of the original data source has multiple matching entries in the joined data source.
