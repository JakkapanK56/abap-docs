Before [ABAP release](ABENABAP_RELEASE_GLOSRY.html) 7.73, [DDIC database table](ABENDDIC_DB_TABLE_GLOSRY.html) in the [database](ABENDATABASE_GLOSRY.html) that contains the data of multiple [cluster tables](ABENCLUSTER_TABLE_GLOSRY.html). Cluster tables are no longer supported and all table clusters have been removed.
