Defines the [behavior extension body](ABENRAP_BEHAVIOR_EXT_BODY_GLOSRY.html) to extend an existing [RAP BO entity](ABENRAP_BO_ENTITY_GLOSRY.html) in the [RAP BDL](ABENCDS_BDL_GLOSRY.html).
