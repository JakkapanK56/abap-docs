16-bit [Unicode character representation](ABENUNICODE_CHAR_REPRESENT_GLOSRY.html). In this representation, all characters are represented using 2 bytes or 4 bytes. UTF-16 is the [system code page](ABENSYSTEM_CODEPAGE_GLOSRY.html) of a [Unicode system](ABENUNICODE_SYSTEM_GLOSRY.html). UTF-16 uses the same 2-byte encoding as [UCS-2](ABENUCS2_GLOSRY.html) for the Basic Multilingual Plane (BMP) of the Unicode standard and a 4-byte encoding for the other Unicode planes in the [surrogate area](ABENSURROGATE_AREA_GLOSRY.html) that are addressed by [surrogates](ABENSURROGATES_GLOSRY.html). The ABAP programming language supports UCS-2 as a subset of UTF-16 without surrogates.
