In [CDS view entities](ABENCDS_V2_VIEW_GLOSRY.html), the [`SUBSTRING`](ABENCDS_SQL_FUNCTIONS_CHARACTER_V2.html) function has been enhanced. It now accepts not only literals, but also fields, parameters, expressions, and built-in functions as arguments for the parameters `pos` and `len`.
