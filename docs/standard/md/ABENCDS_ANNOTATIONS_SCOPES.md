The [annotation syntax](ABENCDS_ANNOTATIONS_SYNTAX.html) for [CDS annotations](ABENCDS_ANNOTATIONS.html) can be specified in ABAP CDS source code in scopes that are defined with the [`@Scope`](ABENCDS_1049724201-_ANNO.html) annotation in the corresponding annotation definition.

The following table shows the enumeration symbols that can be used as annotation values for [`@Scope`](ABENCDS_1049724201-_ANNO.html) and the corresponding source code positions.

The annotations specified in the source code of a CDS entity can be annotated or overridden using the annotations of [CDS metadata extensions](ABENCDS_METADATA_EXTENSION_GLOSRY.html).
