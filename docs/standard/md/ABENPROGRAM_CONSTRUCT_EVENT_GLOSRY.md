[ABAP runtime framework](ABENABAP_RUNTIME_FRMWK_GLOSRY.html)\\ [event](ABENEVENT_GLOSRY.html) that occurs once only when an [ABAP program](ABENABAP_PROGRAM_GLOSRY.html) (not a [class pool](ABENCLASS_POOL_GLOSRY.html)) is loaded in the [internal session](ABENINTERNAL_SESSION_GLOSRY.html) to call the [program constructor](ABENPROGRAM_CONSTRUCTOR_GLOSRY.html). The corresponding [event keyword](ABENEVENT_KEYWORD_GLOSRY.html) is [`LOAD-OF-PROGRAM`](ABAPLOAD-OF-PROGRAM.html).
