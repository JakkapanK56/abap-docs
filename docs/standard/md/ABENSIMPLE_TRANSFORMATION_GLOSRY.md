ST for short. SAP-specific programming language for transformations between [XML](ABENXML_GLOSRY.html) formats and ABAP data and vice versa. [Serializations](ABENSERIALIZATION_GLOSRY.html) and [deserializations](ABENDESERIALIZATION_GLOSRY.html) can be carried out using ST. Simple Transformations are edited using the [Transformation Editor](ABENTRANSFORMATION_EDITOR_GLOSRY.html) and are called using the statement [`CALL TRANSFORMATION`](ABAPCALL_TRANSFORMATION.html).
