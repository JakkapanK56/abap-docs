[Test classes](ABENTEST_CLASS_GLOSRY.html) and their components cannot be addressed in the production code of programs and must be addressed only in other test classes. A static check and a full check at runtime are now applied to verify this. Until now it was possible to address certain components of test class in production code. In production systems, this required the [profile parameter](ABENPROFILE_PARAMETER_GLOSRY.html)\\ `abap/test_generation` to be set accordingly.
