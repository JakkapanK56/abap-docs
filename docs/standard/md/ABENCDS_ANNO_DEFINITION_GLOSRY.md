Defines a [CDS annotation](ABENCDS_ANNOTATION_GLOSRY.html) as a [CDS object](ABENCDS_OBJECT_GLOSRY.html) using the statement [`DEFINE ANNOTATION`](ABENCDS_F1_DEFINE_ANNOTATION.html) in [`DDLA` source code](ABENDDLA_SOURCE_CODE_GLOSRY.html) in [CDS DDL](ABENCDS_DDL_GLOSRY.html) in [ABAP CDS](ABENABAP_CDS_GLOSRY.html). A CDS annotation definition specifies the [CDS annotation syntax](ABENCDS_ANNOTATION_SYNTAX_GLOSRY.html) that is to be used when the annotation is specified in [CDS source code](ABENCDS_SOURCE_CODE_GLOSRY.html).
