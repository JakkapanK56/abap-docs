[Reference](ABAPLOOP_AT_DBTAB.html)

`LOOP AT dbtab [VERSION vers].    ...  ENDLOOP.`

**Obsolete:** Reads multiple rows of a database table or view `dbtab` in a loop. The content of the [table work area](ABENTABLE_WORK_AREA_GLOSRY.html)\\ `dbtab` is used as the search key.

-   `VERSION vers`\\
    Specifies a different database table in `vers`.
