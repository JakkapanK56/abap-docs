The example demonstrates [arithmetic calculations](ABENARITH_OPERATORS.html) with floating point numbers.

Two operands and different arithmetic operators can be entered. The calculation is performed for the data types `decfloat34`, `decfloat16`, and `f`. For decimal floating point numbers, the losslessness of the calculation is checked using the [lossless operator](ABENLOSSLESS_OPERATOR_GLOSRY.html)\\ [`EXACT`](ABENCONSTRUCTOR_EXPRESSION_EXACT.html).
