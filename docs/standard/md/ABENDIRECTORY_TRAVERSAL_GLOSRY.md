Unauthorized access to files or directories. Directory traversals are possible in ABAP if physical file names or file paths that are transferred from outside ABAP to programs are then passed on to the [ABAP file interface](ABENFILE_INTERFACE_GLOSRY.html) without being checked or escaped. Directory traversals must be prevented by validating file names.
