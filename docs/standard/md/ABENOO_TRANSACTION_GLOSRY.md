The [transaction code](ABENTRANSACTION_CODE_GLOSRY.html) of an object transaction is linked with a [method](ABENMETHOD_GLOSRY.html) of a local or global [class](ABENCLASS_GLOSRY.html). When the transaction is called, the associated program is loaded, for [instance methods](ABENINSTANCE_METHOD_GLOSRY.html) an [object](ABENOBJECT_GLOSRY.html) of the class is created, and the method is executed. Object transactions can be connected to a [transaction service](ABENTRANSACTION_SERVICE_GLOSRY.html) of the [Object Services](ABENOBJECT_SERVICES_GLOSRY.html).
