The new addition [`AS BITFIELD`](ABAPTYPES_INDICATORS.html) behind addition [`INDICATORS`](ABAPTYPES_INDICATORS.html) of the statement [`TYPES`](ABAPTYPES.html) allows a [condensed indicator structure](ABENCONDENSED_IND_STRUCTURE_GLOSRY.html) to be defined as a [byte field type](ABENBYTE_FIELD_TYPE_GLOSRY.html) component of a given [structured type](ABENSTRUCTURED_TYPE_GLOSRY.html). An condensed indicator structure can be used as a [null indicator](ABENNULL_INDICATOR_GLOSRY.html) in ABAP SQL statements.
