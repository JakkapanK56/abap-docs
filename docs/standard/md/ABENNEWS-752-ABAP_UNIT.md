The special [ABAP Doc comment](ABENABAP_DOC_COMMENT_GLOSRY.html)\\ [`"! @testing ...`](ABENTEST_RELATIONS.html) can be used to define [test relations](ABENTEST_RELATION_GLOSRY.html) between test classes or test methods and repository objects. This allows the display and execution of tests for these repository objects in [ADT](ABENADT_GLOSRY.html).
