The [ABAP runtime framework](ABENABAP_RUNTIME_FRMWK_GLOSRY.html) contains an [SAP XSLT processor](ABENXSLT_PROCESSOR_GLOSRY.html) for executing the transformations. It supports almost all XSLT statements and implements enhancements (extension instructions and extension functions) such as calling ABAP methods from XSLT programs.
