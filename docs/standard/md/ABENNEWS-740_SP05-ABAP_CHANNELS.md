In ABAP release 7.40, two new communication channels were introduced, which can also be connected to each other. ABAP Messaging Channels can be used from [SP02](ABENNEWS-740-ABAP_CHANNELS.html).

[1\. ABAP Push Channels](#ABAP_MODIFICATION_1@4@)

[2. New Variant of `WAIT UNTIL`](#ABAP_MODIFICATION_2@4@)

[ABAP Push Channels (APC)](ABENAPC.html) enable bidirectional communication between AS ABAP and the internet using the WebSocket protocol.

The new variant [`WAIT FOR PUSH CHANNELS`](ABAPWAIT_APC.html) waits for APC messages in [ABAP Push Channels (APC)](ABENAPC.html).
