[1\. Empty Key of Table Types](#ABAP_MODIFICATION_1@4@)

[2\. Removal of Extensibility Annotations](#ABAP_MODIFICATION_2@4@)

[Empty table keys](ABENITAB_EMPTY_KEY.html) that are available in ABAP language since 7.40, SP02, can now also be defined for [DDIC table types](ABENDDIC_TABLE_TYPE_GLOSRY.html) for [standard tables](ABENSTANDARD_TABLE_GLOSRY.html).

The following [extensibility annotations](ABENDDICDDL_DEFINE_EXT_ANNOS.html) are no longer available for the [dictionary DDL](ABENDICTIONARY_DDL_GLOSRY.html) of [dictionary objects](ABENDICTIONARY_OBJECT_GLOSRY.html) and they are no longer required for [C0 release](ABENC0_CONTRACT_GLOSRY.html) of a DDIC object:

-   [`@AbapCatalog.enhancement.quotaShareCustomer`](ABENDDICDDL_DEFINE_EXT_ANNOS.html)
-   [`@AbapCatalog.enhancement.quotaSharePartner`](ABENDDICDDL_DEFINE_EXT_ANNOS.html)
