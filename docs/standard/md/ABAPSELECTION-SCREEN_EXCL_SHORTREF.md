`SELECTION-SCREEN EXCLUDE \{\ \{PARAMETERS parameter\}\                           |\ \{SELECT-OPTIONS selcrit\}\                           |\ \{RADIOBUTTON GROUPS group\}\                           |\ \{BLOCKS block\}\                           |\ \{IDS id\}\ \}.`

Deletes the specified element from a version of the [standard selection screen](ABENSTANDARD_SELSCREEN_GLOSRY.html) in a [logical database](ABENLOGICAL_DATA_BASE_GLOSRY.html).

-   `PARAMETERS parameter`\\
    Removes a parameter `parameter`.
-   `SELECT-OPTIONS selcrit`\\
    Removes a [selection criterion](ABENSELECTION_CRITERION_GLOSRY.html)\\ `selcrit`.
-   `RADIOBUTTON GROUPS group`\\
    Removes a radio button group `group`.
-   `BLOCKS block`\\
    Removes a block `block`.
-   `IDS id`\\
    Removes all elements of the ID `id`.
