`SET SCREEN dynnr.`

Defines the [dynpro](ABENDYNPRO_GLOSRY.html) with the [dynpro number](ABENDYNPRO_NUMBER_GLOSRY.html)\\ `dynnr` as the [next dynpro](ABENNEXT_DYNPRO_GLOSRY.html) to follow the current dynpro.
