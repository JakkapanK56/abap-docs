Component of an [extended passport (EPP)](ABENEPP_GLOSRY.html). Contains the [system ID](ABENSYSTEM_ID_GLOSRY.html) and the name of the [application server instance](ABENAS_INSTANCE_GLOSRY.html) of the [ABAP session](ABENABAP_SESSION_GLOSRY.html) on which the current connection is based.
