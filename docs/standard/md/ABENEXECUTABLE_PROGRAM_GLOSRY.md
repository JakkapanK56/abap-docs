[ABAP program](ABENABAP_PROGRAM_GLOSRY.html) that can be executed using the [`SUBMIT`](ABAPSUBMIT.html) statement. An executable program can be started via *System -> Services -> Reporting*, or from other services such as [report transactions](ABENREPORT_TRANSACTION_GLOSRY.html). A `SUBMIT` is always executed internally. An executable program can have its own [dynpros](ABENDYNPRO_GLOSRY.html), be linked to a [logical database](ABENLOGICAL_DATA_BASE_GLOSRY.html), and be executed in [background processing](ABENBACKROUND_PROCESSING_GLOSRY.html).
