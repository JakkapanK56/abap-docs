`...  with BOPF draft;  ...`

The addition `with BOPF draft` enables draft handling for a [BOPF-based RAP BO](ABENRAP_BOPF_BO_GLOSRY.html). It is added in the [behavior definition header](ABENCDS_BDEF_HEADER_GLOSRY.html) and it draft-enables the entire business object. The draft handling is delegated to the [Business Object Processing Framework](ABENBO_PROCESSING_FW_GLOSRY.html) (BOPF) by the [BOPF-managed RAP draft provider](ABENBOPF_MAN_RAP_DRA_PROV_GLOSRY.html).

The following [RAP behavior definition](ABENCDS_BEHAVIOR_DEFINITION_GLOSRY.html)\\ `DEMO_RAP_BOPF_DRAFT` shows a BOPF-based RAP BO with draft handling.

//GENERATED BY MIGRATION TOOL\\
