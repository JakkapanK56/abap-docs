[SAP HANA XS](ABENXS_GLOSRY.html) (SAP HANA extended application services) covers tools used to develop, manage, and execute native objects on the SAP HANA platform. This section describes the different ways of accessing SAP HANA XS objects from ABAP programs. Two SAP HANA XS versions are distinguished here:

[SAP HANA XSA](ABENXSA_GLOSRY.html) replaces [SAP HANA XSC](ABENXSC_GLOSRY.html). The [access methods](ABENSAP_HANA_ACCESS_XSC.html) designed specifically for XSC objects are not suitable for XSA objects. After all XSC objects have been migrated to XSA, the access methods designed for XSC became mostly obsolete. Especially, [SAP HANA XSC](ABENXSC_GLOSRY.html) is not supported by a [SAP HANA Cloud database](ABENSAP_HANA_CLOUD_DB_GLOSRY.html). More information can be found in [SAP Note 2465027](https://launchpad.support.sap.com/#/notes/2465027) and in [SAP Note 3116165](https://launchpad.support.sap.com/#/notes/3116165).

-   The current [SAP HANA XSA](ABENSAP_HANA_ACCESS_XSA.html)
-   The preceding [SAP HANA XSC](ABENSAP_HANA_ACCESS_XSC.html) (obsolete)
