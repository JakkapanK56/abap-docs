Value of any [data type](ABENDATA_TYPE_GLOSRY.html) that is constructed for each read line in a [group key expression](ABENGROUP_KEY_EXPRESSION_GLOSRY.html) when grouping [internal tables](ABENINTERNAL_TABLE_GLOSRY.html) using [`LOOP AT itab ... GROUP BY`](ABAPLOOP_AT_ITAB_GROUP_BY.html) or [`FOR GROUPS ... OF`](ABENFOR_GROUPS_OF.html). All table lines for which the same value is produced are assigned to each group key. In [group level processing](ABENGROUP_LEVEL_PROCESSING_GLOSRY.html) for internal tables, the group key is the content of all columns to the left of a column specified after [`AT`](ABAPAT_ITAB.html).
