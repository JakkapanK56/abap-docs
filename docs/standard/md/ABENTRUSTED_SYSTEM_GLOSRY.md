[AS ABAP](ABENAS_ABAP_GLOSRY.html) that is classified as a trusted caller of [remote function calls](ABENREMOTE_FUNCTION_CALL_GLOSRY.html) in a [trusting system](ABENTRUSTING_SYSTEM_GLOSRY.html). The trusted system provides the [RFC client](ABENRFC_CLIENT_GLOSRY.html). The system is classified in [transaction](ABENTRANSACTION_GLOSRY.html)\\ `SMT1`. Trusted systems log on to the trusting system without passing a password. User-specific logon data is checked in the trusting system.
