When [Control Framework](ABENCONTROL_FRAMEWORK_GLOSRY.html) events are handled, messages of types *W* and *E* are converted to type *A*. All other messages are processed as specified in [dialog processing](ABENABAP_MESSAGE_DIALOG.html).
