Tool of the [ABAP Workbench](ABENABAP_WORKBENCH_GLOSRY.html) for the central maintenance of [repository objects](ABENREPOSITORY_OBJECT_GLOSRY.html). The Object Navigator displays repository objects as nodes of different tree representations (browsers), such as [Repository Browser](ABENREPOSITORY_BROWSER_GLOSRY.html), [Repository Information System](ABENRIS_GLOSRY.html), or [Enhancement Information System](ABENENH_INFO_SYSTEM_GLOSRY.html), in the navigation area. From here, they can be accessed via [forward navigation](ABENFORWARD_NAVIGATION_GLOSRY.html) in the tool area. Called using [transaction](ABENTRANSACTION_CODE_GLOSRY.html)\\ `SE80`.
