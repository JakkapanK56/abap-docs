`CLOSE CURSOR @dbcur.`

[ABAP SQL](ABENABAP_SQL_GLOSRY.html) statement. Closes a [database cursor](ABENDATABASE_CURSOR_GLOSRY.html) linked with `dbcur` and that was opened using `OPEN CURSOR`.
