This example demonstrates how a database function is accessed using static Native SQL.

After [`EXEC SQL`](ABAPEXEC.html), this example creates a scalar [database function](ABENDATABASE_FUNCTION_GLOSRY.html) with two input parameters and uses this function in the `WHERE` condition of a `SELECT` statement.
