This example demonstrates the rounding function [`round`](ABENDEC_FLOATING_POINT_FUNCTIONS.html).

For fraction numbers between 1 and -1, the class depicts the effect of all rounding types from the class `CL_ABAP_MATH` in the rounding function [`round`](ABENDEC_FLOATING_POINT_FUNCTIONS.html). The class reads the possible rounding methods using RTTI and sorts them by name.
