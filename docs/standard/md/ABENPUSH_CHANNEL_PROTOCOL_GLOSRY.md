PCP for short. Describes an SAP communications format used to exchange messages between programs and internal or external interfaces. Currently, PCP is used by [ABAP Channels](ABENABAP_CHANNEL_GLOSRY.html) and [ABAP daemons](ABENABAP_DAEMON_GLOSRY.html).
