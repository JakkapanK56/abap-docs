[1. Returning a Value with `RETURN`](#ABAP_MODIFICATION_1@4@)

[2. `DEFAULT` as New Addition to the `CORRESPONDING` Operator](#ABAP_MODIFICATION_2@4@)

In [functional methods](ABENFUNCTIONAL_METHOD_GLOSRY.html), the statement [`RETURN`](ABAPRETURN.html) can be used to assign the result of an expression [`expr`](ABAPRETURN.html) to the [return value](ABENRETURN_VALUE_GLOSRY.html) when terminating the method.

The addition [`DEFAULT`](ABENCORRESPONDING_CONSTR_MAPPING.html) allows the assignment of values for a target component based on an expression.
