[File](ABENFILE_GLOSRY.html) on the [host computer](ABENHOST_COMPUTER_GLOSRY.html) of an [AS instance](ABENAS_INSTANCE_GLOSRY.html) opened using the addition [`BINARY`](ABAPOPEN_DATASET_MODE.html) of the statement [`OPEN DATASET`](ABAPOPEN_DATASET_MODE.html). The content of a binary file is read or written without being handled.
