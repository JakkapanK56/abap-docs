ADBC now supports [ABAP SQL connection objects](ABENABAP_SQL_CONN_OBJ_GLOSRY.html) using the method `ADAPTER_2_ABAP_SQL` of the `CL_SQL_ADBC_CONNECTION_BUILDER` class. The ABAP SQL connection object must enable the [ADBC adapter](ABENIF_SQL_CONNECTION.html).
