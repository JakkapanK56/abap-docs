[Program directive](ABENPROGRAM_DIRECTIVE_GLOSRY.html) in the form of a special [line end comment](ABENENDLINE_COMMENT_GLOSRY.html). A pseudo comment begins with the expression `#EC` or `#AU` and suppresses either a message of the [extended program check](ABENEXTENDED_PROGRAM_CHECK_GLOSRY.html) or of the [Code Inspector](ABENCODE_INSPECTOR_GLOSRY.html), or determines the test properties for a [test class](ABENTEST_CLASS_GLOSRY.html). Pseudo comments are only still required for suppressing Code Inspector messages and are otherwise obsolete.
