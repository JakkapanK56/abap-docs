Instance for a [user session](ABENUSER_SESSION_GLOSRY.html) of an [AS instance](ABENAS_INSTANCE_GLOSRY.html) with the [ABAP Memory](ABENABAP_MEMORY_GLOSRY.html) as a separate memory area. Each [GUI window](ABENGUI_WINDOW_GLOSRY.html) of a user session is linked with its own ABAP session. A maximum of 16 ABAP sessions can be managed by a single user session. An [ABAP program](ABENABAP_PROGRAM_GLOSRY.html) is executed in an [internal session](ABENINTERNAL_SESSION_GLOSRY.html) of an ABAP session.
