In ABAP release 7.70, a new [ABAP version](ABENABAP_VERSION_GLOSRY.html)\\ [ABAP for Cloud Development](ABENABAP_FOR_CLOUD_DEV_GLOSRY.html) was introduced. The internal version ID in the column `UCCHECK` of the system table `TRDIR` is *5*.
