`NODES node [TYPE type].`

Declares an [interface work area](ABENINTERFACE_WORK_AREA_GLOSRY.html)\\ `node` for data transfer in [logical databases](ABENLOGICAL_DATA_BASE_GLOSRY.html).

-   `TYPE type`\\
    Specifies the data type `type` of the work area from a predefined list.
