Property of a [data object](ABENDATA_OBJECT_GLOSRY.html). ABAP interprets the content of a data object in accordance with its data type. Data types exist either as [bound](ABENBOUND_DATA_TYPE_GLOSRY.html) data types as a property of data objects or as [standalone](ABENSTAND-ALONE_DATA_TYPE_GLOSRY.html) data types. Standalone data types can be defined globally in the [ABAP Dictionary](ABENABAP_DICTIONARY_GLOSRY.html) or in [ABAP CDS](ABENABAP_CDS_GLOSRY.html). Data types defined with [`TYPES`](ABAPTYPES.html) in an [ABAP program](ABENABAP_PROGRAM_GLOSRY.html) can be either local or global. The latter is true for non-private data types of [classes](ABENCLASS_GLOSRY.html) and [interfaces](ABENOO_INTF_GLOSRY.html). The [generic data type](ABENGENERIC_DATA_TYPE_GLOSRY.html) of all data types is [`data`](ABENBUILT_IN_TYPES_GENERIC.html).
