This example demonstrates the statements [`CONVERT TIME STAMP`](ABAPCONVERT_TIME-STAMP.html) and [`CONVERT INTO TIME STAMP`](ABAPCONVERT_DATE_TIME-STAMP.html).

The program is given a date, a time, and a time zone and converts this information into an UTC time stamp in a [time stamp in a packed number](ABENTIME_STAMPS_PACKED.html). The program can also include daylight saving time in its calculations. The resulting UTC time stamp is then converted to the local date and local time of another time zone. The following table shows some possible combinations of input and output, where `spc` stands for a space and `-` for no input or output.

The third and fifth lines demonstrate that invalid daylight saving time specifications produce the value 12 in `sy-subrc`.
