`MODULE mod \{OUTPUT|[INPUT]\}.    ...  ENDMODULE.`

Defines a [dialog module](ABENDIALOG_MODULE_GLOSRY.html)\\ `mod`.

-   `OUTPUT`\\
    Defines a dialog module for [PBO](ABENPBO_GLOSRY.html).
-   `INPUT`\\
    Defines a dialog module for [PAI](ABENPAI_GLOSRY.html), [POH](ABENPOH_GLOSRY.html) and [POV](ABENPOV_GLOSRY.html).
