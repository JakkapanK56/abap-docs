[1\. External Mailbox for Documentation](#ABAP_MODIFICATION_1@4@)

[2\. ADT Documentation Layout](#ABAP_MODIFICATION_2@4@)

From Release 7.58 on, the ABAP Keyword Documentation display offers a function to send a feedback mail to *f1\_help@sap.com* in non-SAP development systems and in the [SAP Help Portal](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/index.htm). Up to now, the possibility for a feedback mail existed only in SAP's own development systems. The function can be switched on or off in the configuration of the ABAP Keyword Documentation, see SAP Note [3051036](https://launchpad.support.sap.com/#/notes/3051036).

From Release 7.58 on, the ABAP Keyword Documentation presents a new layout in [ABAP development tools for Eclipse (ADT)](ABENADT_GLOSRY.html) where a toggle between *Standard ABAP* and *ABAP Cloud* is also available. The new layout includes:

-   New header bar with links to the mail feedback and the web version
-   Highlighted key blocks for hints, code blocks, and others
-   Tables with zebra stripes
