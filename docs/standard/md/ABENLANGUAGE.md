The language environment of an ABAP program is part of the ABAP runtime framework and includes the following:

-   [Language settings](ABENLANGUAGE_SETTINGS.html)
-   The language settings defined by the [logon language](ABENLOGON_LANGUAGE_GLOSRY.html) and a [text environment language](ABENTEXT_ENV_LANGU_GLOSRY.html) control all language-dependent behavior of an ABAP program.
-   [Format settings](ABENCOUNTRY.html)
-   Country-specific formatting settings determine, for example, the format of numbers, dates, times, and time stamps.
