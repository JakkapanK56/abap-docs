Static check performed on [ABAP source code](ABENABAP_SOURCE_CODE_GLOSRY.html) to check whether the syntax is formally correct. The syntax check is called in the [ABAP Editor](ABENABAP_EDITOR_GLOSRY.html) via *Program -> Check -> Syntax* or via the statement [`SYNTAX-CHECK`](ABAPSYNTAX-CHECK_FOR_ITAB.html). An automatic syntax check that checks the source code while it is entered can be configured in the [ABAP development tools for Eclipse (ADT)](ABENADT_GLOSRY.html).
