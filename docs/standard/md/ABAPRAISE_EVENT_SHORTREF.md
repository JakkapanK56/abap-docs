[Reference](ABAPRAISE_EVENT.html)

`RAISE EVENT evt [EXPORTING p1 = a1 p2 = a2 ...].`

Raises an event in a method.

-   `EXPORTING p1 = a1 p2 = a2 ...`\\
    Passes actual parameters `a1`, `a2`, ... to the input parameter `p1`, `p2` of the [event handler](ABENEVENT_HANDLER_GLOSRY.html).
