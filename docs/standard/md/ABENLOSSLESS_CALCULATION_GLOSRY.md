[Arithmetic](ABENARITHMETIC_EXPRESSION_GLOSRY.html) calculation where no rounding occurs. The [lossless operator](ABENLOSSLESS_OPERATOR_GLOSRY.html)\\ [`EXACT`](ABENCONSTRUCTOR_EXPRESSION_EXACT.html) or the addition `EXACT` of the obsolete statement [`COMPUTE`](ABAPCOMPUTE.html) can be used to check whether a calculation is lossless.
