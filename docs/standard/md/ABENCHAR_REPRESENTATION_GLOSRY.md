Binary encoding of characters. [Character sets](ABENCHARACTER_SET_GLOSRY.html) are assigned to a character representation using [code pages](ABENCODEPAGE_GLOSRY.html). The [ISO-8859](ABENISO-8859_GLOSRY.html) code pages are [single-byte](ABENSINGLE_BYTE_CODE_GLOSRY.html) character representations, in which every character is encoded in up to one byte. SJIS and BIG5 are [double-byte](ABENDOUBLE_BYTE_CODE_GLOSRY.html) character representations for Japanese and traditional Chinese fonts. The [Unicode character representations](ABENUNICODE_CHAR_REPRESENT_GLOSRY.html) comprise all characters world-wide and can be encoded using one, two, or four bytes, depending on the [UTF](ABENUTF_GLOSRY.html) representation.
