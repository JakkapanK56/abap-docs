DOM representation of XML data.

XML data is transformed to its DOM representation using the [identity transformation](ABENID_TRAFO_GLOSRY.html)\\ `ID`. An [iterator](ABENABAP_IXML_LIB_DOM_ITERAT_READ.html) is used to read all DOM nodes and pass their most important properties to an internal table and then output them. A second iteration reads the attributes of elements, whose internal table is also output.

The GID is a unique internal indexing of a node.
