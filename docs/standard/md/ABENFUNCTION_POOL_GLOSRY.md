The only kind of [ABAP program](ABENABAP_PROGRAM_GLOSRY.html) that can contain [function modules](ABENFUNCTION_MODULE_GLOSRY.html); usually loaded by calling one of its function modules. A function pool can currently contain a maximum of 99 function modules. It supports its own [dynpros](ABENDYNPRO_GLOSRY.html).
