``FIELD-SYMBOLS <fs> \{\ [`typing`](ABENTYPING_SHORTREF.html)\ |\ \{ STRUCTURE struc DEFAULT dobj \}\ \}.``

Declares a field symbol `<fs>`. The angle brackets are part of the name.

-   `typing`\\
    Types the field symbol. See [`typing`](ABENTYPING_SHORTREF.html).
-   [`STRUCTURE struc DEFAULT dobj`](ABAPFIELD-SYMBOLS_OBSOLETE_TYPING.html)\\
    **Obsolete:** Casts a structure `struc` and assigns a data object `dobj`.
