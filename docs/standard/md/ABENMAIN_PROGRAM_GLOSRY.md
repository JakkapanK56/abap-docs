The first program of a [program group](ABENPROGRAM_GROUP_GLOSRY.html). The first program ([executable program](ABENEXECUTABLE_PROGRAM_GLOSRY.html), [module pool](ABENMODUL_POOL_GLOSRY.html), or [function pool](ABENFUNCTION_POOL_GLOSRY.html)) that is loaded by a program call into an [internal session](ABENINTERNAL_SESSION_GLOSRY.html) is the main program of the [main program group](ABENMAIN_PROGRAM_GROUP_GLOSRY.html). The program (function pool or [class pool](ABENCLASS_POOL_GLOSRY.html)), for which an [additional program group](ABENADDITIONAL_PROG_GROUP_GLOSRY.html) is created when it is loaded is the main program of the additional program group.
