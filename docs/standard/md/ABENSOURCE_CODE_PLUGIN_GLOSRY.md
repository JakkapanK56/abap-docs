Source code plug-ins are used to make modification-free [enhancements](ABENENHANCEMENT_GLOSRY.html) to [ABAP source code](ABENABAP_SOURCE_CODE_GLOSRY.html). Source code plug-ins are included in code by means of the statements [`ENHANCEMENT`](ABAPENHANCEMENT.html) and [`ENDENHANCEMENT`](ABAPENDENHANCEMENT.html) and are created using the [Enhancement Builder](ABENENHANCEMENT_BUILDER_GLOSRY.html) tool.
