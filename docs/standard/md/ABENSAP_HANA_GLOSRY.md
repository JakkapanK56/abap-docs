SAP HANA is a platform for SAP In-Memory Computing based on the [SAP HANA database](ABENHANA_DATABASE_GLOSRY.html). If an SAP HANA database is used as the [standard database](ABENSTANDARD_DB_GLOSRY.html) of an [AS ABAP](ABENAS_ABAP_GLOSRY.html), ABAP programs can use [ABAP SQL](ABENABAP_SQL_GLOSRY.html) to access SAP HANA directly. In addition, [secondary connections](ABENSECONDARY_DB_CONNECTION_GLOSRY.html) can also be used to access SAP HANA from ABAP.
