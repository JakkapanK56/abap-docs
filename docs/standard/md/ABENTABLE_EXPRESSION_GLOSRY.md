A table expression reads a line from an [internal table](ABENINTERNAL_TABLE_GLOSRY.html) and returns it as a result. [Mesh path expressions](ABENMESH_PATH_EXPRESSION_GLOSRY.html) are a special form of table expressions that can be used in the same way as table expressions.
