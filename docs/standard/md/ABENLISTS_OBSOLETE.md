[Classic lists](ABENABAP_DYNPRO_LIST.html) should no longer be used in production application programs. SAP List Viewer (ALV) should be used instead of classic lists. If classic lists are still used for any reason, then the following constructs must be avoided:

-   [Obsolete formatting](ABENFORMAT_OBSOLETE.html)
-   [Obsolete calculations](ABENCALCULATE_OBSOLETE.html)
-   [Obsolete list event](ABENLIST_EVENT_OBSOLETE.html)
-   [Obsolete spooling](ABENPRINTING_OBSOLETE.html)
