Execution of a [test task](ABENTEST_TASK_GLOSRY.html) in [ABAP Unit](ABENABAP_UNIT_GLOSRY.html). Test runs for test tasks of an individual program can be started directly in the corresponding editor via *Program* or *Function Module -> Test -> Unit Test* or *Class -> Unit Test*. More extensive test runs can be executed using [Code Inspector](ABENCODE_INSPECTOR_GLOSRY.html). The results of a test run are displayed in the ABAP Unit interface.
