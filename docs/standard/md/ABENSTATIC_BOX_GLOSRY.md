[Boxed component](ABENBOXED_COMPONENT_GLOSRY.html) for supporting [initial value sharing](ABENINITIAL_VALUE_SHARING_GLOSRY.html) when the initial value exists only once on the [AS instance](ABENAS_INSTANCE_GLOSRY.html). A static box is declared using the `BOXED` addition when defining [substructures](ABENSUBSTRUCTURE_GLOSRY.html) using [`TYPES`](ABAPTYPES_BOXED.html) or when declaring structured [attributes](ABENATTRIBUTE_GLOSRY.html) of [classes](ABENCLASS_GLOSRY.html) or [interfaces](ABENOO_INTF_GLOSRY.html) using [`DATA`](ABAPDATA_BOXED.html).
