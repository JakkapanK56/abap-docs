Short for SAP HANA extended application services Classic Model. Previous version of [XS](ABENXS_GLOSRY.html) and predecessor to [XSA](ABENXSA_GLOSRY.html). XSC objects are stored in the [SAP HANA repository](ABENSAP_HANA_REPOSITORY_GLOSRY.html) and are usually developed and managed in [SAP HANA Studio](ABENHANA_STUDIO_GLOSRY.html). SAP HANA XSC is not supported by a [SAP HANA Cloud database](ABENSAP_HANA_CLOUD_DB_GLOSRY.html).
