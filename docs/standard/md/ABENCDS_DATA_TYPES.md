ABAP CDS provides a set of [CDS built-in data types](ABENCDS_BUILT_IN_TYPE_GLOSRY.html). Custom data types can be defined as [CDS user-defined types](ABENCDS_USER_DEFINED_TYPE_GLOSRY.html). [CDS generic types](ABENCDS_GENERIC_TYPE_GLOSRY.html) cover a range of CDS built-in data types and can be used like a placeholder that allows for different built-in data types.

For further details on data types in ABAP CDS, see [*Data Types in ABAP CDS*](https://help.sap.com/docs/ABAP_Cloud/aaae421481034feab3e71dd9e0f643bf/f04708ca02d345e18fecbbd9a1eb9eb7?version=sap_cross_product_abap) in the ABAP Data Models guide on SAP Help Portal.
