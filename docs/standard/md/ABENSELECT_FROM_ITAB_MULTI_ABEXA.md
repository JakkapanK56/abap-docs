This example demonstrates a multiple join of internal tables.

The `SELECT` statement joins four internal tables. The result is the same as for the respective statement from [`SELECT`, Multiple Joins](ABENJOIN_JOINS_ABEXA.html), where database tables of the same structure and content are joined.

The `SELECT` statements can be processed by the [ABAP SQL in-memory engine](ABENABAP_SQL_INMEMENG.html), since only internal tables and no database tables are involved.
