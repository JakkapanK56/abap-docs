Short for Extended Binary Coded Decimal Interchange Code. An 8-bit [character set](ABENCHARACTER_SET_GLOSRY.html) based on [BCD](ABENBCD_GLOSRY.html). In general, EBCDIC consists of the same characters as [ASCII](ABENASCII_GLOSRY.html).
