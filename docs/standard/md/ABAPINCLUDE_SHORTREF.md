[Reference](ABAPINCLUDE_PROG.html)

`INCLUDE incl [IF FOUND].`

Includes the [include program](ABENINCLUDE_PROGRAM_GLOSRY.html)\\ `incl` in the current position in the source code.

-   `IF FOUND`\\
    Suppresses the syntax check error message if the include program is not found.
