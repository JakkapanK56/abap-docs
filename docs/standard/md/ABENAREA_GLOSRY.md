[Repository object](ABENREPOSITORY_OBJECT_GLOSRY.html) used for the managing and storage of [shared objects](ABENSHARED_OBJECTS_GLOSRY.html). An area is the template for [area instance versions](ABENAREA_INSTANCE_VERSION_GLOSRY.html) in the [shared memory](ABENSHARED_MEMORY_GLOSRY.html). An identically named [area class](ABENAREA_CLASS_GLOSRY.html) and any [area root class](ABENROOT_DATA_CLASS_GLOSRY.html) are assigned to each area. Areas are created and managed using [transaction](ABENTRANSACTION_GLOSRY.html)\\ `SHMA`.
