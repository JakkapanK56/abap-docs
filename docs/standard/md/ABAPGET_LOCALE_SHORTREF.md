[Reference](ABAPGET_LOCALE.html)

`GET LOCALE LANGUAGE lang COUNTRY cntry MODIFIER mod.`

Determines the current [text environment](ABENTEXT_ENVIRONMENT_GLOSRY.html).

-   `LANGUAGE lang`\\
    Returns the language key in `lang`.
-   [`COUNTRY cntry`](ABAPGET_LOCALE_OBSOLETE.html)\\
    **Obsolete:** Content of `cntry` is undefined.
-   [`MODIFIER mod`](ABAPGET_LOCALE_OBSOLETE.html)\\
    **Obsolete:** Initializes `mod`.
