Expression-oriented implementation of a Minesweeper game.

This example uses the implementation of a Minesweeper game to demonstrate the expression-oriented ABAP programming as introduced in ABAP release 7.40. As a comparison, the program `DEMO_MINESWEEPER_700` shows a functionally identical implementation without expressions, which is restricted to mainly using language elements of ABAP release 7.00.

To play the game, use the left and right mouse buttons to enter or select fields.
