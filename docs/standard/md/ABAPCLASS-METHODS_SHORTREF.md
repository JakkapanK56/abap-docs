`CLASS-METHODS meth options                    |\{FOR TABLE FUNCTION cds_entity\}.`

Declares a [static method](ABENSTATIC_METHOD_GLOSRY.html)\\ `meth` of a class or interface.

-   `options`\\
    As in [`METHODS`](ABAPMETHODS_SHORTREF.html).
-   [`FOR TABLE FUNCTION cds_tabfunc`](ABAPCLASS-METHODS_FOR_TABFUNC.html)\\
    Declares an [AMDP function implementation](ABENAMDP_FUNCTION_METHOD_GLOSRY.html) for a [CDS table function](ABENCDS_TABLE_FUNCTION_GLOSRY.html).
