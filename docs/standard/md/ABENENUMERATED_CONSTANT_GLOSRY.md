A [constant](ABENCONSTANT_GLOSRY.html) declared in the definition of an [enumerated type](ABENENUM_TYPE_GLOSRY.html) that contains an [enumerated value](ABENENUMERATED_VALUE_GLOSRY.html) of the enumerated type.

In ABAP, an enumerated constant is a special [enumerated object](ABENENUMERATED_OBJECT_GLOSRY.html). See also [enumerated structure](ABENENUMERATED_STRUCTURE_GLOSRY.html).

In ABAP CDS, an enumerated constant is a special [enumerated element](ABENCDS_ENUM_ELEMENT_GLOSRY.html).
