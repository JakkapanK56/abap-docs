This example demonstrates [asJSON](ABENABAP_ASJSON_ABAP_TYPES_TABLE.html) for additional [XML Schema Data Types](ABENABAP_ASJSON_SCHEMA.html).

A call of the [identity transformation](ABENID_TRAFO_GLOSRY.html)\\ `ID`, for which a [JSON](ABENJSON_WRITER_GLOSRY.html) writer is specified as an XML target, creates the [asJSON](ABENASJSON_GLOSRY.html) representation of a structure. The structure components are typed with regular ABAP types and typed with special [domains](ABENABAP_XSLT_ASXML_SCHEMA.html) for XML schema data types. The time stamp type `utclong` is formatted by default using `yyyy-mm-ddThh:mm:ss.fffffffZ`. [UTC time stamps](ABENUTC_TIMESTAMP_GLOSRY.html) in packed numbers, on the other hand, require special domains.

For comparison, the [JSON-XML](ABENJSON_XML_GLOSRY.html) representation of the JSON data and the [asXML](ABENASXML_GLOSRY.html) representation of the ABAP data is also shown.
