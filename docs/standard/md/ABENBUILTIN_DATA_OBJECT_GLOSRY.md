[Data object](ABENDATA_OBJECT_GLOSRY.html) in an [ABAP program](ABENABAP_PROGRAM_GLOSRY.html) by the [ABAP runtime framework](ABENABAP_RUNTIME_FRMWK_GLOSRY.html) that is either permanently active or only in specific [contexts](ABENOBJ_CONTEXT_GLOSRY.html). The built-in data objects are [system fields](ABENSYSTEM_FIELD_GLOSRY.html), the [constant](ABENCONSTANT_GLOSRY.html)\\ [`space`](ABENSPACE.html), and the [self-reference](ABENSELF_REFERENCE_GLOSRY.html)\\ [`me`](ABENME.html). The [structure](ABENSTRUCTURE_GLOSRY.html)\\ [`screen`](ABENSCREEN_STRUCTURE_OBSOLETE.html) is obsolete.
