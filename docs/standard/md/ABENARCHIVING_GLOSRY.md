Storing of [spool lists](ABENSPOOL_LIST_GLOSRY.html) using the [ArchiveLink](ABENARCHIVELINK_GLOSRY.html) tool. Spool lists stored in the [SAP spool system](ABENSAP_SPOOL_SYSTEM_GLOSRY.html) can be archived. During archiving, spool lists passed from the spool system are saved in the same format that was used there.
