First [program group](ABENPROGRAM_GROUP_GLOSRY.html) of an [internal session](ABENINTERNAL_SESSION_GLOSRY.html). It is created when the internal session is created through a program call. The first program in an internal session is the [main program](ABENMAIN_PROGRAM_GLOSRY.html) of the main program group. The runtime of the main program in the main program group determines its lifespan and that of the entire internal session.
