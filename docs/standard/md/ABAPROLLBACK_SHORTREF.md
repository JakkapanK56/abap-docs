`ROLLBACK WORK.`

Ends an [SAP LUW](ABENSAP_LUW_GLOSRY.html), executing the subroutines registered using `PERFORM ON ROLLBACK` and discarding the subroutines and [update function modules](ABENUPDATE_FUNCTION_MODULE_GLOSRY.html) registered using `PERFORM ON COMMIT` or `CALL FUNCTION IN UPDATE TASK`.
