[Reference](ABAPSELECTION-SCREEN_BLOCK.html)

`SELECTION-SCREEN BEGIN OF BLOCK block                                  [WITH FRAME [TITLE title]]\                                  [NO INTERVALS].  ...  SELECTION-SCREEN END OF BLOCK block.`

Defines a block named `block` on the current [selection screen](ABENSELECTION_SCREEN_GLOSRY.html).

-   `WITH FRAME`\\
    Creates a frame around the block.
-   `TITLE title`\\
    Creates a title `title` for the block.
-   `NO INTERVALS`\\
    Specifies the addition `NO INTERVALS` for all the selection criteria in the block.
