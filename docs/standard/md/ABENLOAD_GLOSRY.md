In ABAP, a load is a binary representation of a [repository object](ABENREPOSITORY_OBJECT_GLOSRY.html), which is optimized for fast access, in the memory or on the database. Examples of loads are program load that is made up of the [byte code](ABENBYTECODE_GLOSRY.html) of the existing program or the package load.
