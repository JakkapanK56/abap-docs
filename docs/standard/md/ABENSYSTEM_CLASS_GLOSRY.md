[Class](ABENCLASS_GLOSRY.html) of the [class library](ABENCLASS_LIBRARY_GLOSRY.html) in which language-related functionality is implemented. Its name usually starts with `CL_ABAP_`, `CL_SYSTEM_`, or `CX_SY_` (in the case of [exception classes](ABENEXCEPTION_CLASS_GLOSRY.html)). An overview of all ABAP system classes can be displayed using the corresponding [input help](ABENINPUT_HELP_GLOSRY.html) of the [Class Builder](ABENCLASS_BUILDER_GLOSRY.html).
