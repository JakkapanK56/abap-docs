A [tuning object](ABENTUNING_OBJECT_GLOSRY.html) is a [repository object](ABENREPOSITORY_OBJECT_GLOSRY.html) that defines technical settings or tunings for another repository object. A tuning object can be defined and transported separately from the repository object for which it is defined. ABAP Dictionary provides the following [HANA tuning objects](ABENHANA_TUNING_OBJECT_GLOSRY.html) to define tunings for dictionary objects:

-   [DDIC - Dependency Rules](ABENDDIC_DEPENDENCY_RULES.html)
-   [DDIC - Dynamic Caches](ABENDDIC_DYNAMIC_CACHES.html)
