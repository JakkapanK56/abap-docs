Obsolete [procedure](ABENPROCEDURE_GLOSRY.html) that can be defined in any [ABAP program](ABENABAP_PROGRAM_GLOSRY.html) (except [type pools](ABENTYPE_POOL_GLOSRY.html), [class pools](ABENCLASS_POOL_GLOSRY.html), and [interface pools](ABENINTERFACE_POOL_GLOSRY.html)), however, can only be implemented outside [classes](ABENCLASS_GLOSRY.html) between the statements [`FORM`](ABAPFORM.html) and [`ENDFORM`](ABAPFORM.html). The statement [`PERFORM`](ABAPPERFORM.html) can be used to call the subroutines of an [ABAP program](ABENABAP_PROGRAM_GLOSRY.html) and of any other programs.
