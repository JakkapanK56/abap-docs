In object-oriented programming, polymorphism means the ability to address differently implemented [methods](ABENMETHOD_GLOSRY.html) that belong to different [objects](ABENOBJECT_GLOSRY.html) of different [classes](ABENCLASS_GLOSRY.html) with the same name and via the same interface.
