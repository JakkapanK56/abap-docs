@EnterpriseSearch annotations add semantics that is needed to use a view entity as a (CDS-based) search model in the Enterprise Search framework.

Assigns an Enterprise Search Configuration Set (which has previously be defined in the Enterprise Search configuration) to the element of a view. The configuration settings bundled in this Configuration Set are then applied when searching.
