[ABAP releases](ABENABAP_RELEASE_GLOSRY.html) 7.5x are based on [7.40, SP08](ABENNEWS-740_SP08.html).

The following table shows the [ABAP releases](ABENABAP_RELEASE_GLOSRY.html) and the associated [ABAP kernel](ABENKERNEL_GLOSRY.html) releases for the [ABAP Platform](ABENABAP_PLATFORM_GLOSRY.html) shipped with each [SAP S/4HANA](ABENSAP_S4HANA_GLOSRY.html), [SAP S/4HANA Cloud Private Edition](ABENSAP_S4HANA_CPRIV_GLOSRY.html) release. The first column links to the respective release news.

The term [ABAP Platform](ABENABAP_PLATFORM_GLOSRY.html) was introduced with ABAP release 7.53 as a name for the [ABAP technology](ABENABAP_TECHNO_GLOSRY.html) used for SAP S/4HANA. The ABAP Platform is not available standalone, but only shipped with SAP S/4HANA. The release number of the ABAP Platform corresponds to the SAP S/4HANA release.

The term *ABAP Platform* replaces the usage of the term *Application Server ABAP* in SAP NetWeaver which itself replaced the term *SAP Basis*. Nowadays, an *ABAP Platform* plays the same role for SAP S/4HANA as formerly the SAP basis did for *SAP R/3*.
