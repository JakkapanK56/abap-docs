[Program group](ABENPROGRAM_GROUP_GLOSRY.html) of an [internal session](ABENINTERNAL_SESSION_GLOSRY.html). Created when a [function pool](ABENFUNCTION_POOL_GLOSRY.html) is loaded into the internal session using an external procedure call or a [class pool](ABENCLASS_POOL_GLOSRY.html) via its global [class](ABENCLASS_GLOSRY.html). The function pool or class pool is the [main program](ABENMAIN_PROGRAM_GLOSRY.html) of the additional program group. An additional program group and its data remain for the entire duration of the internal session.
