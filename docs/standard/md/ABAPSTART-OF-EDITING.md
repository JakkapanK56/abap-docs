`START-OF-EDITING.`

The statement `START-OF-EDITING` defines an [event block](ABENEVENT_BLOCK_GLOSRY.html) in an [executable program](ABENEXECUTABLE_PROGRAM_GLOSRY.html). The event of the program is raised from the [ABAP runtime framework](ABENABAP_RUNTIME_FRMWK_GLOSRY.html) in the [executable program flow](ABENREPORTING_PROCESS.html) directly after [`START-OF-SELECTION`](ABAPSTART-OF-SELECTION.html).

An implementation of the event block `START-OF-EDITING` is unnecessary.
