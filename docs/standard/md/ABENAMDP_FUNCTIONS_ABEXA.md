This example demonstrates AMDP functions and how they are used. The AMDP methods implementing the functions are not [client-safe](ABENCLIENT_SAFE_GLOSRY.html) and are not suited for [ABAP for Cloud Development](ABENABAP_FOR_CLOUD_DEV_GLOSRY.html).

This example accesses [AMDP functions](ABENAMDP_FUNCTION_GLOSRY.html) that are declared and implemented in the [AMDP class](ABENAMDP_CLASS_GLOSRY.html)\\ `CL_DEMO_AMDP_FUNCTIONS_INPCL`.

The CDS table function `DEMO_CDS_GET_SCARR_SPFLI_INPCL` used here has an input parameter for the client, annotated using the annotation [`@Environment.systemField`](ABENCDS_F1_PARAMETER_ANNOTATIONS.html) and the predefined value [`#CLIENT`](ABENCDS_F1_PARAMETER_ANNOTATIONS.html). This input parameter is filled explicitly with the ID of the [current client](ABENCURRENT_CLIENT_GLOSRY.html) by the ABAP SQL statement `SELECT` and used in the implementation of the associated AMDP method for selecting the data. The nearly identical class `CL_DEMO_AMDP_FNCTNS` uses the following CDS table function `DEMO_CDS_GET_SCARR_SPFLI` in which the current client is selected from the result set of the table function.

-   The method `GET_SCARR_SPFLI` is an [AMDP function implementation](ABENAMDP_FUNCTION_METHODS.html) for an [AMDP table function](ABENAMDP_TABLE_FUNCTION_GLOSRY.html) that can only be called in other AMDP methods. It has an explicitly declared interface.

-   The method `SELECT_GET_SCARR_SPFLI` is an [AMDP procedure implementation](ABENAMDP_PROCEDURE_METHODS.html) that uses `SELECT` to access the AMDP table function implemented in the method `GET_SCARR_SPFLI`.

-   The method `GET_SCARR_SPFLI_FOR_CDS` is an [AMDP function implementation](ABENAMDP_FUNCTION_METHODS.html) for the [CDS table function](ABENCDS_TABLE_FUNCTION_GLOSRY.html)\\ `DEMO_CDS_GET_SCARR_SPFLI_INPCL`, which is also shown as an example of the DDL statement `[DEFINE TABLE FUNCTION](ABENCDS_F1_DEFINE_TABLE_FUNCTION.html)`[](ABENCDS_F1_DEFINE_TABLE_FUNCTION.html):

-   The method `GET_MAX_FLTIME_SPFLI` is an [AMDP function implementation](ABENAMDP_FUNCTION_METHODS.html) for an [AMDP scalar function](ABENAMDP_SCALAR_FUNCTION_GLOSRY.html). It has an explicitly declared interface and can be called in ABAP.

-   The method `SELECT_SPFLI_BY_MAX_FLTIME` is an [AMDP procedure implementation](ABENAMDP_PROCEDURE_METHODS.html) where the AMDP scalar function that is implemented in the method `GET_MAX_FLTIME_SPFLI` is called in its `SELECT` statement in the `WHERE` condition.
