`METHODS meth [FINAL] REDEFINITION.`

[Redefines](ABENREDEFINITION_GLOSRY.html) a `meth` instance method in the declaration part of a class. The method is reimplemented in the implementation part.

-   `FINAL`\\
    Makes a redefined method final so that it cannot be redefined again in a subclass.
