Mapping of a [character set](ABENCHARACTER_SET_GLOSRY.html) to bit sequences. For example, Latin1 ([ISO-8859-1](ABENISO-8859_GLOSRY.html)) represents a code page that contains all the characters required for Western European languages, whereas a [Unicode character representation](ABENUNICODE_CHAR_REPRESENT_GLOSRY.html) can represent all characters required world-wide. The way in which characters are represented in an [internal session](ABENINTERNAL_SESSION_GLOSRY.html) is determined by the code page of the [text environment](ABENTEXT_ENVIRONMENT_GLOSRY.html) for the internal session. Each code page supported by SAP is assigned an [SAP code page number](ABENSAP_CODE_PAGE_NUMBER_GLOSRY.html) in the [DDIC database table](ABENDDIC_DB_TABLE_GLOSRY.html)\\ `TCP00`. See also [system code page](ABENSYSTEM_CODEPAGE_GLOSRY.html).
