`BF` for short. [Repository object](ABENREPOSITORY_OBJECT_GLOSRY.html) that represents a self-contained (business) function within the scope of the [Switch Framework](ABENSWITCH_FRAMEWORK_GLOSRY.html). Multiple business functions are grouped into a [Business Function Set](ABENBUSINESS_FUNCTION_SET_GLOSRY.html). Technically, a business function is a grouping of [switches](ABENSWITCH_GLOSRY.html) that serve the same function. When a business function is activated, the corresponding switches are activated as well. Business functions are created in [transaction](ABENTRANSACTION_GLOSRY.html)\\ `SFW2` and switched in transaction `SFW5`.
