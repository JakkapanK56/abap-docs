Stores [graph](ABENGRAPH_GLOSRY.html) data in combination with [edges](ABENEDGE_GLOSRY.html). A vertex can be considered as a node in a network. One vertex is denoted as the source and the other as the target. An edge connects two vertices. Vertices can have any number of [attributes](ABENATTRIBUTE_GLOSRY.html). An attribute consists of a name that is associated with a [data type](ABENDATA_TYPE_GLOSRY.html) and a value.
