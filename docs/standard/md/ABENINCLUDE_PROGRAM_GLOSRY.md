[ABAP program](ABENABAP_PROGRAM_GLOSRY.html) that cannot be compiled independently but must be included in another ABAP program using the statement [`INCLUDE`](ABAPINCLUDE_PROG.html). To be able to be compiled, an include program must be part of a [compilation unit](ABENCOMPILATION_UNIT_GLOSRY.html). An include program should only be assigned to a single compilation unit.
