Initial program that is called after a dialog based logon to an [AS ABAP](ABENAS_ABAP_GLOSRY.html). SAP Easy Access displays an initial menu for the access to the AS ABAP by a dialog user. By default, this menu is either the SAP menu or a user menu. The SAP menu is an [area menu](ABENAREA_MENU_GLOSRY.html) that is centrally defined or overridden by an entry in the [user master record](ABENUSER_MASTER_RECORD_GLOSRY.html). A user menu contains all the functions required by a dialog user and is created and assigned by the system administrator via the role maintenance (transaction `PFCG`). Each user can also define their favorites menu in SAP Easy Access that is displayed together with or instead of the default menu.
