Template for the [public](ABENPUBLIC_GLOSRY.html)\\ [interface](ABENINTF_GEN_GLOSRY.html) of [classes](ABENCLASS_GLOSRY.html) in [ABAP Objects](ABENABAP_OBJECTS_GLOSRY.html). Defined with [`INTERFACE - ENDINTERFACE`](ABAPINTERFACE.html), either as a [global interface](ABENGLOBAL_INTERFACE_GLOSRY.html) in an [interface pool](ABENINTERFACE_POOL_GLOSRY.html) or as a [local interface](ABENLOCAL_INTERFACE_GLOSRY.html) in another [ABAP program](ABENABAP_PROGRAM_GLOSRY.html). An interface contains the declaration of [interface components](ABENINTERFACE_COMPONENT_GLOSRY.html) but does not contain method implementations. Can be implemented in classes using [`INTERFACES`](ABAPINTERFACES.html), which enhances the external interface of the class by adding interface components. OO interfaces are generally referred to as interfaces, unless the distinction to [general interfaces](ABENINTF_GEN_GLOSRY.html) needs to be made.
