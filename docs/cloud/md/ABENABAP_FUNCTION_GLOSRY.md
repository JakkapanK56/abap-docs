A [function](ABENFUNCTION_GLOSRY.html) in ABAP can either be predefined as a [built-in function](ABENBUILTIN_FUNCTION_GLOSRY.html) or can be defined as a [functional method](ABENFUNCTIONAL_METHOD_GLOSRY.html). In an [ABAP program](ABENABAP_PROGRAM_GLOSRY.html), an ABAP function can be called by an [expression](ABENEXPRESSION_GLOSRY.html) by using its name at an [operand position](ABENOPERAND_POSITION_GLOSRY.html). During the evaluation of the expression, the function is called and its result is used as an [operand](ABENOPERAND_GLOSRY.html).
