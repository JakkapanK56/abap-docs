An arrangement of objects in hierarchical structures. The objects of a hierarchy are [hierarchy nodes](ABENHIERARCHY_NODE_GLOSRY.html) that are linked to each other using [hierarchy edges](ABENHIERARCHY_EDGE_GLOSRY.html). In [ABAP SQL](ABENABAP_SQL_GLOSRY.html) and [ABAP CDS](ABENABAP_CDS_GLOSRY.html), hierarchical data can be handled with [ABAP SQL hierarchies](ABENSQL_HIERARCHY_GLOSRY.html) and [CDS hierarchies](ABENCDS_HIERARCHY_GLOSRY.html).
