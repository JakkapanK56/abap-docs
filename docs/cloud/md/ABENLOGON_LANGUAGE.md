The [logon language](ABENLOGON_LANGUAGE_GLOSRY.html) is defined when an [ABAP user](ABENABAP_USER_GLOSRY.html)\\ [logs on](ABENLOGON_GLOSRY.html) to an AS ABAP for the entire [user session](ABENUSER_SESSION_GLOSRY.html) and can no longer be changed.

The logon language affects the selection of the following texts:

-   Texts of [messages](ABENMESSAGE_GLOSRY.html)
-   Texts from [text pools](ABENTEXT_POOL_GLOSRY.html)
