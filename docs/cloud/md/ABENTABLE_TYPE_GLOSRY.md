[Data type](ABENDATA_TYPE_GLOSRY.html) of an [internal table](ABENINTERNAL_TABLE_GLOSRY.html). Defines the [line type](ABENROW_TYPE_GLOSRY.html), [table category](ABENTABLE_CATEGORY_GLOSRY.html), and [table key](ABENTABLE_KEY_GLOSRY.html) of an internal table, as well as an [initial memory requirement](ABENINITIAL_MEM_REQ_GLOSRY.html). The table type can be [generic](ABENGENERIC_DATA_TYPE_GLOSRY.html) with regard to the key. A table type can either be defined in [ABAP programs](ABENABAP_PROGRAM_GLOSRY.html) using the addition [`TABLE OF`](ABAPTYPES_ITAB.html) or as a [DDIC table type](ABENDDIC_TABLE_TYPES.html) in the [ABAP Dictionary](ABENABAP_DICTIONARY_GLOSRY.html).
