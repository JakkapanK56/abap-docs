[Attribute](ABENATTRIBUTE_GLOSRY.html) of a [class](ABENCLASS_GLOSRY.html) that is declared using [`DATA`](ABAPDATA.html). Instance attributes are only valid in the [context](ABENOBJ_CONTEXT_GLOSRY.html) of an [instance](ABENINSTANCE_GLOSRY.html) of the class. The content of the instance attributes determines the state of the [objects](ABENOBJECT_GLOSRY.html) of a class.
