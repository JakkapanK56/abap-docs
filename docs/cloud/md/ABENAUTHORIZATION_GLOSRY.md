Entry in the [user master record](ABENUSER_MASTER_RECORD_GLOSRY.html). An authorization consists of complete or generic values for the [authorization fields](ABENAUTHORIZATION_FIELD_GLOSRY.html) of an [authorization object](ABENAUTHORIZATION_OBJECT_GLOSRY.html). The combination specifies which activities an [ABAP user](ABENABAP_USER_GLOSRY.html) can use to access which data.
