Term in [ABAP Objects](ABENABAP_OBJECTS_GLOSRY.html). A private [component](ABENCLASS_COMPONENT_GLOSRY.html) of a [class](ABENCLASS_GLOSRY.html) can be accessed only by the class itself and its [friends](ABENFRIEND_GLOSRY.html), which also only can create [objects](ABENOBJECT_GLOSRY.html) of a class with private instances. The private [visibility section](ABENVISIBILITY_SECTION_GLOSRY.html) of a class is declared in its [declaration part](ABENDECLARATION_PART_GLOSRY.html) using the statement [`PRIVATE SECTION`](ABAPPRIVATE.html).
