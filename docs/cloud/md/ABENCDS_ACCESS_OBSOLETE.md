-   **Access to and typing with [CDS-managed DDIC views](ABENCDS_MNGDDDIC_VIEW_GLOSRY.html) is obsolete**
-   A CDS-managed DDIC view (obsolete), specified using the annotation [`@AbapCatalog.sqlViewName`](ABENCDS_DEFINE_VIEW_V1.html) in a [CDS DDIC-based view (obsolete)](ABENCDS_V1_VIEW_GLOSRY.html), must be considered as purely technical helper construct and should not be used at all.
-   In particular, accessing a CDS-managed DDIC view from ABAP SQL is obsolete.
