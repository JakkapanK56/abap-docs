Short for XML-binary Optimized Packaging (see [http://www.w3.org/TR/xop10/](http://www.w3.org/TR/xop10/)). Representation of [XML](ABENXML_GLOSRY.html) data in which any binary raw data is stored separately from the data represented by characters. In ABAP, XOP is supported by a [format](ABENABAP_SXML_LIB_FORMATS.html) of the [sXML Library](ABENABAP_SXML_LIB.html).
