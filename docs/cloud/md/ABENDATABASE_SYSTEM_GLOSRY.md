System for storing large amounts of data in a [database](ABENDATABASE_GLOSRY.html). Besides the database, a database system consists of a database management system that manages the database and offers access to the data via a programming interface ([SQL](ABENSQL_GLOSRY.html) for [relational databases](ABENRELATIONAL_DATABASE_GLOSRY.html)).
