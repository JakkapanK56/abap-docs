[ABAP program](ABENABAP_PROGRAM_GLOSRY.html), administrated by the [ABAP Dictionary](ABENABAP_DICTIONARY_GLOSRY.html), that is introduced by the statement `TYPE-POOL` and which contains the definitions of globally visible [data types](ABENDATA_TYPE_GLOSRY.html), [constants](ABENCONSTANT_GLOSRY.html). A type pool is loaded the first time its data types and constants are used.
