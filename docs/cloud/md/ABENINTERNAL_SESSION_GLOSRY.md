Instance within an [ABAP session](ABENABAP_SESSION_GLOSRY.html) with its own [program memory](ABENROLL_AREA_GLOSRY.html) in which the data and [objects](ABENOBJECT_GLOSRY.html) of an [ABAP program](ABENABAP_PROGRAM_GLOSRY.html) are stored during its execution. An internal session is created when an ABAP program is called in the [shared memory](ABENSHARED_MEMORY_GLOSRY.html).
