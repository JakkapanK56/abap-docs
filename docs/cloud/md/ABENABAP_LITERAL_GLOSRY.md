An [unnamed data object](ABENUNNAMED_DATA_OBJECT_GLOSRY.html) defined by textual specification of a value in the [ABAP source code](ABENABAP_SOURCE_CODE_GLOSRY.html) of an [ABAP program](ABENABAP_PROGRAM_GLOSRY.html). In the general ABAP language, untyped [numeric literals](ABENNUMERIC_LITERAL_GLOSRY.html) and untyped [character literals](ABENCHARACTER_LITERAL_GLOSRY.html) are available. A [literal operator](ABENLITERAL_OPERATOR_GLOSRY.html) enables the linking of multiple literals. See also:
