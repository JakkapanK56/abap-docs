Character `->`. Also referred to as [instance component selector](ABENINSTANCE_COMP_SELECTOR_GLOSRY.html). Each visible [component](ABENCOMPONENT_GLOSRY.html)\\ `comp` of a [data object](ABENDATA_OBJECT_GLOSRY.html) or an [instance](ABENINSTANCE_GLOSRY.html) of a [class](ABENCLASS_GLOSRY.html) can be addressed using `ref->comp` if `ref` is a [reference variable](ABENREFERENCE_VARIABLE_GLOSRY.html) that refers to the object.
