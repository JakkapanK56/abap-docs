Component of a [DDIC structure](ABENDDIC_STRUCTURE_GLOSRY.html), or [DDIC database table](ABENDDIC_DB_TABLE_GLOSRY.html) of type `UNIT` defined in the [ABAP Dictionary](ABENABAP_DICTIONARY_GLOSRY.html). A unit key can be linked to a [quantity field](ABENQUANTITY_GLOSRY.html) of a structure, database table, or view. The unit key can contain a [unit ID](ABENUNIT_ID_GLOSRY.html), and defines the unit for the quantity field.
