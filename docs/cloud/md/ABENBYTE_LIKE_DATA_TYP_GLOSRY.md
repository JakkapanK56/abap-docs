[Data type](ABENDATA_TYPE_GLOSRY.html) that allows access to the unencoded bytes of [byte chains](ABENBYTE_CHAIN_GLOSRY.html). These data types are [byte field types](ABENBYTE_FIELD_TYPE_GLOSRY.html) and [byte string types](ABENBYTE_STRING_TYPE_GLOSRY.html). The associated [generic data type](ABENGENERIC_DATA_TYPE_GLOSRY.html) is [`xsequence`](ABENBUILT_IN_TYPES_GENERIC.html).
