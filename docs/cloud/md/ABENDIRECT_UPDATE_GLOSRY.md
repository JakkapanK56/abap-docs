Procedure to create or update unique [secondary table keys](ABENSECONDARY_TABLE_KEY_GLOSRY.html) of [internal tables](ABENINTERNAL_TABLE_GLOSRY.html). After an insertion operation using [`INSERT`](ABAPINSERT_ITAB.html) or [`APPEND`](ABAPAPPEND.html), a modifying operating using [`MODIFY`](ABAPMODIFY_ITAB.html), or a block operation in which the whole table body is filled in one go, unique secondary table keys are updated immediately. See also [delayed update](ABENDELAYED_UPDATE_GLOSRY.html) and [lazy update](ABENLAZY_UPDATE_GLOSRY.html).
