String functions are part of the [built-in functions](ABENBUILT_IN_FUNCTIONS.html). They are divided into:

Description functions have a numeric result and processing functions have a character-like result.

Parameters that are used by more than one function in the same way are described under [Shared Parameters of String Functions](ABENSTRING_FUNCTIONS_COMMON_PARAS.html).

-   [Description functions](ABENDESCRIPTIVE_FUNCTIONS.html)
-   [Processing functions](ABENPROCESS_FUNCTIONS.html)
