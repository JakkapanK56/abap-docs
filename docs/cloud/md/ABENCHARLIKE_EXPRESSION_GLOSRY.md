Represents a [string](ABENCHARACTER_STRING_GLOSRY.html). A character-like expression is a [constructor expression](ABENCONSTRUCTOR_GLOSRY.html), a [string expression](ABENSTRING_EXPRESSION_GLOSRY.html), a [table expression](ABENTABLE_EXPRESSION_GLOSRY.html), or a [function](ABENFUNCTION_GLOSRY.html), the return value of which has a [character-like data type](ABENCHARLIKE_DATA_TYPE_GLOSRY.html).
