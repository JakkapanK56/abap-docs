[Data object](ABENDATA_OBJECT_GLOSRY.html) of [time type](ABENTIME_TYPE_GLOSRY.html)\\ [`t`](ABENBUILTIN_TYPES_DATE_TIME.html) for storing a character-like [time](ABENDAY_TIME_GLOSRY.html). Valid values have the format *hhmmss*. The internal representation takes place in the [character representation](ABENCHAR_REPRESENTATION_GLOSRY.html) of the current [code page](ABENCODEPAGE_GLOSRY.html). For assignments, data objects with [numeric data types](ABENNUMERIC_DATA_TYPE_GLOSRY.html) and for calculations, the content of the time field is converted to the number of seconds since 00:00:00.
