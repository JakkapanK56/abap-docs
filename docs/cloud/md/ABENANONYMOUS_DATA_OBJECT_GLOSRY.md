[Unnamed data object](ABENUNNAMED_DATA_OBJECT_GLOSRY.html) that is created using the statement [`CREATE DATA`](ABAPCREATE_DATA.html), the [instance operator](ABENINSTANCE_OPERATOR_GLOSRY.html)\\ [`NEW`](ABENCONSTRUCTOR_EXPRESSION_NEW.html), or the addition [`NEW`](ABAPSELECT_INTO_TARGET.html) of the [`INTO`](ABAPINTO_CLAUSE.html) clause of a [query](ABENASQL_QUERY_GLOSRY.html) and can be addressed using [reference variables](ABENREFERENCE_VARIABLE_GLOSRY.html) or [field symbols](ABENFIELD_SYMBOL_GLOSRY.html). Anonymous data objects are stored in the [heap](ABENHEAP_GLOSRY.html). An anonymous data object is kept alive by [heap references](ABENHEAP_REFERENCE_GLOSRY.html) or by field symbols that address it or part of it. See also [named data object](ABENNAMED_DATA_OBJECT_GLOSRY.html) and [literal](ABENABAP_LITERAL_GLOSRY.html).
