In an [ABAP program](ABENABAP_PROGRAM_GLOSRY.html), a logpoint is defined statically as an unconditional [checkpoint](ABENCHECKPOINT_GLOSRY.html) using the statement `LOG-POINT`, or dynamically in the [ABAP development tools for Eclipse (ADT)](ABENADT_GLOSRY.html). When an active logpoint is reached, a log entry is created. A static logpoint must be assigned to a [checkpoint group](ABENCHECKPOINT_GROUP_GLOSRY.html) when it is created. This group controls its [activation](ABENACTIVATABLE_CHECKPOINT_GLOSRY.html).
