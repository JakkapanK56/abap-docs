[Object reference variable](ABENOBJECT_REFER_VARIABLE_GLOSRY.html) called [`me`](ABENME.html) that is predefined as a local [constant](ABENCONSTANT_GLOSRY.html) in each [instance method](ABENINSTANCE_METHOD_GLOSRY.html). It always references the current [instance](ABENINSTANCE_GLOSRY.html) in which the method is being executed. The static type of `me` is the [class](ABENCLASS_GLOSRY.html) in which the instance method is implemented.
