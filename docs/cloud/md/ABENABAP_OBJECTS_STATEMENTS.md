The statements in ABAP Objects can be differentiated as follows:

-   [Statements for defining classes and interfaces](ABENDEFINING_CLASSES_INTERFACES.html)
-   These statements define classes and interfaces and their components. They can be used for local and global classes or interfaces.
-   [Statements for implementing methods](ABENDEFINING_METHODS.html)
-   These statements implement the functions of classes in methods.
-   [Statements in class pools and interface pools](ABENCLASS_INTERACE_POOLS.html)
-   In addition to the statements for defining the global class, statements for local declarations and implementations can also be used in [class pools](ABENCLASS_POOL_GLOSRY.html). In [interface pools](ABENINTERFACE_POOL_GLOSRY.html), only the global interface can be declared.
