[Character-like data type](ABENCHARLIKE_DATA_TYPE_GLOSRY.html) for storing [character strings](ABENCHARACTER_STRING_GLOSRY.html) of variable length. The associated [built-in ABAP type](ABENBUILTIN_ABAP_TYPE_GLOSRY.html) is [`string`](ABENBUILTIN_TYPES_CHARACTER.html). The [built-in DDIC types](ABENBUILTIN_DDIC_TYPE_GLOSRY.html) are [`STRING`](ABENDDIC_BUILTIN_TYPES.html#abenddic-builtin-types-3--and---sstring---abenddic-builtin-types-html-@ITOC@@ABENDDIC_BUILTIN_TYPES_3). The internal representation takes place in the [character representation](ABENCHAR_REPRESENTATION_GLOSRY.html) of the current [code page](ABENCODEPAGE_GLOSRY.html). Internal administration takes place using an internal [reference](ABENREFERENCE_GLOSRY.html) and the data type is [deep](ABENDEEP_GLOSRY.html). A [data object](ABENDATA_OBJECT_GLOSRY.html) of a text field type is a [text string](ABENTEXT_STRING_GLOSRY.html).
