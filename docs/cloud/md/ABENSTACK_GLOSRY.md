Part of the [program memory](ABENROLL_AREA_GLOSRY.html) in which [named data objects](ABENNAMED_DATA_OBJECT_GLOSRY.html) of programs and [procedures](ABENPROCEDURE_GLOSRY.html) are stacked. The memory used by a procedure is released again once the procedure ends. See also [heap](ABENHEAP_GLOSRY.html).
