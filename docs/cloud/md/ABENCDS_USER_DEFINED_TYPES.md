A [CDS user-defined types](ABENCDS_USER_DEFINED_TYPE_GLOSRY.html) is defined as a [CDS object](ABENCDS_OBJECT_GLOSRY.html) in [ABAP CDS](ABENABAP_CDS_GLOSRY.html). CDS user-defined types can be used as [global types](ABENGLOBAL_TYPE_GLOSRY.html).

The following CDS user-defined types are available:

For details, see the topic [ABAP CDS - Type Definitions](ABENCDS_TDL.html).

-   [CDS simple types](ABENCDS_SIMPLE_TYPES.html) for defining [elementary data types](ABENELEMENTARY_DATA_TYPE_GLOSRY.html).
-   [CDS enumerated types](ABENCDS_ENUMERATION_TYPES.html) for defining [enumerated types](ABENENUM_TYPE_GLOSRY.html).
