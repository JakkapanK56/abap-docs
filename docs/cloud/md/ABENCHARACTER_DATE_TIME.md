The ABAP types [`d`](ABENBUILTIN_TYPES_DATE_TIME.html) and [`t`](ABENBUILTIN_TYPES_DATE_TIME.html) are built-in types for data objects that specify dates and times. The date type `d` describes a character-like local date in the format *yyyymmdd*. The date type `t` describes a character-like local time in the format `hhmmss`.
