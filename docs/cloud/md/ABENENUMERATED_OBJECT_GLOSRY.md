[Data object](ABENDATA_OBJECT_GLOSRY.html) with an [enumerated type](ABENENUM_TYPE_GLOSRY.html). An enumerated object can only contain the [enumerated values](ABENENUMERATED_VALUE_GLOSRY.html) of the enumerated type. The data type of an enumerated object is the enumerated type. The technical data type of the content is the [base type](ABENBASE_TYPE_GLOSRY.html) of the enumerated type. [Enumerated variables](ABENENUMERATED_VARIABLE_GLOSRY.html) are variable enumerated objects that can only contain the associated enumerated values. [Enumerated constants](ABENENUMERATED_CONSTANT_GLOSRY.html) and components of [enumerated structures](ABENENUMERATED_STRUCTURE_GLOSRY.html) are special enumerated objects that define the value set of an enumerated type.
