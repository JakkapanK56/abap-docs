[1\. Managed RAP BO](#ABAP_MODIFICATION_1@4@)

[2\. Business Object Projection](#ABAP_MODIFICATION_2@4@)

The new statement [`managed`](ABENBDL_IMPL_TYPE.html) can be used to create [managed RAP BOs](ABENMANAGED_RAP_BO_GLOSRY.html) within the framework of the ABAP RESTful Application Programming Model. This scenario is intended for greenfield developments that are developed from scratch. Standard operations and services are provided by the RAP framework.

The new statement [`projection`](ABENBDL_IMPL_TYPE.html) can be used to create [projections](ABENRAP_PROJECTION_BO_GLOSRY.html) of a business object. This projects and aliases a subset of a business object for a specific business service within the framework of the ABAP RESTful Application Programming Model.
