It is possible to extend a [BDEF extension](ABENRAP_EXTENSION_GLOSRY.html) with further BDEF extensions. This is referred to as [extension layering](ABENRAP_EXT_LAYERING_GLOSRY.html). As a prerequisite, a BDEF extension must be explicitly enabled for extensibility. It must fulfill technical requirements and define extension points.

The following topics describe how to enable [base BDEF extensions](ABENRAP_BASE_BDEF_EXT_GLOSRY.html), [BDEF projection extensions](ABENRAP_PROJEC_BDEF_EXT_GLOSRY.html), and [abstract BDEF extensions](ABENRAP_ABS_BDEF_EXT_GLOSRY.html) for extensibility.

The general rules for extension layering are described in the topic [RAP - Extension Layering](ABENBDL_EXTENSION_LAYERING.html).

-   [RAP - Extensibility Enabling for Base BDEF Extensions](ABENBDL_EXT_ENABL_EXT_BASE.html)
-   [RAP - Extensibility Enabling for Projection BDEF Extensions](ABENBDL_EXT_ENABL_EXT_PROJ.html)
-   [RAP - Extensibility Enabling for Abstract BDEF Extensions](ABENBDL_EXT_ENABL_EXT_ABS.html)
