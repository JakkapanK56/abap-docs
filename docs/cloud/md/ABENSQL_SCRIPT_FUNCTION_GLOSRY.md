A [database function](ABENDATABASE_FUNCTION_GLOSRY.html) of the [SAP HANA database](ABENHANA_DATABASE_GLOSRY.html) programmed in the language [SQLScript](ABENSQL_SCRIPT_GLOSRY.html). [ABAP Managed Database Procedures (AMDP)](ABENABAP_MANAGED_DB_PROC_GLOSRY.html) enable SQLScript functions to be implemented in [AMDP procedure implementations](ABENAMDP_FUNCTION_METHOD_GLOSRY.html) of [AMDP classes](ABENAMDP_CLASS_GLOSRY.html). See also [SQLScript procedure](ABENSQL_SCRIPT_PROCEDURE_GLOSRY.html).
