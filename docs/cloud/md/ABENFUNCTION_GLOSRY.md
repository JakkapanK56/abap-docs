A function calculates a [return value](ABENRETURN_VALUE_GLOSRY.html) from one or more [actual parameters](ABENACTUAL_PARAMETER_GLOSRY.html) or [arguments](ABENARGUMENT_GLOSRY.html). Functions can be specified as [expressions](ABENEXPRESSION_GLOSRY.html) or part of expressions in [operand positions](ABENOPERAND_POSITION_GLOSRY.html).
