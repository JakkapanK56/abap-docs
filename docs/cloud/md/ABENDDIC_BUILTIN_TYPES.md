The following tables show the built-in types in [ABAP Dictionary](ABENABAP_DICTIONARY_GLOSRY.html) and their fundamental technical properties. For each built-in data type in ABAP Dictionary, the last column specifies the [built-in ABAP type](ABENBUILTIN_ABAP_TYPE_GLOSRY.html) to which the built-in dictionary type is mapped when used as type reference in an ABAP program. Most built-in dictionary types are assigned an initial value. These initial values are applied, for example, when database fields are [initialized](ABENDDIC_DATABASE_TABLES_INIT.html), and in the ABAP SQL condition [`IS INITIAL`](ABENWHERE_LOGEXP_INITIAL.html). The initial value of ABAP Dictionary types is not used for data objects declared in ABAP programs when referring to these types. Here, the initial value of the mapped [built-in ABAP types](ABENBUILT_IN_TYPES_COMPLETE.html) is used.
