Operator for a [calculation assignment](ABENCALCULATION_ASSIGNMENT_GLOSRY.html). The following calculation assignment operators are available: [addition assignment operator](ABENADD_ASSIGNMENT_OP_GLOSRY.html) ([`+=`](ABENCALCULATION_ASSIGNMENTS.html)), a [subtraction assignment operator](ABENSUB_ASSIGNMENT_OP_GLOSRY.html) ([`-=`](ABENCALCULATION_ASSIGNMENTS.html)), a [multiplication assignment operator](ABENMULT_ASSIGNMENT_OP_GLOSRY.html) ([`*=`](ABENCALCULATION_ASSIGNMENTS.html)), a [division assignment operator](ABENDIV_ASSIGNMENT_OP_GLOSRY.html) ([`/=`](ABENCALCULATION_ASSIGNMENTS.html)), or a [concatenation assignment operator](ABENCONCAT_ASSIGNMENT_OP_GLOSRY.html) ([`&&=`](ABENCALCULATION_ASSIGNMENT_STRING.html)).
