Superordinate term for [static attribute](ABENSTATIC_ATTRIBUTE_GLOSRY.html), [static event](ABENSTATIC_EVENT_GLOSRY.html), and [static method](ABENSTATIC_METHOD_GLOSRY.html). The static components of a [class](ABENCLASS_GLOSRY.html) can be addressed using the name of the class. A class that only contains static components is referred to as a [static class](ABENSTATIC_CLASS_GLOSRY.html).
