[Character-like data type](ABENCHARLIKE_DATA_TYPE_GLOSRY.html) for storing [character strings](ABENCHARACTER_STRING_GLOSRY.html) of fixed length. The associated [built-in ABAP type](ABENBUILTIN_ABAP_TYPE_GLOSRY.html) is [`c`](ABENBUILTIN_TYPES_CHARACTER.html). The [built-in DDIC types](ABENBUILTIN_DDIC_TYPE_GLOSRY.html) are [`CHAR`](ABENDDIC_BUILTIN_TYPES.html#abenddic-builtin-types-3--and---lchr---abenddic-builtin-types-html-@ITOC@@ABENDDIC_BUILTIN_TYPES_3). The internal representation takes place in the [character representation](ABENCHAR_REPRESENTATION_GLOSRY.html) of the current [code page](ABENCODEPAGE_GLOSRY.html). A [data object](ABENDATA_OBJECT_GLOSRY.html) of a text field type is a [text field](ABENTEXT_FIELD_GLOSRY.html).
