A set of [ATC](ABENATC_GLOSRY.html) checks is shipped with SAP Basis that [ATC](ABENATC_GLOSRY.html) checks shipped by SAP that check if [released APIs](ABENRELEASED_API_GLOSRY.html) fulfill the rules of their [release contracts](ABENRELEASE_CONTRACT_GLOSRY.html).

The following topics display the [KTD](ABENKTD_GLOSRY.html) documentation of these checks. The consistency checks are relevant for rules regarding the prerequisites for release and the compatibility checks are relevant for the stability rules after release.

-   [Consistency Checks for Released APIs](ABENRESTRICTED_APIS_ATC_CONSI.html)
-   [Compatibility Checks for Released APIs](ABENRESTRICTED_APIS_ATC_COMPA.html)
