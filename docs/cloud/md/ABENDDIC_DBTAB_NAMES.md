The name of a database table defined in the ABAP Dictionary must adhere to the following rules:

-   It can have a maximum of 16 characters.
-   It must follow the [naming rules for DDIC types](ABENDDIC_TYPES_NAMES.html).
-   It must follow the special [naming rules for DDIC structures](ABENDDIC_STRUCTURES_NAMES.html).
-   It is in the namespace of [DDIC data types](ABENDDIC_DATA_TYPES.html) and hence in the namespace of all [global types](ABENGLOBAL_TYPE_GLOSRY.html) of an AS ABAP.
