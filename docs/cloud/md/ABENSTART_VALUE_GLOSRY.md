Value specified when a [data object](ABENDATA_OBJECT_GLOSRY.html) is declared. The data object is filled with this value when it is created. The start value is set using the [`VALUE`](ABAPDATA_OPTIONS.html) addition of the [`DATA`](ABAPDATA.html) statement. If the `VALUE` addition is not used, the system uses the type-specific [initial value](ABENINITIAL_VALUE_GLOSRY.html).
