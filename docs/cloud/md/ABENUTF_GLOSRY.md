Short for Universal Transformation Format. Different representations of the [Unicode](ABENUNICODE_GLOSRY.html)\\ [character set](ABENCHARACTER_SET_GLOSRY.html). All UTF character representations objectively refer to the characters defined in Unicode and can be transformed into each other. The representations differ only in the type of addressing. See [UTF-8](ABENUTF8_GLOSRY.html), [UTF-16](ABENUTF16_GLOSRY.html), and [UTF-32](ABENUTF32_GLOSRY.html) plus [UCS-2](ABENUCS2_GLOSRY.html).
