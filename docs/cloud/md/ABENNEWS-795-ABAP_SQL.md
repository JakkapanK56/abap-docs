[1\. Caching of Hierarchies](#ABAP_MODIFICATION_1@4@)

[2\. Database Hints for the ABAP SQL In-Memory Engine](#ABAP_MODIFICATION_2@4@)

The new addition `CACHE` for [`SELECT, FROM HIERARCHY`](ABENSELECT_HIERARCHY_GENERATOR.html) allows the caching policy to be defined for hierarchies.

Behind the addition [`%_HINTS`](ABENABAP_SQL_DB_HINTS.html), the database system `ABAP` can now be specified for listing [database hints](ABENDATABASE_HINT_GLOSRY.html) for the [ABAP SQL in-memory engine](ABENABAP_SQL_INMEMENG_GLOSRY.html). These database hints are documented under [ABAP SQL In-Memory Engine, Database Hints](ABENSQL_ENGINE_DBHINTS.html). In the current release, there are not yet any hints available for the ABAP SQL in-memory engine.
