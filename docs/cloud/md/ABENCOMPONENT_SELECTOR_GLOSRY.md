Syntax that can be used to address [components](ABENCOMPONENT_GLOSRY.html) of higher units. There is the [structure component selector](ABENSTRUCTURE_COMPONENT_SEL_GLOSRY.html) (`-`), the [class component selector](ABENCLASS_COMPONENT_SELECT_GLOSRY.html) (`=>`), the [interface component selector](ABENINTERFACE_COMP_SELECTOR_GLOSRY.html) (`~`), and the [object component selector](ABENOBJECT_COMPONENT_SELECT_GLOSRY.html) (`->`). [ABAP SQL](ABENABAP_SQL_GLOSRY.html) provides the additional [column selector](ABENTABLE_COMP_SELECTOR_GLOSRY.html)\\ `~`.
