A conditional expression is a [constructor expression](ABENCONSTRUCTOR_EXPRESSIONS.html) that creates a value or raises a class-based exception depending on a logical expression or a case distinction. Conditional expressions are constructed using the following conditional operators:

-   [`COND`](ABENCONDITIONAL_EXPRESSION_COND.html)
-   [`SWITCH`](ABENCONDITIONAL_EXPRESSION_SWITCH.html)
