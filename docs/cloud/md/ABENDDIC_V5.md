In [SAP BTP ABAP environment](ABENSAP_BTP_ABAP_ENV_GLOSRY.html) and [SAP S/4HANA Cloud ABAP environment](ABENS4_HANA_CLOUD_ABAP_ENV_GLOSRY.html), customers and partners can define DDIC data elements, DDIC structures, and DDIC database tables in ADT. The restriction applies to these dictionary objects that only repository objects [released as API](ABENRELEASED_API_GLOSRY.html) for [ABAP language version](ABENABAP_VERSION_GLOSRY.html)\\ [ABAP for Cloud Development](ABENABAP_FOR_CLOUD_DEV_GLOSRY.html) or objects in the same software component can be used.

See [Released APIs](ABENRELEASED_APIS.html) for a list of repository objects that are released as APIs in the current system.
