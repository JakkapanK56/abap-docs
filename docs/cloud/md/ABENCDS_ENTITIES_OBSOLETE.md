-   [CDS DDIC-based Entities](ABENCDS_DDIC_ENTITY.html): CDS DDIC-based entities are obsolete and as a consequence, CDS DDIC-based views (obsolete), representing a sub-category of CDS DDIC-based entities, are obsolete. Accessing a [CDS-managed DDIC view](ABENCDS_MNGDDDIC_VIEW_GLOSRY.html), which is created for each CDS DDIC-based view (obsolete), is [obsolete](ABENCDS_ACCESS_OBSOLETE.html) as well.
