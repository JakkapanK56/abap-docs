Relation between a [test class](ABENTEST_CLASS_GLOSRY.html) or a [test method](ABENTEST_METHOD_GLOSRY.html) and another [repository object](ABENREPOSITORY_OBJECT_GLOSRY.html). A test relation is defined by the special [ABAP Doc comment](ABENABAP_DOC_COMMENT_GLOSRY.html)\\ [`"! @testing`](ABENTEST_RELATIONS.html) and allows repository objects to be tested with [ABAP Unit](ABENABAP_UNIT_GLOSRY.html) in [ADT](ABENADT_GLOSRY.html) that do not support test classes themselves.
