Special [class](ABENCLASS_GLOSRY.html) that is the basis for [catchable](ABENCATCHABLE_EXCEPTION_GLOSRY.html) exceptions. If an [exception](ABENEXCEPTION_GLOSRY.html) is raised, an [object](ABENOBJECT_GLOSRY.html) of an exception class is created. There are predefined exception classes for exceptions of the runtime framework, and user-defined exception classes for custom applications. The superclasses for all exception classes are `CX_STATIC_CHECK`, `CX_DYNAMIC_CHECK`, or `CX_NO_CHECK`, through which exceptions are divided into [exception categories](ABENEXCEPTION_CATEGORY_GLOSRY.html).
