The following [built-in functions](ABENBUILTIN_FUNCTION_GLOSRY.html) for processing UTC time stamps in time stamp fields are available:

-   [`utclong_current`](ABENUTCLONG_CURRENT.html) creates a current time stamp
-   [`utclong_add`](ABENUTCLONG_ADD.html) adds values to a time stamp
-   [`utclong_diff`](ABENUTCLONG_DIFF.html) determines the difference between two time stamps
