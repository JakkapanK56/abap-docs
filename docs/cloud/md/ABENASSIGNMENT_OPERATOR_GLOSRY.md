[Operator](ABENOPERATOR_GLOSRY.html) for assigning the content of a [data object](ABENDATA_OBJECT_GLOSRY.html) to a [variable](ABENVARIABLE_GLOSRY.html). The general assignment operator in ABAP is the equal sign ([`=`](ABENEQUALS_OPERATOR.html)). The [casting operator](ABENCASTING_OPERATOR_GLOSRY.html) ([`?=`](ABAPMOVE_CAST.html)) is a special assignment operator. Furthermore, there is the [addition assignment operator](ABENADD_ASSIGNMENT_OP_GLOSRY.html) ([`+=`](ABENCALCULATION_ASSIGNMENTS.html)), the [subtraction assignment operator](ABENSUB_ASSIGNMENT_OP_GLOSRY.html) ([`-=`](ABENCALCULATION_ASSIGNMENTS.html)), the [multiplication assignment operator](ABENMULT_ASSIGNMENT_OP_GLOSRY.html) ([`*=`](ABENCALCULATION_ASSIGNMENTS.html)), the [division assignment operator](ABENDIV_ASSIGNMENT_OP_GLOSRY.html) ([`/=`](ABENCALCULATION_ASSIGNMENTS.html)), and the [concatenation assignment operator](ABENCONCAT_ASSIGNMENT_OP_GLOSRY.html) ([`**=`](ABENCALCULATION_ASSIGNMENT_STRING.html)) for [calculation assignments](ABENCALCULATION_ASSIGNMENT_GLOSRY.html).
