The following table shows the SQL functions for strings that can be used by [ABAP CDS](ABENABAP_CDS_GLOSRY.html) and [ABAP SQL](ABENABAP_SQL_GLOSRY.html). The columns **CDS DDIC-Based Views**, **CDS View Entities** and **ABAP SQL** indicate where a function can be used. The **ABAP SQL In-Memory Engine** column indicates whether the function can be executed in the [ABAP SQL in-memory engine](ABENABAP_SQL_INMEMENG_GLOSRY.html).

The possible operands and data types are described in the corresponding documentation for [ABAP CDS, DDIC-Based Views](ABENCDS_SQL_FUNCTIONS_NUMERIC_V1.html), [ABAP CDS, View Entities](ABENCDS_SQL_FUNCTIONS_NUMERIC_V2.html) and [ABAP SQL](ABENSQL_STRING_FUNC.html).
