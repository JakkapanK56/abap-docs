ABAP [data object](ABENDATA_OBJECT_GLOSRY.html) specified in an operand position of an [ABAP SQL](ABENABAP_SQL_HOST_VARIABLES.html) statement. The ABAP data object can be a [variable](ABENVARIABLE_GLOSRY.html) or a [constant](ABENCONSTANT_GLOSRY.html) (see [host constant](ABENHOST_CONSTANT_GLOSRY.html)). In ABAP SQL, a host variable must be prefixed with the [escape character](ABENESCAPE_CHARACTER_GLOSRY.html)\\ `@`.
