ABAP provides a set of built-in functions. A built-in function returns a return value for one or more [arguments](ABENARGUMENT_GLOSRY.html) passed to the function. There are functions with exactly one unnamed [formal parameter](ABENFORMAL_PARAMETER_GLOSRY.html) and functions with one or more named formal parameters.

Built-in functions can be called functionally in [certain operand positions](ABENOPERANDS_EXPRESSIONS.html). When a built-in function is called in an [operand position](ABENEXPRESSION_POSITIONS.html), the function is executed before the operand is used and the return value of the built-in function is used as the operand, in accordance with its data type.
