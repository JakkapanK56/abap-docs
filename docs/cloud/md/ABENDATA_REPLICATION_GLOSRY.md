The process of maintaining multiple copies of the same data in different systems in a cross-system scenario. The data is replicated to the consumer system for a close to real-time consumption. All requests are executed on the replicated data in the consumer system.

Currently not supported in ABAP.

See also: [Data federation](ABENDATA_FEDERATION_GLOSRY.html)
