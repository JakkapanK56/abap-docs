When values are assigned to the nodes of [data roots](ABENST_TT_ROOT.html), it must be noted that the content of a data root cannot be modified by serialization. Only writes can be performed on a data root in deserializations.

-   [`tt:assign`, Value Assignment](ABENST_TT_ASSIGN.html)
-   [`tt:cast`, Downcast](ABENST_TT_CAST.html)
-   [`tt:clear`, Initialization](ABENST_TT_CLEAR.html)
