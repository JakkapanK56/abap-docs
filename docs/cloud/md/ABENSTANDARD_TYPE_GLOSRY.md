[Complete data type](ABENCOMPLETE_DATA_TYPE_GLOSRY.html) used for [generically typed](ABENGENERIC_TYPING_GLOSRY.html)\\ [field symbols](ABENFIELD_SYMBOL_GLOSRY.html) or [formal parameters](ABENFORMAL_PARAMETER_GLOSRY.html) if their data type is required for compilation, or if a `LIKE` reference is made to a generic field symbol, to which no memory area is assigned. Different standard types are defined for replacement parameters of optional formal parameters, generically typed field symbols, and for [inline declarations](ABENINLINE_DECLARATION_GLOSRY.html), where they depend on the current statement.
