Transfer of the value of a [data object](ABENDATA_OBJECT_GLOSRY.html), a [return value](ABENRETURN_VALUE_GLOSRY.html) or result of a [functional method](ABENFUNCTIONAL_METHOD_GLOSRY.html), a [built-in function](ABENBUILTIN_FUNCTION_GLOSRY.html), a [constructor expression](ABENCONSTRUCTOR_EXPRESSION_GLOSRY.html), a [table expression](ABENTABLE_EXPRESSION_GLOSRY.html), or the result of a [calculation expression](ABENCALCULATION_EXPRESSION_GLOSRY.html) to a target data object. If the [data types](ABENDATA_TYPE_GLOSRY.html) are [compatible](ABENCOMPATIBLE_GLOSRY.html), the content is copied unchanged. If they are incompatible and a suitable [conversion rule](ABENCONVERSION_RULE_GLOSRY.html) exists, the content is converted.
