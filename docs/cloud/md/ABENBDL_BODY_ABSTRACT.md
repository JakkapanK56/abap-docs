In the [entity behavior body](ABENCDS_ENTITY_BODY_GLOSRY.html) of an [abstract BDEF](ABENCDS_ABSTRACT_BDEF_GLOSRY.html), the following elements can be specified:

-   [field characteristics](ABENBDL_FIELD_ABSTRACT.html): [RAP field characteristics](ABENRAP_FIELD_CHAR_GLOSRY.html) are available to define field attributes for the fields of an abstract BDEF.
-   [Associations](ABENBDL_ASSOCIATION_ABSTRACT.html) can be included in the hierarchical BDEF derived type.
-   [Mapping](ABENBDL_TYPE_MAPPING_ABSTRACT.html): Data types of the abstract BDEF can be mapped to DDIC partner types. There are two possibilities: type mapping or deep mapping.
