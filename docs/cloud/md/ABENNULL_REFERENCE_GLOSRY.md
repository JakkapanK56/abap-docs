[Initial value](ABENINITIAL_VALUE_GLOSRY.html) of a [reference variable](ABENREFERENCE_VARIABLE_GLOSRY.html). The null reference does not point to an [object](ABENOBJECT_GLOSRY.html) and can be assigned to any reference variable. [Data objects](ABENDATA_OBJECT_GLOSRY.html) accessed using the null reference generally produce an [uncatchable exception](ABENUNCATCHABLE_EXCEPTION_GLOSRY.html). Method calls using the null reference produce a [catchable exception](ABENCATCHABLE_EXCEPTION_GLOSRY.html).
