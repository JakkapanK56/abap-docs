Creates the content of a [data object](ABENDATA_OBJECT_GLOSRY.html) by evaluating one or more [internal tables](ABENINTERNAL_TABLE_GLOSRY.html) using [table iterations](ABENTABLE_ITERATION_GLOSRY.html) with [iteration expressions](ABENITERATION_EXPRESSION_GLOSRY.html) in a [constructor expression](ABENCONSTRUCTOR_EXPRESSION_GLOSRY.html) with the [reduction operator](ABENREDUCE_OPERATOR_GLOSRY.html)\\ [`REDUCE`](ABENCONSTRUCTOR_EXPRESSION_REDUCE.html). This usually reduces the content of the internal table to a less complex data object.
