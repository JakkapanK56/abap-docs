SRID for short. An identifier that specifies a [spatial reference system](ABENSPATIAL_REF_SYSTEM_GLOSRY.html). The default value for the identifier of the spatial reference system is 0 and indicates a Cartesian coordinate system, i.e. 2D geometries.
