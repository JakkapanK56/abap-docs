Identity used by a [database user](ABENDATABASE_USER_GLOSRY.html) to log on to a [database system](ABENDATABASE_SYSTEM_GLOSRY.html). A database user name is assigned a [database schema](ABENDATABASE_SCHEMA_GLOSRY.html), whose objects can be accessed by the database user implicitly or as an unqualified user.
