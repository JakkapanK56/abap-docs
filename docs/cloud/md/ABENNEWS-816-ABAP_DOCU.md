[2\. CDS Annotation Documentation](#ABAP_MODIFICATION_2@4@)

From release 8.16 on, the ABAP Keyword Documentation is delivered and supported in English only.

From release 8.16 on, properties and documentation for [CDS annotations](ABENCDS_ANNOTATION_GLOSRY.html) delivered as [SAP annotations](ABENSAP_ANNOTATION_GLOSRY.html) are shown under [ABAP CDS - SAP Annotation Documentation](ABENCDS_ANNOTATIONS_KTD_DOCU.html). If available, the [KTD](ABENKTD_GLOSRY.html) of an annotation definition is embedded there and/or links to help portal documentation are shown.

The documentation of the [ABAP annotations](ABENABAP_ANNOTATION_GLOSRY.html) that formerly appeared at their point of use in the ABAP Keyword Documentation was migrated to KTDs and is shown under [ABAP Annotations](ABENCDS_ABAP_ANNOS.html) now.
