Text that can be accessed using the statement [`MESSAGE`](ABAPMESSAGE.html). Messages are always displayed in the [logon language](ABENLOGON_LANGUAGE_GLOSRY.html), which cannot be changed during a [user session](ABENUSER_SESSION_GLOSRY.html). Message texts are organized into [message classes](ABENMESSAGE_CLASS_GLOSRY.html) and [message numbers](ABENMESSAGE_NUMBER_GLOSRY.html). Messages are also used as exception texts of [exception classes](ABENEXCEPTION_CLASS_GLOSRY.html).
