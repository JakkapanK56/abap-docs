This example demonstrates how ABAP types are mapped using formatting options.

This example passes various types of ABAP data to the transformation `DEMO_ST_FORMAT_OPTION` and provides the result for XML and [JSON](ABENABAP_JSON.html). The transformation applies those [formatting options](ABENST_OPTION_FORMAT.html) to the ABAP data that match its data type:
