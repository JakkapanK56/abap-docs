[Interface](ABENINTF_GEN_GLOSRY.html) of an [AS ABAP](ABENAS_ABAP_GLOSRY.html) to its [database](ABENDATABASE_GLOSRY.html) that is integrated into the [ABAP runtime framework](ABENABAP_RUNTIME_FRMWK_GLOSRY.html). Statements in [ABAP SQL](ABENABAP_SQL_GLOSRY.html) access the database using the database interface. \\nThe database interface is responsible for data transport between the AS ABAP and the [database system](ABENDATABASE_SYSTEM_GLOSRY.html), implicit [client handling](ABENCLIENT_HANDLING_GLOSRY.html), and [table buffering](ABENTABLE_BUFFERING_GLOSRY.html). The database interface can also be involved in maintaining [ABAP-managed database objects](ABENABAP_MANAGED_DB_OBJECT_GLOSRY.html).
