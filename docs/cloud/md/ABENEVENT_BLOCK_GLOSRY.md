[Processing block](ABENPROCESSING_BLOCK_GLOSRY.html) An event block is processed when the corresponding [event](ABENEVENT_GLOSRY.html) of the [ABAP runtime framework](ABENABAP_RUNTIME_FRMWK_GLOSRY.html) is raised. It starts with an [event keyword](ABENEVENT_KEYWORD_GLOSRY.html) and ends at the start of the next processing block.
