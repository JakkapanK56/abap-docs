Short for Extensible Stylesheet Language Transformation, see [www.w3.org/TR/xslt](http://www.w3.org/TR/xslt). XSLT allows the conversion of [XML](ABENXML_GLOSRY.html) formats into any other XML formats. The [ABAP runtime framework](ABENABAP_RUNTIME_FRMWK_GLOSRY.html) contains the [SAP XSLT processor](ABENXSLT_PROCESSOR_GLOSRY.html) for the execution of [XSL transformations](ABENXSL_TRANSFORMATION_GLOSRY.html).
