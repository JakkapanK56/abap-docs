In [embedded expressions](ABENSTRING_TEMPLATES_EXPRESSIONS.html) of [string templates](ABENSTRING_TEMPLATE_GLOSRY.html) the formatting option `DECIMALS` can now be combined with `TIMESTAMP` and `TIME ZONE` to define the number of decimal places in UTC time stamps in packed numbers.
