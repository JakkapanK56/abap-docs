[UTC time stamp](ABENUTC_TIMESTAMP_GLOSRY.html) where each day has 86400 seconds in accordance with the standard [POSIX](https://en.wikipedia.org/wiki/POSIX) 1:2001 and [leap seconds](ABENLEAP_SECOND_GLOSRY.html) are not supported. A time stamp that is created during a leap second 23:59:60 is rounded to a valid time.
