In a [`LEFT OUTER JOIN`](ABENCDS_JOINED_DATA_SOURCE_V1.html), an addition `TO ONE` or `TO MANY` can be specified for the [cardinality](ABENCARDINALITY_GLOSRY.html). This addition is evaluated by an [SAP HANA database](ABENHANA_DATABASE_GLOSRY.html) as a note for optimization.

This option exists in earlier releases in ABAP CDS, but has been documented only from ABAP release 7.66. In [ABAP SQL](ABENNEWS-766-ABAP_SQL.html), the corresponding syntax was introduced in ABAP release 7.66.
