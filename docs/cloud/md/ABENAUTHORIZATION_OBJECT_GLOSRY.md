A [repository object](ABENREPOSITORY_OBJECT_GLOSRY.html) that is the basis of [authorizations](ABENAUTHORIZATION_GLOSRY.html). An authorization object consists of up to ten [authorization fields](ABENAUTHORIZATION_FIELD_GLOSRY.html). The combination of authorization fields that represent data and activities is used for [authorization assignments](ABENAUTHORIZATION_ASSIGN_GLOSRY.html). Authorization objects are grouped in authorization object classes.
