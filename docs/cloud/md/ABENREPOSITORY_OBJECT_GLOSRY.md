Repository objects are development objects in the [ABAP repository](ABENABAP_REPOSITORY_GLOSRY.html) such as [ABAP programs](ABENABAP_PROGRAM_GLOSRY.html), [classes](ABENCLASS_GLOSRY.html), [CDS entities](ABENCDS_ENTITY_GLOSRY.html), [dictionary objects](ABENDICTIONARY_OBJECT_GLOSRY.html), or [CDS objects](ABENCDS_OBJECT_GLOSRY.html). They are maintained [ABAP development tools for Eclipse (ADT)](ABENADT_GLOSRY.html). The names of repository objects are stored in upper case. Each repository object is assigned to a [package](ABENPACKAGE_GLOSRY.html).
