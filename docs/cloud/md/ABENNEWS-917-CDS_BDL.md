A new [RAP draft handling](ABENRAP_DRAFT_HANDLING_GLOSRY.html) with collaborative capabilities is now available. The [RAP collaborative draft](ABENRAP_COLLABORATIVE_DRAFT_GLOSRY.html) allows multiple users to work concurrently on the same draft instance. It is defined using the syntax [`with collaborative draft`](ABENBDL_WITH_COLL_DRAFT.html).
