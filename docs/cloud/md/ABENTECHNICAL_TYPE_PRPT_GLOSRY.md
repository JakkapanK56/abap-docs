Property of a [data type](ABENDATA_TYPE_GLOSRY.html) that is important for the [compatibility](ABENCOMPATIBLE_GLOSRY.html) with other data types except [reference types](ABENREFERENCE_TYPE_GLOSRY.html). For [elementary data types](ABENELEMENTARY_DATA_TYPE_GLOSRY.html), this concerns the [built-in ABAP type](ABENBUILTIN_ABAP_TYPE_GLOSRY.html), the length of the [fraction portion](ABENDECIMAL_PLACE_GLOSRY.html) and the number of decimal places. For [structured types](ABENSTRUCTURED_TYPE_GLOSRY.html), this concerns how the type is constructed from [substructures](ABENSUBSTRUCTURE_GLOSRY.html) and elementary data types. For [table types](ABENTABLE_TYPE_GLOSRY.html), this concerns the [table category](ABENTABLE_CATEGORY_GLOSRY.html), the [line type](ABENROW_TYPE_GLOSRY.html), and the [table key](ABENTABLE_KEY_GLOSRY.html). For [mesh types](ABENMESH_TYPE_GLOSRY.html), this concerns the layout of components and their [mesh associations](ABENMESH_ASSOCIATION_GLOSRY.html).
