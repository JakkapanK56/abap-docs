Object plug-ins are used to make modification-free functional [enhancements](ABENENHANCEMENT_GLOSRY.html) to ABAP programs. They are instances of [BAdI implementation classes](ABENBADI_IMPLEMENT_CLASS_GLOSRY.html) and are administered by [BAdI objects](ABENBADI_OBJECT_GLOSRY.html). Object plug-ins are created when the [`GET BADI`](ABAPGET_BADI.html) statement is executed and in accordance with the filter specifications.
