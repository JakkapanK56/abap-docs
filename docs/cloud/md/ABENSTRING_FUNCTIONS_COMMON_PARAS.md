The following parameters are common for multiple [string functions](ABENSTRING_FUNCTIONS.html) and in [predicate functions for character-like arguments](ABENPREDICATE_FUNCTIONS_STRGS.html) and have the same effect:

-   [`cstring_func` - `case`](ABENSTRING_FUNCTIONS_CASE.html)
-   [`cstring_func` - `occ`](ABENSTRING_FUNCTIONS_OCC.html)
-   [`cstring_func` - `off, len`](ABENSTRING_FUNCTIONS_OFF_LEN.html)
-   [`cstring_func` - `pcre`, `xpath`](ABENSTRING_FUNCTIONS_REGEX.html)
-   [`cstring_func` - `sub`](ABENSTRING_FUNCTIONS_SUB.html)
-   [`cstring_func` - `val`](ABENSTRING_FUNCTIONS_VAL.html)
