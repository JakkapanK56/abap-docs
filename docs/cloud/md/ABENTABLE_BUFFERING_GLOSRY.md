[Buffering](ABENSAP_BUFFERING_OLD_GLOSRY.html) of data from [DDIC database tables](ABENDDIC_DB_TABLE_GLOSRY.html), [DDIC database views](ABENDATABASE_VIEW_GLOSRY.html), or from [CDS entities](ABENCDS_ENTITY_GLOSRY.html) in a [table buffer](ABENTABLE_BUFFER_GLOSRY.html) in the [shared memory](ABENSHARED_MEMORY_GLOSRY.html) of the current [AS instance](ABENAS_INSTANCE_GLOSRY.html). Whether and how a DDIC database table or a DDIC database view is buffered is determined by its [technical settings](ABENDDIC_DATABASE_TABLES_BUFFER.html). For a CDS entity, the buffering is determined by annotations and a [CDS entity buffer](ABENCDS_ENTITY_BUFFER_GLOSRY.html). Buffering is managed by the [database interface](ABENDATABASE_INTERFACE_GLOSRY.html). Table buffering is applied implicitly when the database is accessed using [ABAP SQL](ABENABAP_SQL_GLOSRY.html) In ABAP SQL, the [ABAP SQL in-memory engine](ABENABAP_SQL_INMEMENG_GLOSRY.html) processes the tabular data in the table buffer.
