[Data type](ABENDATA_TYPE_GLOSRY.html) that is provided in an [ABAP program](ABENABAP_PROGRAM_GLOSRY.html) by the [ABAP runtime framework](ABENABAP_RUNTIME_FRMWK_GLOSRY.html). The built-in data types are [built-in ABAP types](ABENBUILTIN_ABAP_TYPE_GLOSRY.html), [generic ABAP types](ABENGENERIC_ABAP_TYPE_GLOSRY.html), [built-in data types](ABENDDIC_BUILTIN_TYPES.html) from [ABAP Dictionary](ABENABAP_DICTIONARY_GLOSRY.html), and [CDS built-in types](ABENCDS_BUILT_IN_TYPE_GLOSRY.html). The names of the elementary and generic built-in ABAP types are a subset of the [reserved names](ABENABAP_WORD_GLOSRY.html) (except for internal types `b` and `s`, which cannot be used directly in a program).
