[Byte-like data type](ABENBYTE_LIKE_DATA_TYP_GLOSRY.html) for storing [byte strings](ABENBYTE_STRING_GLOSRY.html) of fixed length. The associated [built-in ABAP type](ABENBUILTIN_ABAP_TYPE_GLOSRY.html) is [`x`](ABENBUILTIN_TYPES_BYTE.html). The [built-in DDIC types](ABENBUILTIN_DDIC_TYPE_GLOSRY.html) are [`RAW`](ABENDDIC_BUILTIN_TYPES.html#abenddic-builtin-types-4--and---lraw---abenddic-builtin-types-html-@ITOC@@ABENDDIC_BUILTIN_TYPES_4). A [data object](ABENDATA_OBJECT_GLOSRY.html) of a byte field type is a [byte field](ABENBYTE_FIELD_GLOSRY.html).
