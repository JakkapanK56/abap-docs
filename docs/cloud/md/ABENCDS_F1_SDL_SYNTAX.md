A [CDS service definition](ABENCDS_SERVICE_DEFINITION_GLOSRY.html) in ABAP CDS is created using the data definition language [CDS SDL](ABENCDS_SDL_GLOSRY.html) in [SDL source code](ABENSDL_SOURCE_CODE_GLOSRY.html). CDS SDL complies with the [general CDS syntax rules](ABENCDS_GENERAL_SYNTAX_RULES.html) and contains the following language element for service definitions:

-   [`DEFINE SERVICE`](ABENSRVD_DEFINE_SERVICE.html)

-   The [SDL source code](ABENSDL_SOURCE_CODE_GLOSRY.html) of service definitions has its own editor in [ADT](ABENADT_GLOSRY.html), which is documented there.
