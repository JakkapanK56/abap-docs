An [index](ABENINDEX_GLOSRY.html) of an [internal table](ABENINTERNAL_TABLE_GLOSRY.html) managed by the [ABAP runtime framework](ABENABAP_RUNTIME_FRMWK_GLOSRY.html). A [primary table index](ABENPRIMARY_TABLE_INDEX_GLOSRY.html) is used for [index tables](ABENINDEX_TABLE_GLOSRY.html) and [secondary table indexes](ABENSECONDARY_TABLE_INDEX_GLOSRY.html) are used for the management of [sorted](ABENSORTED_KEY_GLOSRY.html)\\ [secondary table keys](ABENSECONDARY_TABLE_INDEX_GLOSRY.html). The line numbers of every table index start at 1 and run consecutively with no gaps up to the current number of lines in the table. If necessary, the physical index is changed to a logical index, which increases runtime demand. The logical index is created in exactly the moment in which a line is inserted before another line, the order of the table lines is changed, or if any line except for the bottom line is deleted.
