It is possible to define whether DDIC database tables can be displayed or maintained using certain tools such as Data Preview in [ADT](ABENADT_GLOSRY.html).

The following settings are possible:

-   **Display/maintenance not allowed**

-   Tables cannot be displayed using Data Preview in ADT.

-   **Display/maintenance allowed with restrictions**

-   Tables can be displayed using Data Preview in ADT.

-   **Display/maintenance allowed**

-   Tables can be displayed using Data Preview in ADT.

-   **Only display allowed**

-   Tables can be displayed using Data Preview in ADT.
