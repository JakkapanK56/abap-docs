``... @[`annotation`](ABENCDS_ANNOTATIONS.html) ...``

Specifies an [annotation](ABENCDS_ANNOTATION_GLOSRY.html)\\ [`annotation`](ABENCDS_ANNOTATIONS_SYNTAX.html) in the definition of an aspect [`DEFINE ASPECT`](ABENCDS_F1_DEFINE_GENERIC_ASPECT.html). The character `@` must be placed in front of the name `annotation` of the annotation.

The following [ABAP annotations](ABENABAP_ANNOTATION_GLOSRY.html) can be specified for `annotation`. No other annotations can be specified.

-   EndUserText Annotations

-   `[EndUserText.label](ABENCDS_924137870_ANNO.html)`
