[Calculation expressions](ABENCALCULATION_EXPRESSION_GLOSRY.html) are [arithmetic expressions](ABENARITHMETIC_EXPRESSION_GLOSRY.html), [string expressions](ABENSTRING_EXPRESSION_GLOSRY.html), and [bit expressions](ABENBIT_EXPRESSION_GLOSRY.html). Any calculation expression can be located on the right side of the [assignment operator `=`](ABENEQUALS_OPERATOR.html).

-   [Assign Arithmetic Expression](ABENEQUALS_ARITH_EXPR.html)
-   [Assign String Expression](ABENEQUALS_STRING_EXPR.html)
-   [Assign Bit Expression](ABENEQUALS_BIT_EXPR.html)
