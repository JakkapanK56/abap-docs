Character that substitutes other content. In [relational expressions](ABENCOMPARISON_EXPRESSION_GLOSRY.html) with the operators [`CP`](ABENLOGEXP_STRINGS.html) and [`NP`](ABENLOGEXP_STRINGS.html), `*` usually stands for any [string](ABENCHARACTER_STRING_GLOSRY.html) and `+` for any individual character. In [SQL conditions](ABENSQL_COND_GLOSRY.html) with the [operator](ABENOPERATOR_GLOSRY.html)\\ [`LIKE`](ABENWHERE_LOGEXP_LIKE.html), `%` stands for any string and `_` for individual characters. In [regular expressions](ABENREGULAR_EXPRESSION_GLOSRY.html), an extended set of special characters is used to define patterns. The special functions of wildcard characters can be switched off using [escape characters](ABENESCAPE_CHARACTER_GLOSRY.html)
