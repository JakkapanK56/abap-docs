[Database](ABENDATABASE_GLOSRY.html) whose data is managed according to the [relational data model](ABENRELATIONAL_DATA_MODEL_GLOSRY.html), that is, in [database tables](ABENDATABASE_TABLE_GLOSRY.html). Further objects such as [views](ABENVIEW_GLOSRY.html) can be created to access the data. These objects are organized in [database schemas](ABENDATABASE_SCHEMA_GLOSRY.html). The objects are generally defined and accessed using [SQL](ABENSQL_GLOSRY.html).
