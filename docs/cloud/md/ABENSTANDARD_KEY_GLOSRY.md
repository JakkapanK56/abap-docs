[Primary table key](ABENPRIMARY_TABLE_KEY_GLOSRY.html) of an [internal table](ABENINTERNAL_TABLE_GLOSRY.html), whose [key fields](ABENKEY_FIELD_GLOSRY.html) in a structured [line type](ABENROW_TYPE_GLOSRY.html) are all table fields with [character-like data types](ABENCHARLIKE_DATA_TYPE_GLOSRY.html) and [byte-like data types](ABENBYTE_LIKE_DATA_TYP_GLOSRY.html). If the line type contains [substructures](ABENSUBSTRUCTURE_GLOSRY.html), they are broken down into elementary [components](ABENSTRUCTURE_COMPONENT_GLOSRY.html). The standard key for non-structured line types is the entire table line, if the line type itself is not a table type. If there are no corresponding table fields, or the line type itself is a table type, the standard key of [standard tables](ABENSTANDARD_TABLE_GLOSRY.html) is empty or contains no key fields.
