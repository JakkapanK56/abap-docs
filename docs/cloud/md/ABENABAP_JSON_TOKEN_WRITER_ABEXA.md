The example demonstrates the rendering of [JSON data](ABENJSON_OVIEW.html).

An [XML string writer](ABENABAP_SXML_LIB_RENDER.html) from [XML Library](ABENABAP_SXML_LIB.html) is created as a [JSON writer](ABENJSON_WRITER_GLOSRY.html) with the type `IF_SXML=>CO_XT_JSON`. [Token-based rendering](ABENABAP_SXML_LIB_RENDER_TOKEN.html) methods are used to fill the writer with XML elements that create a valid [JSON-XML](ABENJSON_XML_GLOSRY.html) representation. The result is a string with valid [JSON data](ABENJSON_OVIEW.html).
