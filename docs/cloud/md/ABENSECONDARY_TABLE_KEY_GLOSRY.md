Up to 15 secondary [table keys](ABENTABLE_KEY_GLOSRY.html) can be created for each [internal table](ABENINTERNAL_TABLE_GLOSRY.html). Possible secondary table keys are [sorted keys](ABENSORTED_KEY_GLOSRY.html) and [hash keys](ABENHASH_KEY_GLOSRY.html). A [secondary table index](ABENSECONDARY_TABLE_INDEX_GLOSRY.html) is created internally for each sorted secondary key. When internal tables are accessed, the table key to be used must be specified.
