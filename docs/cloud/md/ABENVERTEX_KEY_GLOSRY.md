A vertex key represents one [vertex](ABENVERTEX_GLOSRY.html) attribute or a set of [vertex attributes](ABENVERTEX_ATTR_GLOSRY.html) that must uniquely identify vertices. If the vertex key consists of multiple [attributes](ABENVERTEX_ATTR_GLOSRY.html), then the [edge table's](ABENEDGE_TABLE_GLOSRY.html) source and target must consist of a corresponding set of attributes.
