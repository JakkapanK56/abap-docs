Composite specification of an [ABAP operand](ABENABAP_OPERAND_GLOSRY.html) in an [operand position](ABENOPERAND_POSITION_GLOSRY.html) in which the individual components are chained using [component selectors](ABENCOMPONENT_SELECTOR_GLOSRY.html). Examples of possible chainings are [chained identifiers](ABENCHAINED_NAME_GLOSRY.html), [chained method calls](ABENCHAINED_METHOD_CALL_GLOSRY.html), [chained attribute accesses](ABENCHAINED_ATTRIBUTE_ACC_GLOSRY.html), chainings of [table expressions](ABENTABLE_EXPRESSION_GLOSRY.html), and various combinations of all four.
