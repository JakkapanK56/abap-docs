Procedure to create or update unique [secondary table keys](ABENSECONDARY_TABLE_KEY_GLOSRY.html) of [internal tables](ABENINTERNAL_TABLE_GLOSRY.html). After a modifying operation using [field symbols](ABENFIELD_SYMBOL_GLOSRY.html) or [data references](ABENDATA_REFERENCE_GLOSRY.html), unique secondary table keys are not updated immediately; instead they are updated the next time the internal table is accessed. See also [direct update](ABENDIRECT_UPDATE_GLOSRY.html) and [lazy update](ABENLAZY_UPDATE_GLOSRY.html).
