[Data type](ABENDATA_TYPE_GLOSRY.html) whose [data objects](ABENDATA_OBJECT_GLOSRY.html) contain [characters](ABENCHARACTER_GLOSRY.html) encoded according to the current [code page](ABENCODEPAGE_GLOSRY.html). The actual character-like data types are [text field type](ABENTEXT_FIELD_TYPE_GLOSRY.html), [numeric text field type](ABENNUMERIC_TEXT_FIELD_TYPE_GLOSRY.html), and [text string type](ABENTEXT_STRING_TYPE_GLOSRY.html). [Date type](ABENDATE_TYPE_GLOSRY.html) and [time type](ABENTIME_TYPE_GLOSRY.html) are special character-like data types. The associated [generic data type](ABENGENERIC_DATA_TYPE_GLOSRY.html) is [`clike`](ABENBUILT_IN_TYPES_GENERIC.html).
