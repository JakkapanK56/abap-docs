Property of [comparison expressions](ABENCOMPARISON_EXPRESSION_GLOSRY.html) whose [operands](ABENOPERAND_GLOSRY.html) have [elementary data types](ABENELEMENTARY_DATA_TYPE_GLOSRY.html), including [calculation expressions](ABENCALCULATION_EXPRESSION_GLOSRY.html). The comparison type is determined from the operands and defines how the comparison is executed. The comparison type can be of any [built-in ABAP type](ABENBUILTIN_ABAP_TYPE_GLOSRY.html). Operands of another type are converted to the comparison type before the comparison.
