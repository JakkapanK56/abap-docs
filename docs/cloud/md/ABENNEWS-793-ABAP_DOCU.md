From Release 7.93 on, the ABAP Keyword Documentation presents a new layout in [ABAP development tools for Eclipse (ADT)](ABENADT_GLOSRY.html) where a toggle between *Standard ABAP* and *ABAP Cloud* is also available. The new layout includes:

-   New header bar with links to the mail feedback and the web version
-   Highlighted key blocks for hints, code blocks, and others
-   Tables with zebra stripes
