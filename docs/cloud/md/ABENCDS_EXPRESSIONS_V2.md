[CDS DDL expressions](ABENCDS_EXPRESSION_GLOSRY.html) are expressions that can be specified in a CDS DDL operand position of a [CDS view entity](ABENCDS_V2_VIEW_GLOSRY.html).

The following CDS DDL expressions exist:

-   [Arithmetic expressions](ABENCDS_ARITHMETIC_EXPRESSION_V2.html)
-   [Aggregate expressions](ABENCDS_AGGREGATE_FUNCTIONS_V2.html)
-   [Case distinctions](ABENCDS_CASE_EXPRESSION_V2.html)
-   [Cast expressions](ABENCDS_CAST_EXPRESSION_V2.html)
-   [Calls of built-in CDS functions](ABENCDS_BUILTIN_FUNCTIONS_V2.html)
