Defines the properties of local program entities. [Data types](ABENDATA_TYPE_GLOSRY.html), [data objects](ABENDATA_OBJECT_GLOSRY.html), [classes](ABENCLASS_GLOSRY.html), [interfaces](ABENOO_INTF_GLOSRY.html), and [field symbols](ABENFIELD_SYMBOL_GLOSRY.html) can be declared. Declarations are made either with [declaration statements](ABENDECLARATION_STATEMENT_GLOSRY.html) or as [inline declarations](ABENINLINE_DECLARATION_GLOSRY.html) using [declaration expressions](ABENDECLARATION_EXPRESSION_GLOSRY.html). The declared unit is visible statically as soon as it is declared and is valid throughout the entire [context](ABENOBJ_CONTEXT_GLOSRY.html).
