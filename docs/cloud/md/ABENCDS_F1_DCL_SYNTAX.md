The data control language CDS DCL in ABAP CDS follows the [general CDS syntax rules](ABENCDS_GENERAL_SYNTAX_RULES.html) and comprises the following statements for the definition of [CDS roles](ABENCDS_ROLE_GLOSRY.html) and [CDS access policies](ABENCDS_ACCESS_POLICY_GLOSRY.html) in [DCL source code](ABENDCL_SOURCE_CODE_GLOSRY.html):

-   [`DEFINE ROLE`](ABENCDS_F1_DEFINE_ROLE.html)
-   [`DEFINE ACCESSPOLICY`](ABENCDS_F1_DEFINE_ACCESSPOLICY.html)

-   The [DCL source code](ABENDCL_SOURCE_CODE_GLOSRY.html) for CDS access control has its own editor in [`ADT`](ABENADT_GLOSRY.html), which is documented there.
-   The names of the items of DCL source code for [CDS roles](ABENCDS_ROLE_GLOSRY.html) and [CDS access policies](ABENCDS_ACCESS_POLICY_GLOSRY.html) are in the same namespace.
