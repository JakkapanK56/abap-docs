The following ABAP SQL statements implement the [DQL](ABENDQL_GLOSRY.html) part of [SQL](ABENSQL_GLOSRY.html) in the ABAP language for performing reads on the database:

-   [`SELECT`](ABAPSELECT.html)
-   [`WITH`](ABAPWITH.html)

-   `FETCH NEXT CURSOR`
-   `CLOSE CURSOR`

-   [`UNION`](ABAPUNION.html), [`INTERSECT`](ABAPUNION.html), [`EXCEPT`](ABAPUNION.html)
