The following special language constructs are used to assign components from structures or columns of internal tables:

-   [`MOVE-CORRESPONDING`](ABAPMOVE-CORRESPONDING.html) for the assignment of identically named components.
-   Component operator [`CORRESPONDING`](ABENCONSTRUCTOR_EXPR_CORRESPONDING.html) for the assignment of components using identical names or a static mapping rule.
-   System class [`CL_ABAP_CORRESPONDING`](ABENCL_ABAP_CORRESPONDING.html) for dynamic mapping rules.
