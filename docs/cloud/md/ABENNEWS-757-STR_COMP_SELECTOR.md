The following syntax can be used for the [structure component selector `-`](ABENSTRUCTURE_COMPONENT_SELECTOR.html) to access components dynamically now:

`... struct-(comp)`

This is especially possible for [dynamic components](ABAPASSIGN_DYNAMIC_COMPONENTS.html) in the statement `ASSIGN`.
