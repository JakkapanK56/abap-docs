Except for [reference variables](ABENREFERENCE_VARIABLE_GLOSRY.html), two non-[generic data types](ABENGENERIC_DATA_TYPE_GLOSRY.html) are compatible if all their [technical properties](ABENTECHNICAL_TYPE_PRPT_GLOSRY.html) are the same. Any [data type](ABENDATA_TYPE_GLOSRY.html) is compatible with a generic data type if the generic data type includes its technical properties. [Data objects](ABENDATA_OBJECT_GLOSRY.html) are compatible if their data types are compatible. Compatibility is the basis for checking the [typing](ABENTYPING_GLOSRY.html). In an [assignment](ABENASSIGNMENT_GLOSRY.html) or [comparison](ABENCOMPARISON_GLOSRY.html) of compatible data objects, no conversion takes place.
