Persistent storage for [data types](ABENDATA_TYPE_GLOSRY.html) that are visible in all [repository objects](ABENREPOSITORY_OBJECT_GLOSRY.html). In particular, [DDIC database tables](ABENDDIC_DB_TABLE_GLOSRY.html) on the [standard database](ABENSTANDARD_DB_GLOSRY.html) that can be used in [ABAP programs](ABENABAP_PROGRAM_GLOSRY.html) and in [ABAP CDS](ABENABAP_CDS_GLOSRY.html) are defined in the ABAP Dictionary. [Dictionary objects](ABENDICTIONARY_OBJECT_GLOSRY.html) are maintained using the [ABAP development tools for Eclipse (ADT)](ABENADT_GLOSRY.html).
