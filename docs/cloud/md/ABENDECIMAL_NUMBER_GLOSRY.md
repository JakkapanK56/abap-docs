Number in the decimal numbering system. It consists of [places](ABENPLACE_GLOSRY.html), which can contain the decimal numbers 0-9, and can also contain a [decimal separator](ABENDECIMAL_POINT_SEPARAT_GLOSRY.html) and a leading plus/minus sign.
