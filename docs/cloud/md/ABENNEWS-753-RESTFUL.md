From release 7.33 on, the [ABAP RESTful Application Programming Model (RAP)](ABENARAP_GLOSRY.html) is introduced.

This mainly involves:

In the beginning, so-called [unmanaged RAP BOs](ABENUNMANAGED_RAP_BO_GLOSRY.html) can be created with the addition [`unmanaged`](ABENBDL_IMPL_TYPE.html) of the [RAP BDL](ABENCDS_BDL_GLOSRY.html) when defining a [RAP BO](ABENRAP_BO_GLOSRY.html). This scenario is intended for brownfield developments with existing application code. All operations and services must be implemented by the application developer.
