``... \{ [`element1`](ABENCDS_ASPECT_ELEMENT.html); [`element2`](ABENCDS_ASPECT_ELEMENT.html); ... \}``

Defines an element of a CDS DDL aspect. The elements can be specified in a list separated by semicolons: `element1`; `element2` ...

The names of the elements of a CDS aspect must be unique to the fields included in the CDS DDL aspect as well as the entities that the aspect is to be included in.
