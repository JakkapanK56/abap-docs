A [hierarchy node](ABENHIERARCHY_NODE_GLOSRY.html) from which a [parent-child relationship](ABENPCR_GLOSRY.html) to a [child node](ABENCHILD_NODE_GLOSRY.html) originates. A parent node can have multiple child nodes. A parent node is an [ancestor node](ABENANCESTOR_NODE_GLOSRY.html) of its child nodes, their child nodes, and so on.
