The following additions can be used to use a database connection other than the [standard connection](ABENSTANDARD_DB_CONNECTION_GLOSRY.html) in ABAP SQL:

Here, a service connection can be specified by using an [ABAP SQL connection object](ABENABAP_SQL_CONN_OBJ_GLOSRY.html).

-   The addition [`CONNECTION`](ABAPSELECT_CONNECTION.html) for DQL statements
-   The identical addition [`CONNECTION`](ABAPIUMD_CONN.html) for DML statements
