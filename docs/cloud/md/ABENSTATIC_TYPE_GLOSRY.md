[Data type](ABENDATA_TYPE_GLOSRY.html) of a [reference variable](ABENREFERENCE_VARIABLE_GLOSRY.html) ([reference type](ABENREFERENCE_TYPE_GLOSRY.html)). It determines the [objects](ABENOBJECT_GLOSRY.html) a reference variable can point to. For [object reference variables](ABENOBJECT_REFER_VARIABLE_GLOSRY.html), the static type is a [class](ABENCLASS_GLOSRY.html) or an [interface](ABENOO_INTF_GLOSRY.html). For [data reference variables](ABENDATA_REFERENCE_VARIABLE_GLOSRY.html), the static type is a data type. The static type is always more general than or the same as the [dynamic type](ABENDYNAMIC_TYPE_GLOSRY.html).
