The example demonstrates how to use a PCRE regular expression to parse a syntax.

Advanced example for using [PCRE syntax](ABENREGEX_PCRE_SYNTAX.html) for regular expressions. Using subroutines defined with [`(?<name> ...)`](ABENREGEX_PCRE_SYNTAX_SPECIALS.html) and callouts with [`(?C"...")`](ABENREGEX_PCRE_SYNTAX_SPECIALS.html), an abstract syntax tree (AST) is generated from an input if it follows the example's syntax rules.
