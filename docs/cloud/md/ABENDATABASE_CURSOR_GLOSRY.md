Pointer to the result set of a database selection. The database cursor is always assigned to a line of the result set. In [ABAP SQL](ABENABAP_SQL_GLOSRY.html), cursor handling is implicit, except for the statements `OPEN CURSOR`, `FETCH`, and `CLOSE CURSOR`. A program can open a maximum of 17 database cursors simultaneously using the [ABAP SQL interface](ABENABAP_SQL_INTERFACE_GLOSRY.html).
