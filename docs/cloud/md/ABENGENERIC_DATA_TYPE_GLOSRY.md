[Data type](ABENDATA_TYPE_GLOSRY.html) that does not set all properties of a [data object](ABENDATA_OBJECT_GLOSRY.html). Generic data types can only be used for the [typing](ABENTYPING_GLOSRY.html) of [formal parameters](ABENFORMAL_PARAMETER_GLOSRY.html) and [field symbols](ABENFIELD_SYMBOL_GLOSRY.html). A generic data type is either a [generic ABAP type](ABENGENERIC_ABAP_TYPE_GLOSRY.html) or a [table type](ABENTABLE_TYPE_GLOSRY.html) with no [primary table key](ABENPRIMARY_TABLE_KEY_GLOSRY.html) or an incomplete one.
