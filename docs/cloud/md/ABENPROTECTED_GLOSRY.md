Term in [ABAP Objects](ABENABAP_OBJECTS_GLOSRY.html). A protected [component](ABENCLASS_COMPONENT_GLOSRY.html) of a [class](ABENCLASS_GLOSRY.html) can be accessed only by the class itself or by its subclasses. Only the class itself and its subclasses can create [objects](ABENOBJECT_GLOSRY.html) in the case of a protected instantiation. Here, [friends](ABENFRIEND_GLOSRY.html) of the class have the same rights as the class itself. The protected [visibility section](ABENVISIBILITY_SECTION_GLOSRY.html) of a class is declared in the statement [`PROTECTED SECTION`](ABAPPROTECTED.html) in the [declaration section](ABENDECLARATION_PART_GLOSRY.html).
