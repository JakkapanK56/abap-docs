The following persistent texts can be accessed in ABAP programs:

These texts can be translated into different languages. By default, a program accesses the texts in the [logon language](ABENLOGON_LANGUAGE_GLOSRY.html).

Another central repository for texts is Online Text Repository (OTR), which can be accessed using the transaction `SOTR_EDIT`. In ABAP, OTR is important for [exception texts](ABENEXCEPTION_TEXTS.html) and is also used by other frameworks.
