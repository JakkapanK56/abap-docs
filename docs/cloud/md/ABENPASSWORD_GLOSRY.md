A sequence of characters that is only known to a specific [ABAP user](ABENABAP_USER_GLOSRY.html) and is required from this user to [log on](ABENLOGON_GLOSRY.html) to [AS ABAP](ABENAS_ABAP_GLOSRY.html). Used together with the [user name](ABENUSER_NAME_GLOSRY.html), the password enables authorized use of an AS ABAP.
