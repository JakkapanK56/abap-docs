The process of transferring data from a consumer system to a provider system in a cross-system scenario. The consumer system consumes the required data in real time without copying it to the local system.

The [SAP HANA database](ABENHANA_DATABASE_GLOSRY.html) uses data federation as a part of its [SAP HANA Smart Data Access](ABENSAP_HANA_SDA_GLOSRY.html) technology to access and query data from other (remote) enterprise data stores.

In ABAP CDS, access to the federated data of an external system is supported by [CDS external entities](ABENCDS_EXTERNAL_ENTITY_GLOSRY.html).

See also: [Data replication](ABENDATA_REPLICATION_GLOSRY.html)
