The following figure shows the location of the ABAP Keyword Documentation in respect to other documentation for [ABAP Cloud](ABENABAP_CLOUD_GLOSRY.html).

ABAP Cloud is the ABAP development model to build cloud-ready business apps, services, and extensions. The [*ABAP Cloud Guide*](https://help.sap.com/docs/ABAP_Cloud/49f2f38ba2bd4f1cb8e0fd76d5a5f0d7/177b34d16990494a94bba8c608e14d87?version=latest) gives an overview of this model and how to use it to develop in ABAP Cloud.

The [SAP Fiori App Development Overview](https://help.sap.com/docs/SAP_FIORI_OVERVIEW) page provides a single point of access for everyone who is looking for information about SAP Fiori, including how to develop SAP Fiori apps.

The [*ABAP RESTful Application Programming Model (RAP model)*](https://help.sap.com/docs/ABAP_Cloud/f055b8bf582d4f34b91da667bc1fcce6/289477a81eec4d4e84c0302fb6835035?version=sap_cross_product_abap) defines the architecture for efficient end-to-end development of intrinsically SAP HANA-optimized OData services (such as Fiori apps) in ABAP Cloud.

The [*ABAP Analytics Guide*](https://help.sap.com/docs/ABAP_Cloud/67127c83ae1a4917b6b65ec222416894/4ae5db4d9bfd472ba70613b358dbb16b?version=sap_cross_product_abap) describes how you can build comprehensive multidimensional data models using ABAP and ABAP CDS, while leveraging the full querying capabilities of SAP HANA.

The SAP BTP ABAP environment enables you to connect to other systems using various protocols, consume services from other systems, and provide services for consumption in other systems. The [*ABAP Integration and Connectivity Guide*](https://help.sap.com/docs/ABAP_Cloud/eede1416d18c436e8810eaaeb20c38ae/7f07b344bc7a4e8b9ca748c00e379e74) shows you how to do so.

The [*ABAP Concepts Guide*](https://help.sap.com/docs/ABAP_Cloud/f2961be2bd3d403585563277e65d108f/e59f2729ea7b49e188c8e20ee4eb9633?version=sap_cross_product_abap) explains concepts of the ABAP programming language.

The [*ABAP Data Models Guide*](https://help.sap.com/docs/ABAP_Cloud/aaae421481034feab3e71dd9e0f643bf/977a051cea1c4059be2500dcbf2d3d7d?version=sap_cross_product_abap) describes the means ABAP provides for creating, extending, and consuming data models of ABAP applications for ABAP Cloud. ABAP data models are based on ABAP Core Data Services (ABAP CDS) and can be used as the foundation for transactional and analytical ABAP applications, as well as enterprise search models.

SQLScript is a collection of extensions to the Structured Query Language (SQL). The [*SAP HANA SQLScript Reference Guide*](https://help.sap.com/docs/SAP_HANA_PLATFORM/de2486ee947e43e684d39702027f8a94/28f2d64d4fab4e789ee0070be418419d) describes how to use the SQL extension SAP HANA SQLScript to embed data-intensive application logic into SAP HANA.

The [*Extensibility White Paper*](https://www.sap.com/documents/2022/10/52e0cd9b-497e-0010-bca6-c68f7e60039b.html) provides an overview of the extensibility options for ABAP-based extensions for SAP S/4HANA in the cloud and on-premise, and provides guidelines for project managers, key users, and ABAP developers.

The [*Tools Product Page*](https://help.sap.com/docs/ABAP_DEVELOPMENT_TOOLS_FOR_ECLIPSE) gives an overview of the tools used for ABAP development. ABAP development tools for Eclipse is SAP's state-of-the-art integrated development environment (IDE) for ABAP development on the open Eclipse platform. On this page you can find an overview of the new features and functions, the development guide for ADT, and more.
