[ABAP program](ABENABAP_PROGRAM_GLOSRY.html) that contains the definition of one [global class](ABENGLOBAL_CLASS_GLOSRY.html) and is loaded when the class is used. In addition to the global class, a class pool can contain further [local classes](ABENLOCAL_CLASS_GLOSRY.html), that are defined in a [CCDEF include](ABENCCDEF_GLOSRY.html), [CCIMP include](ABENCCIMP_GLOSRY.html), or a [test include](ABENTEST_INCLUDE_GLOSRY.html).
