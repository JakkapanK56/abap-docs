Short for Universal Character Set. A 4-byte [character set](ABENCHARACTER_SET_GLOSRY.html), standardized in ISO-10646, that contains all other character sets. The `UCS` character set is currently identical to the [Unicode](ABENUNICODE_GLOSRY.html) character set; that is, no other characters than those contained in Unicode are currently defined. [Character representations](ABENCHAR_REPRESENTATION_GLOSRY.html) for the `UCS` character set are the [UTF](ABENUTF_GLOSRY.html) representations [UCS-2](ABENUCS2_GLOSRY.html) and UCS-4.
