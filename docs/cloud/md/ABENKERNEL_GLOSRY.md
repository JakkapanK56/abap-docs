Part of the [ABAP runtime framework](ABENABAP_RUNTIME_FRMWK_GLOSRY.html) which is primarily programmed in C/C++. In particular, [ABAP statements](ABENABAP_STATEMENT_GLOSRY.html) are implemented by C functions in the ABAP kernel. When an ABAP statement is executed, the corresponding C functions or the ABAP kernel are called.
