The statement [`IMPORT`](ABAPIMPORT_DATA_CLUSTER.html) now allows assignments of data of type [`n` to data objects of type `c`](ABAPIMPORT_PARAMETERLIST.html) if they have the same length. This change might lead to incompatible behavior if handling of the former exception leads to different results than the new behavior.
