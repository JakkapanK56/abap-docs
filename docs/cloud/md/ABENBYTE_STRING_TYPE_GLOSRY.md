[Byte-like data type](ABENBYTE_LIKE_DATA_TYP_GLOSRY.html) for storing [byte chains](ABENBYTE_CHAIN_GLOSRY.html) of variable length. The associated [built-in ABAP type](ABENBUILTIN_ABAP_TYPE_GLOSRY.html) is [`xstring`](ABENBUILTIN_TYPES_BYTE.html). The [built-in DDIC type](ABENBUILTIN_DDIC_TYPE_GLOSRY.html) is [`RAWSTRING`](ABENDDIC_BUILTIN_TYPES.html#@@ITOC@@ABENDDIC_BUILTIN_TYPES_2). Internal administration takes place using an internal [reference](ABENREFERENCE_GLOSRY.html) and the data type is [deep](ABENDEEP_GLOSRY.html). A [data object](ABENDATA_OBJECT_GLOSRY.html) of a byte field type is a [byte string](ABENBYTE_STRING_GLOSRY.html).
