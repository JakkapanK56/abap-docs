Test configuration that produces unique test behavior. A fixture comprises test data, test objects, resources, and connections. A fixture is implemented in the methods `[class_]setup` and `[class_]teardown` of [test classes](ABENTEST_CLASS_GLOSRY.html) in [ABAP Unit](ABENABAP_UNIT_GLOSRY.html).
