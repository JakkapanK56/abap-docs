[Exception](ABENEXCEPTION_GLOSRY.html) that can be handled in the program using the statements [`TRY - CATCH - ENDTRY`](ABAPTRY.html) without causing a [runtime error](ABENRUNTIME_ERROR_GLOSRY.html). Catchable exceptions are based on [exception classes](ABENEXCEPTION_CLASS_GLOSRY.html) and are implemented by [exception objects](ABENEXCEPTION_OBJECT_GLOSRY.html). A catchable exception is raised either by the [ABAP runtime framework](ABENABAP_RUNTIME_FRMWK_GLOSRY.html) or by the statement [`RAISE EXCEPTION`](ABAPRAISE_EXCEPTION.html) in a program, or by the addition [`THROW`](ABENCONDITIONAL_EXPRESSION_RESULT.html) in a [conditional expression](ABENCONDITIONAL_EXPRESSION_GLOSRY.html).
