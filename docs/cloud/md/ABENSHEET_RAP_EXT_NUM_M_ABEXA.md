**Behavior implementation**

For the above RAP behavior definition, one [ABP](ABENBEHAVIOR_POOL_GLOSRY.html) is created. The global class of the behavior pool is `BP_DEMO_CS_RAP_RO_M`. The actual behavior implementation takes place in local classes that are defined and implemented in the `BP_DEMO_CS_RAP_RO_M===========CCIMP` of the behavior pool. See comments in the code for more information on the implementation.

**Access with ABAP using EML**

The above source code uses [EML](ABENEML_GLOSRY.html) to access the RAP business object from an ABAP class.

The class demonstrates various RAP BO operations and aspects:

See comments in the code for more information.
