Language that an [ABAP user](ABENABAP_USER_GLOSRY.html) specifies when logging on to an [AS ABAP](ABENAS_ABAP_GLOSRY.html). The logon language is persistent for the entire [user session](ABENUSER_SESSION_GLOSRY.html). The logon language affects [messages](ABENMESSAGE_GLOSRY.html), [text elements](ABENTEXT_ELEMENT_GLOSRY.html), and the [text environment](ABENTEXT_ENVIRONMENT_GLOSRY.html).
