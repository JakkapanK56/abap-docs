[Table category](ABENTABLE_CATEGORY_GLOSRY.html) of an [internal table](ABENINTERNAL_TABLE_GLOSRY.html) that is managed using a [primary table index](ABENPRIMARY_TABLE_INDEX_GLOSRY.html) and that has no unique [primary table key](ABENPRIMARY_TABLE_KEY_GLOSRY.html). When a standard table is accessed using its primary table key, it is searched linearly. [Secondary table keys](ABENSECONDARY_TABLE_KEY_GLOSRY.html) can be defined for more efficient key access to standard tables. The associated [generic data type](ABENGENERIC_DATA_TYPE_GLOSRY.html) is [`[standard] table`](ABENBUILT_IN_TYPES_GENERIC.html).
