From release 7.92 on, the ABAP Keyword Documentation display offers a function to send a feedback mail to *f1\_help@sap.com* in non-SAP development systems and in the [SAP Help Portal](https://help.sap.com/doc/abapdocu_latest_index_htm/latest/en-US/index.htm). Up to now, the possibility for a feedback mail existed only in SAP's own development systems. The function can be switched on or off in the configuration of the ABAP Keyword Documentation, see SAP Note [3051036](https://launchpad.support.sap.com/#/notes/3051036).
