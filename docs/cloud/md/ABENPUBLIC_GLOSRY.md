Term in [ABAP Objects](ABENABAP_OBJECTS_GLOSRY.html). A public [component](ABENCLASS_COMPONENT_GLOSRY.html) of a [class](ABENCLASS_GLOSRY.html) can be accessed at any point at which the class is known. In case of a public instantiation, [objects](ABENOBJECT_GLOSRY.html) can be created at every point at which a class is known. The public [visibility section](ABENVISIBILITY_SECTION_GLOSRY.html) of a class is declared in its [declaration part](ABENDECLARATION_PART_GLOSRY.html) using the statement [`PUBLIC SECTION`](ABAPPUBLIC.html).
