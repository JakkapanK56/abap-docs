[Regular expression](ABENREGULAR_EXPRESSION_GLOSRY.html) using the syntax and semantics described under [Query 1.0 and XPath 2.0 Functions and Operators](https://www.w3.org/TR/xquery-operators/#regex-syntax). ABAP supports [XPath syntax](ABENREGEX_XPATH_SYNTAX.html) for regular expressions by transforming them to [PCRE](ABENPCRE_GLOSRY.html) internally. See also [PCRE regular expression](ABENPCRE_REGEX_GLOSRY.html), [XSD regular expression](ABENXSD_REGEX_GLOSRY.html) and [POSIX regular expression](ABENPOSIX_REGEX_GLOSRY.html) (obsolete).
