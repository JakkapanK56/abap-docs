For some [CDS entities](ABENCDS_ENTITY_GLOSRY.html) defined with [CDS DDL for data definitions](ABENCDS_F1_DDL_SYNTAX.html), tuning objects can be defined and transported separately with a specialized CDS DDL.

The following [ABAP tuning objects](ABENABAP_TUNING_OBJECT_GLOSRY.html) can be defined:

-   [ABAP CDS - Entity Buffers](ABENCDS_ENTITY_BUFFERS.html)
-   [ABAP CDS - Static Caches](ABENDDIC_STATIC_CACHES.html)
