Template for [objects](ABENOBJECT_GLOSRY.html) in [ABAP Objects](ABENABAP_OBJECTS_GLOSRY.html). Defined with [`CLASS - ENDCLASS`](ABAPCLASS.html), either as a [global class](ABENGLOBAL_CLASS_GLOSRY.html) in a [class pool](ABENCLASS_POOL_GLOSRY.html) or as a [local class](ABENLOCAL_CLASS_GLOSRY.html) in almost any [ABAP program](ABENABAP_PROGRAM_GLOSRY.html). The definition of a class comprises a [declaration part](ABENDECLARATION_PART_GLOSRY.html) for the declaration of the [class components](ABENCLASS_COMPONENT_GLOSRY.html) and an [implementation part](ABENIMPLEMENTATION_PART_GLOSRY.html) for the implementation of the [methods](ABENMETHOD_GLOSRY.html).
