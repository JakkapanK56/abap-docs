[Single code page system](ABENSINGLE_CODEPAGE_SYSTEM_GLOSRY.html) in which characters are encoded in a [Unicode character representation](ABENUNICODE_CHAR_REPRESENT_GLOSRY.html). The [system code page](ABENSYSTEM_CODEPAGE_GLOSRY.html) of a Unicode system is [UTF-16](ABENUTF16_GLOSRY.html) with platform-dependent [byte order](ABENBYTE_ORDER_GLOSRY.html) of which the ABAP programming language supports the subset covered by [UCS-2](ABENUCS2_GLOSRY.html).
