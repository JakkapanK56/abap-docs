Methods are the [procedures](ABENPROCEDURE_GLOSRY.html) of a class whose functionality is implemented between the following statements:

[`METHOD`](ABAPMETHOD.html)\\ 
  ...\\ 
[`ENDMETHOD`](ABAPENDMETHOD.html)

The statement `METHOD` does not usually have any additions. Exceptions to this rule are:

-   Methods used to implement [ABAP Managed Database Procedures](ABENAMDP.html) (AMDP)
