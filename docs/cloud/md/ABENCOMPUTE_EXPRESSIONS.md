Numeric calculations are executed mainly using [arithmetic expressions](ABENARITHMETIC_EXPRESSION_GLOSRY.html) that can be used in operand positions and, in particular, on the right side of the [assignment operator`=`](ABENEQUALS_OPERATOR.html). [Numeric data objects](ABENNUMERIC_DATA_OBJECT_GLOSRY.html), that is, data objects that can be converted to [numeric data types](ABENNUMERIC_DATA_TYPE_GLOSRY.html), and [numeric functions](ABENMATHEMATICAL_FUNKTION_GLOSRY.html) that have a numeric return value can be used as operands of an arithmetic expression. Some [system classes](ABENSYSTEM_CLASS_GLOSRY.html) complement the numeric functions with additional functions. Finally, there are four standalone assignment operators that can be used for the basic arithmetic operations.

In addition to the language elements shown here, there is also a set of [obsolete calculation statements](ABENCOMPUTING_OBSOLETE.html).

-   [Numeric Data Types](ABENNUMBER_TYPES.html)
-   [Arithmetic Expressions (`arith_exp`)](ABAPCOMPUTE_ARITH.html)
-   [Numeric Functions (`num_func`)](ABENMATHEMATICAL_FUNCTIONS.html)
-   [System Classes for Numbers](ABENCL_ABAP_MATH.html)
-   [`+=`, `-=`, `*=`, `/=`, Calculation Assignments](ABENCALCULATION_ASSIGNMENTS.html)
