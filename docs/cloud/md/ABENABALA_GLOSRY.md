The ABAP language or [ABAP programming language](ABENABAP_PROGRAMMING_LANGU_GLOSRY.html) or [ABAP](ABENABAP_GLOSRY.html) for short is the main programming language of the [ABAP development environment (ABAP DE)](ABENABAP_DEV_ENVIR_GLOSRY.html). It is used as the programming interface of an [Application Server ABAP](ABENAS_ABAP_GLOSRY.html) (AS ABAP) of an [ABAP system](ABENABAP_SYSTEM_GLOSRY.html). Depending on the program paradigm [ABAP Cloud](ABENABAP_CLOUD_GLOSRY.html) or [classic ABAP](ABENCLASSIC_ABAP_GLOSRY.html), different [ABAP language versions](ABENABAP_VERSION_GLOSRY.html) can be used.
