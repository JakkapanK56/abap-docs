The sequential and segmental transfer of data or the transfer of [streams](ABENSTREAM_GLOSRY.html). Streaming is intended for all possible resources and is currently implemented for [LOBs](ABENLOB_GLOSRY.html) in ABAP SQL, [strings](ABENSTRING_GLOSRY.html), and [internal tables](ABENINTERNAL_TABLE_GLOSRY.html).
