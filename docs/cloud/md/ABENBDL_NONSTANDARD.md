[Non-standard operations](ABENRAP_NSTANDARD_OPERATION_GLOSRY.html) are [RAP BO operations](ABENRAP_BO_OPERATION_GLOSRY.html) that provide transactional behavior and that are user-implemented. There are two kinds of non-standard operations available:

-   [Actions](ABENBDL_ACTION.html): User-implemented operations that change the data of a BO instance.
-   [Functions](ABENBDL_FUNCTION.html): User-implemented operations that return information without any side effects.
