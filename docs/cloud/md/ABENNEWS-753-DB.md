As of this release, only an [SAP HANA database](ABENHANA_DATABASE_GLOSRY.html) is supported as the [standard database](ABENSTANDARD_DB_GLOSRY.html) of an [AS ABAP](ABENAS_ABAP_GLOSRY.html). For this reason, Open SQL was renamed to ABAP SQL (see [ABAP SQL in ABAP Release 7.53](ABENNEWS-753-ABAP_SQL.html)).
