Description of an [object](ABENOBJECT_GLOSRY.html) in [ABAP Objects](ABENABAP_OBJECTS_GLOSRY.html). Either a [class](ABENCLASS_GLOSRY.html) for the entire object or [interface](ABENOO_INTF_GLOSRY.html) for a part of an object. The associated [generic data type](ABENGENERIC_DATA_TYPE_GLOSRY.html) is [`object`](ABENBUILT_IN_TYPES_GENERIC.html). Object types are either defined as [program global type](ABENPROGRAM_GLOBAL_TYPE_GLOSRY.html) in an [ABAP program](ABENABAP_PROGRAM_GLOSRY.html) or as a [global type](ABENGLOBAL_TYPE_GLOSRY.html) in the [class library](ABENCLASS_LIBRARY_GLOSRY.html).
