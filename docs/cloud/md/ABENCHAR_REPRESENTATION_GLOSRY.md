Binary encoding of characters. [Character sets](ABENCHARACTER_SET_GLOSRY.html) are assigned to a character representation using [code pages](ABENCODEPAGE_GLOSRY.html). The [Unicode character representations](ABENUNICODE_CHAR_REPRESENT_GLOSRY.html) comprise all characters world-wide and can be encoded using one, two, or four bytes, depending on the [UTF](ABENUTF_GLOSRY.html) representation.
